Linux Date 命令

2019年6月3日 • 约 1200 字 , 预计阅读时间 5 分钟

date 命令用来显示或设置系统日期。它经常用于以不同格式打印日期和时间、计算未来和过去的日期。

在本教程中，我们将介绍 date 命令的基础知识。

使用 Linux date 命令

date 命令的语法如下：

date [OPTION]... [+FORMAT]

要使用默认格式显示当前系统时间和日期，请调用不带任何选项的命令：

date

输出将包括星期几，月份，日期，时间，时区和年份：

Sat Jun  1 14:31:01 CEST 2019

日期格式选项

date 可以使用一系列格式控制字符格式化输出，定义格式以 + 符号开头。格式元素以 % 符号开头，并由其值替换。

date +"Year: %Y, Month: %m, Day: %d"

该 %Y 字符表示的是年份， %m 标识月份， %d 标识月份中天：

Year: 2019, Month: 06, Day: 02

这是另一个例子：

date "+DATE: %D%nTIME: %T"
DATE: 06/02/19
TIME: 01:47:04

下面是一些最常见格式字符的小列表：

  • %a  - 本地的星期几的缩写(例如 Mon)
  • %A  - 本地的星期几的完整名称(例如 Monday)
  • %b  - 本地的月份的缩写(例如 Jan)
  • %B  - 本地的的月份的完整名称(例如 January)
  • %d  - 月份日期(例如 01)
  • %H  - 小时 (00..23)
  • %I  - 小时 (01..12)
  • %j  - 一年中的一天 (001..366)
  • %m  - 月 (01..12)
  • %M  - 分钟 (00..59)
  • %S  - 秒 (00..60)
  • %u  - 星期几 (1..7)
  • %Y  - 全年(例如 2019 年)

您可以在终端中运行 date --help 或 man date 获取所有格式选项的完整列表。

日期字符串

该 -d 选项允许您操作指定的日期。您可以将日期指定为人类可读的日期字符串，如下所示：

date -d "2010-02-07 12:10:53"
Sun Feb  7 12:10:53 CET 2010

使用自定义格式：

date -d '16 Dec 1974' +'%A, %d %B %Y'
Monday, 16 December 1974

日期字符串还可以接受诸如 “tomorrow”, “friday”, “last friday” “next friday”, “next month”, “next week” 等等的值。

date -d "last week"
Sat May 25 14:31:42 CEST 2019

您还可以使用日期字符串选项显示不同时区的当地时间。例如，要显示澳大利亚东海岸下周一早上 6:30 的当地时间，您可以输入：

date -d 'TZ="Australia/Sydney" 06:30 next Monday'
Sun Jun  2 22:30:00 CEST 2019

覆盖时区

date 命令使用默认系统时区。要使用不同的时区，请将环境变量  TZ 设置为所需的时区。

例如，要显示墨尔本，澳大利亚时间，您可以输入：

TZ='Australia/Melbourne' date
Sat Jun  1 22:35:10 AEST 2019

您可以列出目录 /usr/share/zoneinfo 中的文件或使用该 timedatectl list-timezones 命令来列出所有可用时区，。

Epoch 转换器

该 date 命令可用作 Epoch 转换器。 Epoch 或 Unix 时间戳是自 1970 年1月1日 00:00:00 UTC 以来经过的秒数。

要打印从纪元到当天的秒数，请使用 %s 格式控件：

date +%s
1559393792

要将自纪元以来的秒数转换为日期，请将秒设置为前缀为 @ 的日期字符串：

date -d @1234567890
Sat Feb 14 00:31:30 CET 2009

将日期与其他命令一起使用

date 命令最常用于创建包含当前时间和日期的文件名。

下面的命令将以下列格式创建一个 Mysql 备份文件database_name-20190601.sql

mysqldump  database_name > database_name-$(date +%Y%m%d).sql

您也可以在 shell 脚本中使用 date 命令。在下面的例子中，我们分配的输出 date 到 date_now 变量：

date_now=$(date "+%F-%H-%M-%S")

echo $date_now
2019-06-01-15-02-27

显示文件的上次修改时间

date 带有 -r 选项来显示文件的最后修改时间。例如：

date -r /etc/hosts
Tue Jul 24 11:11:48 CEST 2018

如果要修改文件时间戳，请使用该 touch 命令。

设置系统时间和日期

通常不建议使用 date 命令手动设置系统时间和日期，因为在大多数 Linux 发行版中，系统时钟使用 ntp 或 systemd-timesyncd 服务进行同步。

但是，如果要手动设置系统时钟，可以使用 --set= 选项。例如，如果要将日期和时间设置为 2019 年 6 月 1 日下午 5:30 ，则应键入：

date --set="20190601 17:30"

结论

到目前为止，您应该很好地理解了如何使用 Linux date 命令。
