如何在CentOS 7上安装OpenCV

May 9, 2019

OpenCV（开源计算机视觉库）是一个开源计算机视觉库，具有C ++，Python和Java的绑定，并支持所有主要的操作系统。它可以利用多核处理，并具有GPU加速功能，可实现实时操作。

OpenCV用于广泛的应用，包括医学图像分析，拼接街景图像，监控视频，检测和识别人脸，跟踪移动物体，提取3D模型等等。

在本教程中，我们将展示如何在CentOS 7上安装OpenCV。

从CentOS存储库安装OpenCV

OpenCV软件包可从CentOS 7分发存储库获得，但它已经过时了。在撰写本文时，存储库中的版本是2.4.5。

键入以下命令安装OpenCV软件包：

sudo yum install opencv opencv-devel opencv-python

安装完成后，您可以通过运行来验证它：

pkg-config --modversion opencv
2.4.5

或者通过导入Python cv2模块并打印OpenCV版本：

python -c "\
import cv2
print(cv2.__version__)"
2.4.5

从Source安装OpenCV

从源代码构建OpenCV库将允许您拥有最新的可用版本。它将针对您的特定系统进行优化，您可以完全控制构建选项。

要从源安装最新的OpenCV版本，请按照下列步骤操作：

  1. 安装必需和可选的依赖项：

    sudo yum install epel-release git gcc gcc-c++ cmake3 qt5-qtbase-devel python python-devel python-pip cmakesudo yum install python-devel numpy python34-numpy gtk2-devel libpng-devel jasper-devel openexr-devel libwebp-develsudo yum install libjpeg-turbo-devel libtiff-devel  libdc1394-devel tbb-devel eigen3-devel gstreamer-plugins-base-develsudo yum install freeglut-devel mesa-libGL mesa-libGL-devel  boost boost-thread boost-devel libv4l-devel
    1. 创建一个目录，该目录将保存存储库并克隆OpenCV和OpenCV contrib存储库：
    mkdir ~/opencv_build && cd ~/opencv_buildgit clone https://github.com/opencv/opencv.gitgit clone https://github.com/opencv/opencv_contrib.git

    在撰写本文时，github存储库中的默认版本是4.0.0版。如果要安装旧版本的OpenCV，CD既opencv和opencv_contrib目录，并运行git checkout <opencv-version>

  2. 下载完成后，创建一个临时构建目录，然后切换到该目录：

    cd ~/opencv_build/opencv && mkdir build && cd build

    使用以下CMake命令配置OpenCV构建：

    cmake3 -D CMAKE_BUILD_TYPE=RELEASE \
    -D CMAKE_INSTALL_PREFIX=/usr/local \
    -D INSTALL_C_EXAMPLES=ON \
    -D INSTALL_PYTHON_EXAMPLES=ON \
    -D OPENCV_GENERATE_PKGCONFIG=ON \
    -D OPENCV_EXTRA_MODULES_PATH=~/opencv_build/opencv_contrib/modules \
    -D BUILD_EXAMPLES=ON ..

    完成CMake构建系统后，您将看到如下所示的内容：

    -- Configuring done
-- Generating done
-- Build files have been written to: /home/linuxize/opencv_build/opencv/build
    1. 通过运行以下命令启动编译过程。-j根据您的处理器修改标志。如果您不知道处理器的核心数量，可以通过键入来找到它nproc。我的系统有8个核心，所以我使用的是-j8旗帜
    make -j8

    编译可能需要几分钟或更长时间，具体取决于您的系统配置。完成后，您将看到如下内容：

    [100%] Built target example_tutorial_Threshold_inRange
[100%] Linking CXX shared module ../../lib/cv2.so
[100%] Built target opencv_python2
    1. 安装OpenCV：
    sudo make install

  3. 创建符号链接opencv4.pc文件到/usr/share/pkgconfig目录并运行ldconfig以重建库缓存。

    sudo ln -s /usr/local/lib64/pkgconfig/opencv4.pc /usr/share/pkgconfig/sudo ldconfig

    键入以下命令检查OpenCV版本：

    pkg-config --modversion opencv4
    4.0.1
    1. 要启用Python cv2模块运行：
    sudo ln -s /usr/local/lib/python2.7/site-packages/cv2  /usr/lib/python2.7/site-packages/

    导入模块并打印OpenCV版本：

    python -c "\
import cv2
print(cv2.__version__)"
    4.0.1-dev

结论

我们已经向您展示了在CentOS 7服务器上安装OpenCV的两种不同方法。您选择的方法取决于您的要求和偏好。即使从CentOS存储库安装打包版本更容易，从源代码构建OpenCV可以提供更大的灵活性，它应该是您安装OpenCV时的第一个选择。

opencv python centos
