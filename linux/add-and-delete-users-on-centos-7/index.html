在 CentOS 7 中添加和删除用户

2019年2月26日 • 约 700 字 , 预计阅读时间 4 分钟

和所有其他Linux发行版的一样，CentOS是一个多用户操作系统。每个独立用户都可以具有不同的权限级别和各种命令行和GUI应用程序的特定设置。了解如何添加和删除用户是每个Linux用户应该了解的最重要的技能之一。在本教程中，我们将解释如何在CentOS 7系统上添加和删除用户。

先决条件

您需要以 root 用户或具有 sudo 权限的用户才能创建和删除用户。

在 CentOS 中添加用户

在 CentOS 中，您可以使用 useradd 命令行实用程序创建新的用户帐户。

要创建一个名为的新用户帐户，username您将运行：

sudo adduser username

上面的命令不显示输出。它将创建新用户的主目录 /home/username，并将文件从 /etc/skel 目录复制到用户的主目录。在主目录中，用户可以编写，编辑和删除文件和目录。

接下来，您需要为新用户设置密码，以便用户可以登录。为此，请使用以下passwd命令：

sudo passwd username

系统将提示您输入并确认密码。确保使用强密码。

Changing password for user username.
New password: 
Retype new password: 
passwd: all authentication tokens updated successfully.

默认情况下，在CentOS上，组 wheel 的成员被授予 sudo 访问权限。

如果您希望新创建的用户具有管理权限，请将用户添加到 wheel 组：

sudo usermod -aG wheel username

如果您以 root 身份登录，则不必使用 sudo 预先添加每个命令。

在 CentOS 中删除用户

如果不再需要用户帐户，则可以使用 deluser 命令行工具将其删除。

如果要删除用户而不删除用户文件，请运行：

sudo userdel username

如果要删除并且用户的主目录和邮件假脱机使用该-r标志：

sudo userdel -r username

成功时，userdel 命令不会产生任何输出。

如果用户被授予 sudo 权限，则将删除 wheel 组，以及用户所属的任何其他组。

结论

在本教程中，您学习了如何在 CentOS 中添加和删除用户。同样的，相同的命令也适用于任何其他 Linux 发行版。
