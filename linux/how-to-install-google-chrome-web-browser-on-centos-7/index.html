如何在CentOS 7上安装Google Chrome Web浏览器

2019年5月9日 • 约 1000 字 , 预计阅读时间 4 分钟

谷歌浏览器是世界上使用最广泛的网络浏览器。它是为现代网络构建的快速，易用且安全的浏览器。

Chrome不是一个开源浏览器，它不包含在CentOS存储库中。它基于Chromium，这是一个开源浏览器，可在EPEL存储库中找到。

本教程讲授如何在CentOS 7上安装Google Chrome网络浏览器。相同的说明适用于任何基于RHEL的发行版，包括Fedora和Scientific Linux。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

在CentOS上安装Google Chrome

按照下面列出的步骤在CentOS系统上安装Google Chrome：

  1. 首先打开终端并.rpm使用以下wget命令下载最新的Google Chrome 包：

    wget https://dl.google.com/linux/direct/google-chrome-stable_current_x86_64.rpm

  2. 下载文件后，键入以下命令在CentOS 7系统上安装Google Chrome：

    sudo yum localinstall google-chrome-stable_current_x86_64.rpm

    上面的命令将提示您输入用户密码，然后它将安装Chrome和所有其他必需的软件包。

启动Google Chrome

现在您已经在CentOS系统上安装了Google Chrome，您可以通过输入google-chrome &或点击Google Chrome图标（Applications -> Internet -> Google Chrome）从命令行启动它：

当您第一次启动Google Chrome时，会出现如下窗口，询问您是否要将Google Chrome设置为默认浏览器，并向Google发送使用统计信息和崩溃报告：

根据您的偏好选择，然后单击OK继续。

谷歌浏览器将打开，您将看到默认的Chrome欢迎页面。

此时，您已在CentOS计算机上安装了Chrome。您可以使用自己的Google帐户登录Chrome，以便在所有设备上同步书签，历史记录，密码和其他设置。

正在更新Google Chrome

在安装过程中，官方Google存储库将添加到您的系统中。您可以使用cat命令验证文件内容：

cat /etc/yum.repos.d/google-chrome.repo

[google-chrome]
name=google-chrome
baseurl=http://dl.google.com/linux/chrome/rpm/stable/x86_64
enabled=1
gpgcheck=1
gpgkey=https://dl.google.com/linux/linux_signing_key.pub

这可确保在通过桌面标准软件更新工具发布新版本时，您的Google Chrome安装将自动更新。

结论

在本教程中，我们向您展示了如何在CentOS 7桌面计算机上安装Google Chrome。如果您之前使用过其他浏览器，例如Firefox或Opera，则可以将书签和设置导入Chrome。
