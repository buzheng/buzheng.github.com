如何在 Raspberry Pi 上安装 Plex Media Server

2019年6月15日 • 约 1400 字 , 预计阅读时间 5 分钟

Raspberry Pi 可用于许多不同的项目。Raspberry Pi 最受欢迎的用例之一是将 Raspberry Pi 变成家庭媒体中心。

Plex 是一种流行的流媒体服务器，可让您随时随地组织视频，音乐和照片集，并将其流式传输到您的所有设备。

在本教程中，我们将向您展示如何在 Raspberry Pi 上安装 Plex Media Server。

先决条件

我们假设您在 Raspberry Pi 上安装了 Raspbian。Plex Media Server 不需要图形界面，因此我们建议使用 Raspbian Stretch Lite 映像并启用 S​​SH。这样，Raspberry Pi 将具有更多可用的处理能力和内存来运行 Plex 媒体服务器。

建议使用外部存储来存储媒体文件。备用 SSD 或 USB 驱动器可以完成这项工作。

安装 Plex Media Server

在 Raspberry Pi 上安装和管理 Plex Media Server 的最简单方法是使用官方的 Plex 存储库。它不需要技术知识，安装和配置媒体服务器不应超过 20 分钟。

通过 SSH 登录您的 Raspberry Pi，并按照以下步骤安装 Plex Media Server：

  1. 首先更新软件包索引并安装通过 HTTPS 启用新存储库所需的依赖项：

        sudo apt update
        sudo apt install apt-transport-https ca-certificates curl

  2. 接下来，通过运行以下命令，导入存储库的 GPG 密钥并将 APT 存储库添加到系统的软件存储库列表中：

        curl https://downloads.plex.tv/plex-keys/PlexSign.key | sudo apt-key add -echo deb https://downloads.plex.tv/repo/deb public main | sudo tee /etc/apt/sources.list.d/plexmediaserver.list

  3. 启用 Plex 存储库后，更新程序包列表并安装最新版本的 Plex Media Server：

        sudo apt updatesudo apt install plexmediaserver

    安装可能需要几分钟。

  4. 要验证 Plex 服务是否正在运行，请键入：

        sudo systemctl status plexmediaserver

    输出应该如下所示：

        ● plexmediaserver.service - Plex Media Server
       Loaded: loaded (/lib/systemd/system/plexmediaserver.service; enabled; vendor preset: enabled)
      Active: active (running) since Sun 2019-05-26 0809 EDT; 27min ago
      Process: 11470 ExecStartPre=/bin/sh -c /usr/bin/test -d "${PLEX_MEDIA_SERVER_APPLICATION_SUPPORT_DIR}" || /bin/mkdir -p "${PLEX_MEDIA_SERVER_APPLICATION_SUPPORT_DIR}" (code=exited, status=0/SUCCESS)
    Main PID: 11474 (sh)
      CGroup: /system.slice/plexmediaserver.service

配置 Plex 媒体服务器

在启动 Plex 设置向导之前，让我们创建将存储 Plex 媒体文件的目录：

sudo mkdir -p /opt/plexmedia/{movies,series}

Plex Media Server 作为用户 plex 运行，该用户必须具有对媒体文件和目录的读取和执行权限。要设置正确的所有权，请运行以下命令。

sudo chown -R plex: /opt/plexmedia

您可以选择存储媒体文件的任何位置，只需确保设置正确的权限即可。

现在我们可以继续进行服务器配置了。打开浏览器，输 http://YOUR_SERVER_IP:32400/web，您将看到以下屏幕：

要使用 Plex Media Server，您需要创建一个帐户。

按 Google，Facebook 或电子邮件按钮创建免费的 Plex 帐户。如果您想要访问高级功能，可以购买 Plex Pass 计划。

注册后，您将被重定向到一个页面，其中包含有关 Plex 如何工作的信息，如下所示：

单击 Got it 按钮。

在下一个屏幕上输入您的 Plex 服务器名称，Allow me to access my media outside my home 选中该复选框，然后单击 Next。

下一步是添加媒体库。单击 Add Library 按钮。

弹出窗口显示时，选择电影作为库类型并单击 Next。

在下一步中，单击 Browse for media folder 并添加包含 Movies 媒体文件的目录的路径（在我们的示例中） /opt/plexmedia/movies。

单击 Add 按钮，然后单击 Add Library。

您可以根据需要添加任意数量的库。

单击 Next，然后 Done 您将被重定向到 Plex Web 仪表板。

现在您已经完成了设置向导，您可以开始探索 Plex 选项及其可以执行的所有操作。

更新 Plex 媒体服务器

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Plex Media Server 软件包：

sudo apt update
sudo apt upgrade

结论

在本文中，您学习了如何在 Raspberry Pi 上安装 Plex Media Server 以及如何创建媒体库。

您现在可以在 Android，iPhone，智能电视，Xbox，Roku 或任何其他受支持的设备上安装 Plex 应用程序。您可以在 Plex 下载页面上找到支持的应用程序和设备列表，也可以从设备的应用程序商店中安装该应用程序。

有关更多信息，请访问官方Plex 快速入门指南和Plex 文档页面。
