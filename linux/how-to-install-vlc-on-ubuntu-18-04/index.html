如何在 Ubuntu 18.04 上安装 VLC Media Player

April 14, 2019

VLC 是最受欢迎的开源多媒体播放器之一。它是跨平台的，几乎可以播放所有多媒体文件以及 DVD ，音频 CD 和不同的流媒体协议。

本教程介绍如何在 Ubuntu 18.04 上安装 VLC 媒体播放器。相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。

先决条件

您需要以具有 sudo 访问权限的用户身份登录才能在 Ubuntu 系统上安装软件包。

在 Ubuntu 上安装 VLC Media Player

官方 Ubuntu 18.04 存储库中包含的 deb 软件包总是落后于最新版本的 VLC 。

在 Ubuntu 18.04 上安装 VLC 的推荐方法是使用 snappy 打包系统。这样，您将始终拥有最新的主要 VLC 版本，其中包含安装在桌面上的所有编解码器，安全性和关键错误修复以及光学媒体支持`。

在撰写本文时， VLC 媒体播放器的最新稳定版本是 3.0.6 版。

使用 Ctrl+Alt+T 键盘快捷方式打开终端并键入以下命令在 Ubuntu 桌面上安装 VLC ：

sudo sudo snap install vlc

安装完成后，您将看到以下输出：

vlc 3.0.6 from VideoLAN✓ installed

而已。 VLC 媒体播放器已安装在您的 Ubuntu 系统上，您可以通过单击 VLC 媒体播放器图标（ Activities -> VLC Media Player ）启动播放器。

首次启动 VLC 时，将显示如下窗口，其中包含有关 VLC 隐私和网络访问策略的信息。

将 VLC 设置为默认媒体播放器

要将 VLC 设置为 Ubuntu 18.04 中的默认媒体播放器，请打开“Activities”窗口，搜索“默认应用程序”并单击它。

当“默认应用程序”窗口打开时，在音乐下拉列表中选择“ VLC 媒体播放器”：

结论

您已经学习了如何在 Ubuntu 18.04 桌面上安装 VLC 。您现在可以浏览新的媒体播放器并开始观看视频流或收听您喜爱的艺术家。

要查找有关 VLC 的更多信息，请访问 VideoLAN 的“文档”页面。
