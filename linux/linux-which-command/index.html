Linux which 命令

June 5, 2019

在本教程中，我们将介绍 Linux 中的 which 命令。

当您在终端提示符中键入可执行文件名(命令)时，which 命令用于确定给定可执行文件的位置。该命令在 PATH 环境变量中列出的目录中搜索指定为参数的可执行文件。

什么是 PATH

在 Linux 中， PATH 是一个环境变量，它告诉 shell 和其他程序搜索可执行文件的目录。它由一系列冒号分隔的绝对路径列表组成，这些路径包含可执行文件。

要查看 PATH 变量的内容，使用 echo 命令与 $PATH 作为参数：

echo $PATH

输出结果如下所示：

/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin

如何使用 which 命令

该 which 命令的语法如下：

which [OPTIONS] FILE_NAME...

例如，要查找 ping 命令的完整路径，请键入以下内容：

which ping

输出将是这样的：

/bin/ping

您还可以为 which 命令提供多个参数：

which netcat uptime

输出将包括 netcat 和 uptime 可执行文件的完整路径：

/bin/netcat
/usr/bin/uptime

搜索从左到右进行，如果在 PATH 路径变量中列出的目录中找到多个匹配项， which 则只打印第一个。要打印所有匹配项，请使用以下 -a 选项：

which -a touch

输出将显示 touch 命令的两个完整路径：

/usr/bin/touch
/bin/touch

通常，其中一个可执行文件只是另一个的符号链接，但在某些情况下，您可能在不同位置安装了相同命令的两个版本，或者使用相同名称的完全不同的命令。

结论

该 which 命令用于在 PATH 环境变量指定的目录中搜索可执行命令来查找命令的位置。

which terminal
