如何在CentOS 7上设置或更改时区

2019年5月9日 • 约 1000 字 , 预计阅读时间 4 分钟

在CentOS上，系统的时区在安装期间设置，但可以在以后轻松更改。

对于许多与系统相关的任务和流程，使用正确的时区非常重要。例如，cron守护程序使用系统的时区执行cron作业，日志文件中的时间戳基于同一系统的时区。

本教程介绍如何在CentOS 7上设置或更改时区。

先决条件

您需要以具有sudo权限的用户身份登录才能更改系统的时区。

检查当前时区

在CentOS和其他现代Linux发行版中，您可以使用该timedatectl命令显示和设置当前系统的时间和时区。

timedatectl

如下面的输出所示，系统的时区设置为UTC：

      Local time: Wed 2019-02-06 22:43:42 UTC
  Universal time: Wed 2019-02-06 22:43:42 UTC
        RTC time: Wed 2019-02-06 22:43:42
       Time zone: Etc/UTC (UTC, +0000)
     NTP enabled: no
NTP synchronized: yes
 RTC in local TZ: no
      DST active: n/a

通过符号链接/etc/localtime到/usr/share/zoneinfo目录中的二进制时区标识符来配置系统时区。因此，检查时区的另一个选项是使用ls命令显示符号链接指向的路径：

ls -l /etc/localtime
lrwxrwxrwx. 1 root root 29 Dec 11 09:25 /etc/localtime -> ../usr/share/zoneinfo/Etc/UTC

更改CentOS中的时区

在更改时区之前，您需要找到要使用的时区的长名称。时区命名约定通常使用“区域/城市”格式。

要列出所有可用时区，您可以列出目录中的文件/usr/share/zoneinfo或使用该timedatectl命令。

timedatectl list-timezones
...
America/Tijuana
America/Toronto
America/Tortola
America/Vancouver
America/Whitehorse
America/Winnipeg
...

确定哪个时区对您的位置准确无误后，请以sudo用户身份运行以下命令：

sudo timedatectl set-timezone your_time_zone

例如，要将系统的时区更改为America/Toronto：

sudo timedatectl set-timezone America/Toronto

运行timedatectl命令以验证更改：

timedatectl
      Local time: Wed 2019-02-06 17:47:10 EST
  Universal time: Wed 2019-02-06 22:47:10 UTC
        RTC time: Wed 2019-02-06 22:47:10
       Time zone: America/Toronto (EST, -0500)
     NTP enabled: no
NTP synchronized: yes
 RTC in local TZ: no
      DST active: no
 Last DST change: DST ended at
                  Sun 2018-11-04 01:59:59 EDT
                  Sun 2018-11-04 01:00:00 EST
 Next DST change: DST begins (the clock jumps one hour forward) at
                  Sun 2019-03-10 01:59:59 EST
                  Sun 2019-03-10 03:00:00 EDT

通过创建符号链接更改时区

如果您运行的是较旧版本的CentOS，并且timedatectl系统上没有该命令，则可以通过符号链接更改目录中的时区来更改时/etc/localtime区/usr/share/zoneinfo。

删除当前/etc/localtime文件或符号链接：

sudo rm -rf /etc/localtime

确定要配置的时区并创建符号链接：

sudo ln -s /usr/share/zoneinfo/America/Toronto /etc/localtime

您可以通过列出/etc/localtime文件或发出date命令来验证它：

date
Wed Feb  6 17:52:58 EST 2019

结论

在本指南中，我们向您展示了如何更改CentOS系统的时区。
