如何在Ubuntu 18.04上安装Skype

May 9, 2019

Skype是世界上最受欢迎的通信应用程序之一，它允许您免费进行在线音频和视频通话，并为全球的移动电话和固定电话提供价格合理的国际通话服务。

Skype不是一个开源应用程序，它不包含在Ubuntu存储库中。

在本教程中，我们将向您展示如何在Ubuntu 18.04上安装Skype。相同的说明适用于Ubuntu 16.04和任何其他基于Debian的发行版，包括Debian，Kubuntu，Linux Mint和Elementary OS。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

在Ubuntu上安装Skype

完成以下步骤以在Ubuntu上安装Skype。

1.下载Skype

使用Ctrl+Alt+T键盘快捷键或单击终端图标打开终端。

[](https://www.skype.com/en/get-skype/)`.deb`使用以下[wget命令](/linux/wget-command-examples/)[下载最新的Skype](https://www.skype.com/en/get-skype/)软件包：

wget https://go.skype.com/skypeforlinux-64.deb

2.安装Skype

下载完成后，通过以具有sudo权限的用户身份运行以下命令来安装Skype：

sudo apt install ./skypeforlinux-64.deb

系统将提示您输入密码。输出看起来像这样：

3.启动Skype

现在您已在Ubuntu桌面上安装了Skype，您可以通过键入skypeforlinux或单击Skype图标（Activities -> Skype）从命令行启动它：

当您第一次启动Skype时，将出现如下所示的窗口：

从这里，您可以使用您的Microsoft帐户登录Skype，并开始与您的朋友和家人聊天和交谈。

更新Skype

在安装过程中，官方Skype存储库将添加到您的系统中。使用cat命令验证文件内容：

cat /etc/apt/sources.list.d/skype-stable.list
deb [arch=amd64] https://repo.skype.com/deb stable main

这可确保在通过桌面标准软件更新工具发布新版本时，您的Skype安装将自动更新。

结论

在本教程中，您已经学习了如何在Ubuntu 18.04桌面上安装Skype。
