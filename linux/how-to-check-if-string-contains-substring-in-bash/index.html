在 Bash 中如何检查字符串是否包含的子串

2019年7月21日 • 约 600 字 , 预计阅读时间 4 分钟

在 Bash 中使用字符串时最常见的操作之一是确定字符串是否包含另一个字符串。

在本文中，我们将向您展示检查字符串是否包含子字符串的几种方法。

使用通配符

最简单的方法就是使用星号通配符 * 包裹子字符串，并与字符串比较。通配符是用于表示零个，一个或多个字符的符号。

如果测试返回 true ，则子字符串包含在字符串中。

在下面的示例中，我们使用 if 语句和相等运算符 (==) 来检查是字符串 STR 中是否包含子字符串 SUB：

#!/bin/bash

STR='GNU/Linux is an operating system'
SUB='Linux'
if [[ "$STR" == *"$SUB"* ]]; then
  echo "It's there."
fi

执行时脚本将输出：

It's there.

使用 case 运算符

除了使用 if 语句，您也可以使用 case 语句来检查字符串是否包含另一个字符串。

#!/bin/bash

STR='GNU/Linux is an operating system'
SUB='Linux'

case $STR in

  *"$SUB"*)
    echo -n "It's there."
    ;;
esac

使用正则表达式运算符

确定字符串中是否出现指定子字符串的另一个选项是使用正则表达式运算符 =~ 。使用此运算符时，右字符串被视为正则表达式。

.* 表示零或多个除了换行符之外的任何字符。

#!/bin/bash

STR='GNU/Linux is an operating system'
SUB='Linux'

if [[ "$STR" =~ .*"$SUB".* ]]; then
  echo "It's there."
fi

该脚本将 echo 以下内容：

It's there.

使用 Grep

使用 grep 命令也可以用来查找其他子串。

在下面的示例中，我们将字符串 $STR 作为输入传递给 grep 并检查字符串 $SUB 是否在输入字符串中找到。该命令将根据情况返回 true 或 false。

#!/bin/bash

STR='GNU/Linux is an operating system'
SUB='Linux'

if grep -q "$SUB" <<< "$STR"; then
  echo "It's there"
fi

选项-q 告诉 grep 要静默执行，省略输出。

结论

检查字符串是否包含子字符串是 Bash 脚本中最基本和最常用的操作之一。

阅读本教程后，您应该很好地理解如何测试字符串是否包含另一个字符串。您还可以使用其他命令，例如 awk 或 sed 用于测试。
