如何在Ubuntu 18.04上安装Steam

2019年5月9日 • 约 600 字 , 预计阅读时间 4 分钟

Steam是Valve Corporation开发的用于购买和玩视频游戏的跨平台娱乐平台。它使您可以访问数以千计的游戏，并允许您结识新朋友。

在本教程中，我们将向您展示如何在Ubuntu 18.04上安装Steam。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

先决条件

您需要以具有sudo权限的用户身份登录才能在Ubuntu系统上安装软件包。

在Ubuntu上安装Steam

执行以下步骤在Ubuntu桌面上安装Steam：

  1. 首先启用包含不符合Ubuntu许可证策略的软件的Multiverse存储库：

    sudo add-apt-repository multiverse

    'multiverse' distribution component enabled for all sources

  2. 接下来，steam键入以下命令安装包：

    sudo apt install steam

  3. 安装完成后，您可以通过键入steam或单击Steam图标（活动 - > Steam）从命令行启动它：

  4. 当您第一次启动Steam时，它会将自己更新为最新版本。这可能需要几分钟。

  5. 更新完成后，Steam将启动。

    从这里您可以登录您的Steam帐户或创建一个新帐户。

结论

您已经学习了如何在Ubuntu 18.04上安装Steam。您现在可以开始购买和玩视频游戏了。
