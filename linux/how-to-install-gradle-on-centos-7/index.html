如何在CentOS 7上安装Gradle

May 9, 2019

Gradle是一个主要用于Java项目的开源构建自动化系统。它结合了Ant和Maven的最佳功能。与使用XML进行脚本编写的前辈不同，Gradle使用Groovy，这是一种动态的，面向对象的Java平台编程语言，用于定义项目和构建脚本。

本教程概述了在CentOS 7系统上安装最新版Gradle所需的步骤。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

在CentOS上安装Gradle

以下部分提供有关如何在CentOS 7上安装Gradle的信息。我们将从其官方网站下载最新版本的Gradle。

1.安装OpenJDK

Gradle需要安装Java JDK或JRE版本7或更高版本。

使用以下命令安装OpenJDK 8软件包：

sudo yum install java-1.8.0-openjdk-devel

通过打印Java版本来验证Java安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_191"
OpenJDK Runtime Environment (build 1.8.0_191-b12)
OpenJDK 64-Bit Server VM (build 25.191-b12, mixed mode)

如果您更喜欢Oracle Java而不是OpenJDK，请按照以下说明安装Oracle Java。

2.下载Gradle

在撰写本文时，最新版本的Gradle是5.0。在继续下一步之前，您应该检查Gradle版本页面以查看是否有更新的版本。

首先/tmp使用以下wget命令在目录中下载Gradle Binary-only zip文件：

wget https://services.gradle.org/distributions/gradle-5.0-bin.zip -P /tmp

下载完成后，解压缩目录中的zip文件/opt/gradle：

sudo unzip -d /opt/gradle /tmp/gradle-5.0-bin.zip

通过列出/opt/gradle/gradle-5.0目录来验证是否已提取Gradle文件：

ls /opt/gradle/gradle-5.0
bin  getting-started.html  init.d  lib  LICENSE  media  NOTICE

3.设置环境变量

下一步是配置PATH环境变量以包含Gradle bin目录。为此，请打开文本编辑器并创建一个名为目录gradle.sh内的新文件/etc/profile.d/。

sudo nano /etc/profile.d/gradle.sh

粘贴以下配置：

/etc/profile.d/gradle.sh

export GRADLE_HOME=/opt/gradle/gradle-5.0
export PATH=${GRADLE_HOME}/bin:${PATH}

保存并关闭文件。此脚本将在shell启动时获取。

键入以下命令使脚本可执行：

sudo chmod +x /etc/profile.d/gradle.sh

使用以下命令加载环境变量：

source /etc/profile.d/gradle.sh

4.验证Gradle安装

要验证Gradle是否已正确安装，请使用gradle -v将显示Gradle版本的命令：

gradle -v

您应该看到如下内容：

Welcome to Gradle 5.0!

Here are the highlights of this release:
 - Kotlin DSL 1.0
 - Task timeouts
 - Dependency alignment aka BOM support
 - Interactive `gradle init`

For more details see https://docs.gradle.org/5.0/release-notes.html

------------------------------------------------------------
Gradle 5.0
------------------------------------------------------------

Build time:   2018-11-26 11:48:43 UTC
Revision:     7fc6e5abf2fc5fe0824aec8a0f5462664dbcd987

Kotlin DSL:   1.0.4
Kotlin:       1.3.10
Groovy:       2.5.4
Ant:          Apache Ant(TM) version 1.9.13 compiled on July 10 2018
JVM:          1.8.0_191 (Oracle Corporation 25.191-b12)
OS:           Linux 3.10.0-862.14.4.el7.x86_64 amd64

而已。最新版本的Gradle现已安装在您的CentOS系统上。

结论

您已经在CentOS 7上成功安装了Gradle。您现在可以访问Gradle官方文档页面，了解如何开始使用Gradle。
