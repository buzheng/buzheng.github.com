如何在Ubuntu 18.04上安装Apache Maven

2019年5月9日 • 约 1500 字 , 预计阅读时间 5 分钟

Apache Maven是一个免费的开源项目管理和理解工具，主要用于Java项目。Maven使用项目对象模型（POM），它本质上是一个XML文件，包含有关项目，配置详细信息，项目依赖项等的信息。

在本教程中，我们将向您展示在Ubuntu 18.04上安装Apache Maven的两种不同方法。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Linux Mint，Kubuntu和Elementary OS。

官方Ubuntu存储库包含可以与apt软件包管理器一起安装的Maven软件包。这是在Ubuntu上安装Maven的最简单方法，但是存储库中包含的版本可能落后于最新版本的Maven。

要安装最新版本的Maven，请按照本文第二部分提供的说明从我们的官方网站下载Maven。

选择最适合您环境的安装方法。

先决条件

为了能够在Ubuntu系统上安装软件包，您必须以具有sudo权限的用户身份登录。

使用Apt在Ubuntu上安装Apache Maven

在Ubuntu上安装Maven apt是一个简单，直接的过程。

  1. 首先更新包索引：

    sudo apt update

  2. 接下来，键入以下命令安装Maven：

    sudo apt install maven

  3. 通过运行mvn -version命令验证安装：

    mvn -version

    输出应该如下所示：

    Apache Maven 3.5.2
Maven home: /usr/share/maven
Java version: 10.0.2, vendor: Oracle Corporation
Java home: /usr/lib/jvm/java-11-openjdk-amd64
Default locale: en_US, platform encoding: ISO-8859-1
OS name: "linux", version: "4.15.0-36-generic", arch: "amd64", family: "unix"

而已。Maven现已安装在您的系统上，您可以开始使用它。

安装最新版本的Apache Maven

以下部分提供了有关如何在Ubuntu 18.04上安装最新Apache Maven版本的分步说明。我们将从他们的官方网站下载最新版本的Apache Maven。

1.安装OpenJDK

Maven 3.3+需要安装JDK 1.7或更高版本。我们将安装OpenJDK，它是Ubuntu 18.04中的默认Java开发和运行时。

Java的安装非常简单。首先更新包索引：

sudo apt update

输入以下命令安装OpenJDK包：

sudo apt install default-jdk

通过运行以下命令验证安装：

java -version

输出应该如下所示：

openjdk version "10.0.2" 2018-07-17
OpenJDK Runtime Environment (build 10.0.2+13-Ubuntu-1ubuntu0.18.04.2)
OpenJDK 64-Bit Server VM (build 10.0.2+13-Ubuntu-1ubuntu0.18.04.2, mixed mode)

如果您更喜欢Oracle Java而不是OpenJDK，请按照以下说明安装Oracle Java。

2.下载Apache Maven

在撰写本文时，最新版本的Apache Maven是3.6.0。在继续下一步之前，您应该检查Maven下载页面以查看是否有更新的版本。

首先/tmp使用以下wget命令在目录中下载Apache Maven ：

wget https://www-us.apache.org/dist/maven/maven-3/3.6.0/binaries/apache-maven-3.6.0-bin.tar.gz -P /tmp

下载完成后，在目录中解压缩归档/opt：

sudo tar xf /tmp/apache-maven-*.tar.gz -C /opt

为了更好地控制Maven版本和更新，我们将创建一个maven指向Maven安装目录的符号链接：

sudo ln -s /opt/apache-maven-3.6.0 /opt/maven

稍后如果要升级Maven安装，只需解压缩较新版本并更改符号链接以指向最新版本即可。

3.设置环境变量

接下来，我们需要设置环境变量。为此，请打开文本编辑器并创建一个名为目录mavenenv.sh内的新文件/etc/profile.d/。

sudo nano /etc/profile.d/maven.sh

粘贴以下配置：

/etc/profile.d/maven.sh

export JAVA_HOME=/usr/lib/jvm/default-java
export M2_HOME=/opt/maven
export MAVEN_HOME=/opt/maven
export PATH=${M2_HOME}/bin:${PATH}

保存并关闭文件。此脚本将在shell启动时获取。

键入以下命令使脚本可执行：

sudo chmod +x /etc/profile.d/maven.sh

最后使用以下命令加载环境变量：

source /etc/profile.d/maven.sh

4.验证安装

要验证Maven是否已正确安装，请使用mvn -version将打印Maven版本的命令：

mvn -version

您应该看到如下内容：

Apache Maven 3.6.0 (97c98ec64a1fdfee7767ce5ffb20918da4f719f3; 2018-10-24T18:41:47Z)
Maven home: /opt/maven
Java version: 10.0.2, vendor: Oracle Corporation, runtime: /usr/lib/jvm/java-11-openjdk-amd64
Default locale: en_US, platform encoding: ISO-8859-1
OS name: "linux", version: "4.15.0-36-generic", arch: "amd64", family: "unix"

而已。最新版本的Maven现已安装在您的Ubuntu系统上。

结论

您已在Ubuntu 18.04上成功安装了Apache Maven。您现在可以访问Apache Maven官方文档页面，了解如何开始使用Maven。
