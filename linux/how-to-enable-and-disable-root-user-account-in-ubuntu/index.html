如何在 Ubuntu 中启用和禁用 root 帐户

2019年3月7日 • 约 800 字 , 预计阅读时间 4 分钟

出于安全原因，Ubuntu Linux 默认情况下禁用了 root 帐户。作为新的 Ubuntu 用户，您可能想知道如何以 root 用户身份登录 Ubuntu 系统或默认 root 密码是什么。

本教程介绍如何在 Ubuntu Linux 中启用和禁用 root 用户帐户。

sudo 用户

Ubuntu 鼓励用户通过使用名为 sudo 的工具向常规用户授予管理权限来执行系统管理任务。 sudo 允许授权用户以另一个用户（通常是 root 用户）的身份运行程序。

默认情况下，在 Ubuntu 系统上，组 sudo 的成员被授予 sudo 访问权限。由 Ubuntu 安装程序创建的初始用户已经是 sudo 组的成员，因此您登录的用户已获得管理权限。

如果要向其他用户授予 sudo 访问权限，只需将用户添加到 sudo 组：

usermod -aG sudo username

要临时提升普通用户的权限为 root 用户权限，请在运行的命令前面加 sudo 前缀：

sudo some-command

第一次在会话中使用 sudo 时，系统将提示您输入用户密码。

如果要在不输入密码的情况下运行具有 sudo 权限的命令，则需要编辑该 sudoers 文件。请运行 visudo 以下命令：

sudo visudo

visudo 命令将使用您最喜欢的命令行文本编辑器打开文件 /etc/sudoers 。请在 /etc/sudoers 文件中添加以下行（注意：要替换 username 为您的用户名）：

username ALL=(ALL) NOPASSWD: ALL

在 Ubuntu 中启用 root 用户帐户

如果由于某种原因需要启用 root 帐户，则只需为 root 用户设置密码即可。在 Ubuntu 和其他 Linux 发行版中，您可以使用 passwd 命令设置或更改用户帐户的密码。

Ubuntu 的普通用户，只能更改自己的密码。登录的用户必须具有 sudo 权限才能设置 root 密码。

要在 Ubuntu 中启用 root 帐户，请运行以下命令：

sudo passwd root

系统将提示您输入并确认新的 root 密码：

Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully

键入时，屏幕上不显示密码。

到此为止，您已成功启用 root 帐户。您现在可以使用新密码以 root 用户身份登录 Ubuntu 计算机。

在 Ubuntu 中禁用 Root 用户帐户

如果您以前在 Ubuntu 中启用了 root 用户，现在要禁用它，只需将 root 密码设置为 expire 即可。

要禁用 root 帐户密码，请使用以下命令：

sudo passwd -l root

结论 在本教程中，您学习了如何在 Ubuntu 中启用和禁用 root 用户帐户。

设置密码时，请确保使用强大且唯一的密码。拥有强密码是您帐户安全性的最重要保障。强密码通常至少包含16个字符，至少使用一个大写字母，一个小写字母，一个数字和一个特殊字符。
