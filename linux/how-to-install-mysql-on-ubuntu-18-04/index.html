如何在 Ubuntu 18.04 上安装 MySQL

March 24, 2019

MySQL 是最流行的开源关系数据库管理系统。它快速，易于使用，可扩展，是流行 LAMP 和 LEMP 堆栈应用的组成部分。

在本教程中，我们将向您展示如何在 Ubuntu 18.04 计算机上安装和配置更安全的 MySQL 。

先决条件

确保以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 MySQL

在撰写本文时，官方 Ubuntu 存储库中提供的 MySQL 的最新版本是 MySQL 5.7 版。

要在 Ubuntu 服务器上安装 MySQL ，请按照以下步骤操作：

  1. 首先，输入以下内容来更新 apt 包索引：

    sudo apt update
    1. 然后使用以下命令安装 MySQL ：
    sudo apt install mysql-server

  2. 安装完成后， MySQL 服务将自动启动。要检查 MySQL 服务器是否正在运行，请输入：

    sudo systemctl status mysql

    ● mysql.service - MySQL Community Server
  Loaded: loaded (/lib/systemd/system/mysql.service; enabled; vendor preset: enabled)
  Active: active (running) since Wed 2018-06-20 11:30:23 PDT; 5min ago
Main PID: 17382 (mysqld)
    Tasks: 27 (limit: 2321)
  CGroup: /system.slice/mysql.service
          `-17382 /usr/sbin/mysqld --daemonize --pid-file=/run/mysqld/mysqld.pid

保护 MySQL

MySQL 服务器包附带一个调用的脚本 mysql_secure_installation ，可以执行多个与安全相关的操作。

输入以下命令运行脚本：

sudo mysql_secure_installation

系统将要求您配置密码校验插件用于测试 MySQL 用户密码强度并提高安全性。密码校验策略有三个级别，低，中和强。 如果你不想来设置验证密码插件，直接按下 ENTER 即可。

在下一个提示符下，将要求您为 MySQL root 用户设置密码。执行此操作后，脚本还会要求您删除匿名用户，限制 root 用户本地访问并删除测试数据库。你应该对所有问题回答 “Y”（是）。

以 root 身份登录

要从命令行与 MySQL 服务器交互，您可以使用 MySQL 客户端程序，MySQL 客户端程序作为 MySQL 服务器程序包的依赖项已经安装。

在运行 MySQL 5.7 （及更高版本）的 Ubuntu 系统中，默认情况下， root 用户通过插件 auth_socket 进行身份验证。

这个 auth_socket 插件对来自 localhost 通过 Unix 套接字文件连接的用户进行身份验证。这意味着您无法通过提供密码以 root 身份进行身份验证。

以 root 用户的身份登录 MySQL 服务器：

sudo mysql

您将看到 My​​SQL shell ，如下所示：

Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 8
Server version: 5.7.22-0ubuntu18.04.1 (Ubuntu)

affiliates. Other names may be trademarks of their respective

Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

如果您想从外部程序以 root 用户身份登录 MySQL 服务器，例如 phpMyAdmin， 您有两个方法。

第一个是将身份验证方法更改 auth_socket 为 mysql_native_password 。您可以通过运行以下命令来执行此操作：

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'very_strong_password';
FLUSH PRIVILEGES;

第二个方法是创建一个可以访问所有数据库的新管理用户：

GRANT ALL PRIVILEGES ON *.* TO 'administrator'@'localhost' IDENTIFIED BY 'very_strong_password';

现在您的 MySQL 服务器已启动并运行，并且您知道如何从命令行连接到 MySQL 服务器，您可能需要阅读以下指南：
