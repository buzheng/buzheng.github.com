如何在 Linux 中查看用户列表

2019年4月2日 • 约 1300 字 , 预计阅读时间 5 分钟

您是否想要列出 Linux 系统中的所有用户或计算系统中的用户数？有命令可以创建用户，删除用户，列出登录用户，但是列出 Linux 中所有用户的命令是什么？

本教程将向您展示如何在 Linux 系统中列出用户。

使用 /etc/passwd 文件获取所有用户列表

本地用户信息存储在 /etc/passwd 文件中。此文件中的每一行代表一个用户的登录信息。要打开文件，您可以使用 cat 或 less ：

less /etc/passwd

从上面的输出中可以看出，每行有七个以冒号分隔的字段，其中包含以下信息：

  • 用户名
  • 加密密码 (x 表示密码存储在 /etc/shadow 文件中)
  • 用户 ID 号 (UID)
  • 用户的组 ID 号 (GID)
  • 用户的全名 (GECOS)
  • 用户主目录
  • 登录 shell （默认为 /bin/bash)

如果您只想显示您可以使用的用户名， awk 或者 cut 只打印包含用户名的第一个字段的命令：

awk -F: '{ print $1}' /etc/passwd

cut -d: -f1 /etc/passwd

root
daemon
bin
sys
sync
...
...
sshd
vagrant
jack
anne

使用 getent 命令获取所有用户的列表

getent 命令用来显示在 /etc/nsswitch.conf 文件中配置的数据库中的条目，其中包括 passwd 数据库，该数据库可用于查询所有的用户列表。

要获取所有 Linux 用户的列表，请键入以下命令：

getent passwd

如您所见，输出与显示 /etc/passwd 文件内容时的输出相同。如果使用 LDAP 进行用户身份验证， getent 则将显示 /etc/passwd 文件和 LDAP 数据库中的所有 Linux 用户。

您还可以使用 awk 或 cut 仅打印包含用户名的第一个字段：

getent passwd | awk -F: '{ print $1}'

getent passwd | cut -d: -f1

检查 Linux 系统中是否存在用户

现在我们已经知道如何列出所有用户，如果要检查用户是否存在于我们的 Linux 系统中，我们可以通过将所有用户的列表传递给 grep 命令来过滤用户列表。

例如，要查明我们的 Linux 系统中是否存在具有 jack 名称的用户，我们可以使用以下命令：

getent passwd | grep jack

如果用户存在，则上面的命令将打印用户的登录信息。如果没有输出则表示用户不存在。

我们还可以在不使用 grep 命令的情况下检查用户是否存在，如下所示：

getent passwd jack

与之前相同，如果用户存在，该命令将显示用户的登录信息。

如果要查找系统上有多少用户帐户，请将 getent passwd 输出通过管道传递给 wc 命令：

getent passwd | wc -l

33

从上面的输出可以看出，我的 Linux 系统有 33 个用户帐户。

系统和普通用户

系统用户与常规（普通) 用户之间没有真正的技术差异。通常，在安装操作系统和新软件包时会创建系统用户。在某些情况下，您可以创建将由某个应用程序使用的系统用户。

普通用户是由 root 用户或具有 sudo 权限的其他用户创建的用户。通常，普通用户具有真实的登录 shell 和主目录。

每个用户都有一个名为 UID 的数字用户 ID 。如果在使用 useradd 命令创建新用户时未指定，则将根据 UID_MIN 和 UID_MIN 值自动从文件 /etc/login.defs 中选择 UID  。

要检查系统上的 UID_MIN 和 UID_MIN 的值，可以使用以下命令：

grep -E '^UID_MIN|^UID_MAX' /etc/login.defs

UID_MIN          1000
UID_MAX         60000

从上面的输出中，我们可以看到所有普通用户都应该具有 1000 到 60000 之间的 UID 。了解最小值和最大值允许我们查询系统中所有普通用户的列表。

以下命令将列出 Linux 系统中的所有普通用户：

getent passwd {1000..60000}


vagrant:x:1000:1000:vagrant,,,:/home/vagrant:/bin/bash
jack:x:1001:1001:,,,:/home/jack:/bin/bash
anne:x:1002:1002:Anne Stone,,,:/home/anne:/bin/bash
patrick:x:1003:1003:Patrick Star,,,:/home/patrick:/usr/sbin/nologin

您的系统 UID_MIN 和 UID_MIN 值可能不同，因此上面命令的更通用版本将是：

eval getent passwd {$(awk '/^UID_MIN/ {print $2}' /etc/login.defs)..$(awk '/^UID_MAX/ {print $2}' /etc/login.defs)}

如果只想打印用户名，只需将输出传递给 cut 命令：

eval getent passwd {$(awk '/^UID_MIN/ {print $2}' /etc/login.defs)..$(awk '/^UID_MAX/ {print $2}' /etc/login.defs)} | cut -d: -f1

结论

在本教程中，您学习了如何列出和过滤 Linux 系统中的用户以及系统和普通 Linux 用户之间的主要区别。

相同的命令适用于任何 Linux 发行版，包括 Ubuntu ， CentOS ， RHEL ， Debian 和 Linux Mint 。
