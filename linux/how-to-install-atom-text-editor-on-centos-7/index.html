如何在CentOS 7上安装Atom文本编辑器

2019年5月9日 • 约 700 字 , 预计阅读时间 4 分钟

Atom是由GitHub开发的开源跨平台代码编辑器。它有一个内置的包管理器，嵌入式Git控件，智能自动完成，语法高亮和多个窗格。

引擎盖下Atom是一个使用HTML，JavaScript，CSS和Node.js构建在Electron上的桌面应用程序。

在本教程中，我们将从他们的官方存储库在CentOS上安装Atom Text Editor。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

在CentOS上安装Atom

按照以下步骤在CentOS 7上安装Atom：

  1. 首先导入Atom存储库的GPG密钥：

    sudo rpm --import https://packagecloud.io/AtomEditor/atom/gpgkey
    1. 创建以下repo文件以启用Atom存储库：
    sudo nano /etc/yum.repos.d/atom.repo

    将以下内容粘贴到文件中：

    /etc/yum.repos.d/vscode.repo

    [Atom]
name=Atom Editor
baseurl=https://packagecloud.io/AtomEditor/atom/el/7/$basearch
enabled=1
gpgcheck=0
repo_gpgcheck=1
gpgkey=https://packagecloud.io/AtomEditor/atom/gpgkey

    保存并关闭文件。

    1. 启用存储库后，使用以下命令安装最新版本的Atom：
    sudo yum install atom

    Yum将提示您导入存储库GPG密钥。输入y并点击Enter。

从Atom开始

现在，在您的CentOS系统上安装了Atom，您可以通过键入code或单击Atom图标（Applications -> Programming -> Atom）从命令行启动它。

您现在可以根据自己的喜好开始安装主题和扩展以及配置Atom。

升级Atom

要在发布新版本时升级Atom安装，可以使用yum package manager正常升级过程：

sudo yum update

结论

您已在CentOS 7计算机上成功安装了Atom。要了解有关如何使用Atom的更多信息，从初学者基础知识到高级技术，请访问他们的官方文档页面。
