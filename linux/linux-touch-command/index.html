Linux Touch命令

2019年5月9日 • 约 1300 字 , 预计阅读时间 5 分钟

touch命令允许我们更新现有文件和目录的时间戳，以及创建新的空文件。

在本指南中，我们将通过实际示例和最常见命令选项的详细说明向您展示如何使用touch命令。

Linux文件时间戳

在讨论如何使用touch命令之前，让我们首先回顾一下Linux中的文件时间戳。

Linux中的文件有三个时间戳：

  • atime（访问时间） - 某些命令或应用程序（如cat，vim或grep）访问/打开文件的最后时间。
  • mtime（修改时间） - 上次修改文件内容的时间。
  • ctime（更改时间） - 文件的属性或内容最后一次更改。该属性包括文件权限，文件所有权或文件位置。

要显示包含时间戳的文件状态，可以使用该stat命令。

stat file_name

如何使用touch命令

在没有任何选项的情况下使用它是最简单的形式，如果指定为参数的文件名不存在，touch将创建一个新文件。

如果文件已存在，则touch会将文件的上次访问和修改时间更改为当前时间。

例如，如果文件file1不存在，则以下命令将创建它，否则它将更改其时间戳：

touch file1

要创建od修改多个文件或立即更改，请将文件名指定为参数：

touch file1 file2 file3

如果您不希望touch命令创建新文件，请使用-c（--no-create）选项。

例如，以下命令将更改文件file1时间戳（如果存在），否则它将不执行任何操作：

touch -c file1

仅更改访问或修改时间

默认情况下，如果未使用任何选项，则会将文件的上次访问和修改时间更新为当前时间。通过使用-a和-m选项，您只能更改其中一个时间戳。

仅更改访问时间

使用该-a选项仅更改文件的访问时间：

touch -a file1

仅更改修改时间

使用该-m选项更改文件的修改时间：

touch -m file1

更改修改时间时，更改时间也将更新。

设置特定时间戳

touch命令还允许我们更新或创建具有除当前时间之外的特定时间的文件。

使用-d（--date=）选项指定日期字符串并使用它而不是当前时间。

例如，要将最后访问和修改时间更改为file12018年6月1日11:02，您将使用以下命令：

touch -d '1 June 2018 11:02' file1

日期字符串需要用单引号括起来。

您还可以为touch命令提供部分日期时间字符串。例如，仅提供日期，自动将年份更改为当前日期：

touch -d '12 June' file1

使用该-t选项指定时间戳并使用它而不是当前时间。timestamp参数必须采用以下格式：

use [[CC]YY]MMDDhhmm[.ss]

例如，以下命令将设置file1当前年份的6月1日到2月1日的最后访问和修改时间。

touch -t 12061102 file1

使用另一个文件的时间戳

在-r（--reference=）选项允许我们指定一个参考文件，并使用的，而不是当前的时间的时间戳。

例如，以下命令将告诉touch使用file1for 的次数file2：

touch -r file1 file2

更改符号链接时间戳

默认情况下，如果在符号链接上使用touch命令，它将更改其引用文件的时间戳。

使用-h（--no-dereference）修改符号链接的时间戳。

例如，要更改symlink1您将使用的符号链接的时间戳：

touch -h symlink1

结论

到目前为止，您应该很好地理解如何使用Linux touch命令。
