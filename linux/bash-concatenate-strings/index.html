在 Bash 中拼接字符串

2019年3月24日 • 约 500 字 , 预计阅读时间 3 分钟

字符串拼接最常用的字符串操作之一。字符串拼接只是一个特殊的的编程方法，用于通过将一个字符串附加到另一个字符串的末尾来将字符串拼接在一起。

在本教程中，我们将解释如何在 Bash 中拼接字符串。

拼接字符串

拼接两个或多个字符串变量的最简单方法是一个接一个地写下它们：

VAR1="Hello,"
VAR2=" World"
VAR3="$VAR1$VAR2"
echo "$VAR3"
Hello, World

您还可以使用字符串拼接一个或多个变量：

VAR1="Hello,"
VAR2="${VAR1}World"
echo "$VAR2"
Hello, World

在上面的示例中，变量 VAR1 用大括号括起来是为了保护变量名称不受周围字符的影响。当变量后跟另一个有效的变量名字符时，必须将其括在花括号中 ${VAR1}。

为避免任何单词拆分或通配问题，您应该始终在变量名称周围使用双引号。如果要抑制变量插值和反斜杠字符的特殊处理，不要使用双引号使用单引号。

Bash 不按“类型”分隔变量，变量会根据上下文被判断为整数或字符串。您还可以拼接仅包含数字的变量。

VAR1="Hello, "
VAR2=2
VAR3=" Worlds"
VAR4="$VAR1$VAR2$VAR3"
echo "$VAR4"
Hello, 2 Worlds

使用 += 运算符拼接字符串

在 bash 中拼接字符串的另一种方法是使用 += 运算符将变量或文字字符串附加到变量：

VAR1="Hello, "
VAR1+=" World"
echo "$VAR1"
Hello, World

以下示例使用 += 运算符在 bash for 循环中拼接字符串：

languages.sh
VAR=""
for ELEMENT in 'Hydrogen' 'Helium' 'Lithium' 'Beryllium'; do
  VAR+="${ELEMENT} "
done

```bash
echo "$VAR"
Hydrogen Helium Lithium Beryllium

结论

拼接字符串变量是 Bash 脚本中最基本的操作之一。阅读本教程后，您应该很好地理解如何在 Bash 中拼接字符串。
