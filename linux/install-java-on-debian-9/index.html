如何在 Debian 9 上安装 Java

March 10, 2019

在本教程中，我们将介绍如何在 Debian 9 上安装 Java. Java 是用于构建不同类型的应用程序和系统的最流行的编程语言之一。使用 Java 开发的应用程序更易于扩展，更加灵活且更容易维护。

有两个不同的 Java 包， Java Runtime Environment (JRE)和 Java Development Kit (JDK)。如果您只想运行 Java 程序，那么您需要 JRE ，如果您是 Java 开发人员，那么您将需要 JDK ，其中包括了 JRE 和开发/调试工具和库。

Java 有两种不同的实现: OpenJDK 和 Oracle Java ，它们之间几乎没有区别。不同的只是 Oracle Java 有一些额外的商业功能。

如果您不确定要使用哪个 Java 实现和版本，一般建议使用 Debian 9 上默认提供的 OpenJDK 版本。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

安装 OpenJDK

OpenJDK 是 Java 平台的开源实现，也是是 Debian 9 中的默认 Java 开发和运行时。安装起来很简单。

使用以下命令从 Debian 仓库安装 OpenJDK 8 JDK ：

sudo apt install default-jdk

安装完成后，您可以通过检查 Java 版本来验证是否安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_181"
OpenJDK Runtime Environment (build 1.8.0_181-8u181-b13-2~deb9u1-b13)
OpenJDK 64-Bit Server VM (build 25.181-b13, mixed mode)

到此为止，您已经在 Debian 系统上成功安装了 Java 。

JRE 包含在 JDK 包中。如果只需要 JRE ，请安装 default-jre 包：

安装 Oracle Java 8

在本节中，我们将向您展示如何从 Oracle 网站下载和安装最新的 Oracle Java 版本 8 。在下载 Java tar.gz 文件之前，你应检查 Oracle 的下载页面，来确认是否有新版本。

按照以下步骤从 Oracle 网站安装最新的 Oracle Java 8 ：

  1. 使用以下 curl 命令下载 Java 文件：

    curl -L -b "oraclelicense=a" -O http://download.oracle.com/otn-pub/java/jdk/8u191-b12/2787e4a523244c269598db4e85c51e0c/jdk-8u191-linux-x64.tar.gz

    上面的命令将自动接受 Oracle 许可证并下载 Java 的 tar 打包文件 。

    1. 接下来，为 Java 安装创建一个目录：
    sudo mkdir /usr/local/oracle-java-8

  2. 解压缩 Java .tar.gz 文件到刚刚创建的目录 oracle-java-8：

    sudo tar -zxf jdk-8u191-linux-x64.tar.gz -C /usr/local/oracle-java-8
    1. 解压缩文件后，运行以下命令以创建新的替代方案：
    sudo update-alternatives --install "/usr/bin/java" "java" "/usr/local/oracle-java-8/jdk1.8.0_191/bin/java" 1500
sudo update-alternatives --install "/usr/bin/javac" "javac" "/usr/local/oracle-java-8/jdk1.8.0_191/bin/javac" 1500
sudo update-alternatives --install "/usr/bin/javaws" "javaws" "/usr/local/oracle-java-8/jdk1.8.0_191/bin/javaws" 1500

设置默认版本

如果你在 Debian 机器上安装了多个 Java 版本，通过以下命令来检查哪个版本是默认的 Java 版本：

java -version

输出应该如下所示：

java version "1.8.0_191"
Java(TM) SE Runtime Environment (build 1.8.0_191-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.191-b12, mixed mode)

要更改默认版本，请使用 update-alternativessystem 命令：

sudo update-alternatives --config java
There are 2 choices for the alternative java (providing /usr/bin/java).

  Selection    Path                                            Priority   Status
------------------------------------------------------------
* 0            /usr/local/oracle-java-8/jdk1.8.0_191/bin/java   1500      auto mode
  1            /usr/lib/jvm/java-8-openjdk-amd64/jre/bin/java   1081      manual mode
  2            /usr/local/oracle-java-8/jdk1.8.0_191/bin/java   1500      manual mode

Press <enter> to keep the current choice[*], or type selection number:

您会看到在 Debian 系统上所有已经安装的 Java 版本的列表。输入要用作默认版本的数字，然后按回车。

卸载 Java

如果出于任何原因想要卸载 Java 软件包，可以使用 apt 像其他任何安装的软件包一样卸载它。

例如，如果要卸载 default-jdk 软件包，只需运行：

sudo apt remove default-jdk

结论

现在您已经学习了如何在 Debian 服务器上安装和管理不同的 Java 版本，下一步可能是安装在 Java 上运行的众多应用程序之一，例如 Tomcat, JBoss/WildFly, Jetty, Glassfish, WebLogic, Cassandra, Jenkins 等等。
