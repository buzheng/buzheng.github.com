如何在 Debian 10 Linux 上安装 Git

2019年8月3日 • 约 600 字 , 预计阅读时间 3 分钟

Git 是许多开源和商业项目使用的世界上最流行的分布式版本控制系统。

使用 Git ，您可以与其他开发人员合作开展项目，跟踪代码更改，恢复到以前的阶段，创建分支机构等等。它最初由 Linux 内核的创建者 Linus Torvalds 开发。

本教程将介绍如何在 Debian 10 Buster 上安装和配置 Git 。

安装 Git

Git 包包含在 Debian 的默认存储库中，可以使用 apt 包管理器进行安装。另一种安装方法是从源代码编译 Git ，它允许您安装最新的 Git 版本并自定义构建选项。

在本教程中，我们将使用 apt 包管理器安装 git 。安装非常简单，只需以具有 sudo 权限的用户身份运行以下命令：

sudo apt update
sudo apt install git

通过打印 Git 版本验证安装：

git --version
git version 2.20.1

在撰写本文时， Debian Buster 存储库中可用的当前 Git 版本是 2.20.1 。

配置 Git

现在你已经安装了 Git ，建议你设置你的 Git 提交电子邮件和用户名。为此，请运行以下命令：

git config --global user.name "Your Name"git config --global user.email "youremail@yourdomain.com"

您可以使用以下命令验证更改：

git config --list
user.name=Your Name
user.email=youremail@yourdomain.com

配置设置存储在 ~/.gitconfig 文件中：

[user]
    name = Your Name
    email = youremail@yourdomain.com

要对 Git 配置进行其他更改，您可以使用该 git config 命令或使用文本编辑器直接编辑 ~/.gitconfig 文件。

更新 Git

当新版本的 Git 添加到标准 Debian 存储库时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Git 包：

sudo apt update
sudo apt upgrade

结论

在 Debian Buster 上安装 Git 是运行单个命令的问题。要了解如何使用 Git ，请查看 Pro Git 书籍。
