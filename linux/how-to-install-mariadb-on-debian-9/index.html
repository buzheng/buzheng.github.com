如何在 Debian 9 上安装 MariaDB

May 26, 2019

MariaDB 是一个开源的，多线程的关系数据库管理系统，是 MySQL的替代品, 并提供对 MySQL 的向后兼容。它由 MariaDB Foundation 维护和开发，包括一些 MySQL 的原始开发人员。

随着 Debian 9 的发布， MySQL 被 MariaDB 替换为默认数据库系统。

在本教程中，我们将向您展示如何在 Debian 9 机器上安装 MariaDB 的两种不同方法。第一种方法将引导您完成从 Debian 存储库安装 MariaDB 所需的步骤，而第二种方法将教您如何从官方 MariaDB 存储库安装最新版本的 MariaDB 。

通常，建议从 Debian 存储库安装 MariaDB 。如果您更喜欢 MySQL 而不是 MariaDB ，请查看如何在 Debian 9 上安装 MySQL 教程。如果您的应用程序没有任何特定要求，您应该坚持使用 Debian 9 中的默认数据库系统 MariaDB 。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Debian 9 上安装 MariaDB

在撰写本文时， Debian 中提供的最新 MariaDB 版本是 10.1 版。

要在 Debian 9 上安装 MariaDB ，请按照下列步骤操作：

  1. 首先 apt 输入以下内容来更新包索引：

    sudo apt update
    1. 更新包列表后，通过运行以下命令安装 MariaDB ：
    sudo apt install mariadb-server

  2. MariaDB 服务将自动启动。您可以输入以下命令进行验证：

    sudo systemctl status mariadb

    输出应该如下所示：

    ● mariadb.service - MariaDB database server
Loaded: loaded (/lib/systemd/system/mariadb.service; enabled; vendor preset
Active: active (running) since Wed 2018-09-12 15:10:40 UTC; 1min 48s ago
Main PID: 11356 (mysqld)
Status: "Taking your SQL requests now..."
CGroup: /system.slice/mariadb.service
        └─11356 /usr/sbin/mysqld

从 MariaDB 存储库在 Debian 9 上安装 MariaDB

在撰写本文时，最新版本的 MariaDB 是版本 10.3 ，可以从 MariaDB 的官方存储库安装。在继续下一步之前，请访问 MariaDB Repository 页面并检查是否有新版本可用。

要在 Debian 9 系统上安装 MariaDB 10.3 ，请按照下列步骤操作：

  1. 第一步是启用 MariaDB 存储库并将存储库 GPG 密钥导入系统，运行以下命令执行此操作：

    sudo apt install software-properties-common dirmngrsudo apt-key adv --recv-keys --keyserver hkp://keyserver.ubuntu.com:80 0xF1656F24C74CD1D8sudo add-apt-repository 'deb [arch=amd64,i386,ppc64el] http://mirrors.dotsrc.org/mariadb/repo/10.3/debian stretch main'
    1. 启用存储库后，更新包列表并使用以下命令安装 MariaDB ：
    sudo apt update
sudo apt install mariadb-server

  2. MariaDB 服务将自动启动，以验证它是否运行以下命令：

    sudo systemctl status mariadb
    ● mariadb.service - MariaDB 10.3.8 database server
Loaded: loaded (/lib/systemd/system/mariadb.service; enabled; vendor preset: enabled)

        └─migrated-from-my.cnf-settings.conf
Active: active (running) since Sun 2018-07-29 19:36:30 UTC; 56s ago

        https://mariadb.com/kb/en/library/systemd/

保护 MariaDB

要提高 MariaDB 安装的安全性，请运行以下 mysql_secure_installation 脚本：

sudo mysql_secure_installation

该脚本将提示您为 root 帐户设置密码，删除匿名用户，限制 root 用户对本地计算机的访问权限并删除测试数据库。

最后，脚本将重新加载权限表，确保所有更改立即生效。

所有步骤都会详细解释，建议对所有问题回答 Yes。

从命令行连接到 MariaDB

要通过命令行以 root 帐户类型连接到 MariaDB 服务器：

mysql -u root -p

系统将提示您输入先前使用该 mysql_secure_installation 脚本设置的 root 密码。

输入密码后，您将看到 MariaDB shell ，如下所示：

Welcome to the MariaDB monitor.  Commands end with ; or \g.
Your MariaDB connection id is 2
Server version: 10.1.26-MariaDB-0+deb9u1 Debian 9.1

Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

MariaDB [(none)]>
