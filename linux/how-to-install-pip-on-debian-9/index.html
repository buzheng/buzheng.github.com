如何在 Debian 9 上安装 pip

2019年3月24日 • 约 800 字 , 预计阅读时间 4 分钟

pip 是一个包管理系统，它简化了用 Python 编写的软件包的安装和管理，例如 Python 包索引（ PyPI ）中的软件包。

本教程将指导您在 Debian 9 上安装 Python pip ，并教您如何使用 pip 安装和管理 Python 包。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

值得一提的是，如果要在全局安装 python 模块，则应该优先使用 apt 管理器将其安装为软件包。仅当没有可用的包时，才使用 pip 全局安装 python 模块。

通常，您只能在虚拟环境中使用 pip 。 Python 虚拟环境 允许您在特定项目的指定位置安装 Python 模块，而不是全局安装。这样您就不必担心影响其他 Python 项目。

在以下部分中，我们将向您展示如何使用 apt 包管理器为 Python 2 安装 pip 和为 Python 3 安装 pip3 。

为 Python 2 安装 pip

按照以下步骤在 Debian 系统上为 Python 2 安装 pip ：

  1. 首先更新包索引：

    sudo apt update
    1. 为 Python 2 安装 pip 及其所有依赖项：
    sudo apt install python-pip

  2. 安装完成后，通过以下命令验证安装，该命令将打印 pip 版本：

    pip --version

    版本号可能会有所不同，但它看起来像这样：

    pip 9.0.1 from /usr/lib/python2.7/dist-packages (python 2.7)

为 Python 3 安装 pip

按照以下步骤在 Debian 上为 Python 3 安装 pip3 ：

  1. 首先，使用以下命令更新包列表：

    sudo apt update
    1. 接下来，键入以下命令为 Python 3 安装 pip 及其所有依赖项：
    sudo apt install python3-pip

  2. 通过打印 pip 版本来验证安装：

    pip3 --version

    版本号可能不同，但它看起来如下所示：

    pip 9.0.1 from /usr/lib/python3/dist-packages (python 3.5)

pip 用法

在本节中，我们将向您展示一些有用的基本的 pip 命令。使用 pip ，我们可以从 PyPI 安装软件包，版本控制，本地项目和分发文件。但在大多数情况下，您只是从 PyPI 安装软件包。

假设我们要安装一个名为的软件包 croniter ，我们可以通过发出以下命令来实现：

pip install croniter

croniter 使用类似 cron 的格式为 datetime 对象提供迭代。

要卸载程序包运行：

pip uninstall package_name

要从 PyPI 搜索包：

pip search "search_query"

列出已安装的包：

pip list
