Bash 数组

2019年5月28日 • 约 1500 字 , 预计阅读时间 5 分钟

数组是最常用和最基本的数据结构之一。您可以认为数组是一个可以在其中存储多个变量的变量。

在本文中，我们将讨论 Bash 数组，并列举了几个在 Bash 脚本中使用它们的示例。

Bash 数组

Bash 支持一维数字索引和关联数组类型。使用整数引用数值数组，使用字符串引用关联。指数不必是连续的。

可以使用负索引从末尾访问数字索引数组， -1 引用索引是最后一个元素。

与大多数编程语言不同， Bash 数组元素不必具有相同的数据类型。您可以创建一个包含字符串和数字的数组。 Bash 不支持多维数组，也不能包含也是数组的数组元素。

可以存储在数组中的最大元素数没有限制。

创建 Bash 数组

Bash 中的数组可以以不同的方式初始化。

创建数字索引数组

Bash 变量是无类型的，任何变量都可以用作索引数组而不声明它。

要显式声明一个数组，请使用内置的 declare ：

declare -a array_name

创建索引数组的一种方法是使用以下格式：

array_name[index_1]=value_1
array_name[index_2]=value_2
array_name[index_n]=value_n

这里的 index_ 是正整数。

创建数值数组的另一种方法是指定括号内的元素列表，用空格分隔：

array_name=( element_1 element_2 element_N )

使用上面的表单创建数组时，索引从零开始，即第一个元素的索引为 0 。

创建关联数组

与数字索引不同，必须首先使用内置的 declare 和 -A （大写）选项声明关联数组：

declare -A array_name

要创建关联数组，请使用以下格式：

declare -A array_name

array_name[index_foo]=value_foo
array_name[index_bar]=value_bar
array_name[index_xyz]=value_xyz

这里的 index_ 可以是任何字符串。

您还可以使用以下格式创建关联数组：

declare -A array_name

array_name=( 
  [index_foo]=value_foo 
  [index_bar]=value_bar 
  [index_xyz]=value_xyz 
)

数组操作

Bash 数组的语法起初可能看起来有些奇怪，但是一旦你阅读这篇文章就会更有意义。

参考元素

要引用单个元素，您需要知道元素索引。可以使用以下形式引用任何元素：

${array_name[index]}

使用数组时，${} 需要花括号来避免 shell 的文件扩展名操作符。 没有大括号，${} 访问数组元素的语法类似于大多数编程语言的语法。

让我们打印索引为 1 的元素：

## declare the array
declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## print element
echo ${my_array[1]}

Helium

如果指定的索引是 @ 或 * ，则单词将扩展为数组的所有成员。要打印您将使用的所有元素：

## declare the array
declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## print all elements
echo "${my_array[@]}"

Hydrogen Helium Lithium Beryllium

@ 或者 * 唯一的区别是 ${my_array[x]} 用双引号括起来时。在这种情况下，* 扩展为数组元素用空格分隔的单个单词，而 @ 将每个数组元素扩展为单独的单词。

您还可以通过在数组名称之前添加运算符 ! 来打印数组的键：

${!array_name[index]}

例如，要打印您将使用的所有键：

## declare the array
declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## print all elements
echo "${!my_array[@]}"

0 1 2 3

数组长度

要获取数组的长度，请使用以下格式：

${#array_name[@]}

语法与引用所有元素以及 # 数组名称前的字符时的语法相同。

## declare the array
declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## array Length
echo ${#my_array[@]}

4

循环遍历数组

迭代数组中每个项目的最常用方法是使用 for 循环。

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## Array Loop
for i in "${my_array[@]}"
do 
  echo "$i"
done

上面的代码将遍历数组并在新行中打印每个元素：

Hydrogen
Helium
Lithium
Beryllium

以下是如何打印所有键和值的示例

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## Array Loop
for i in "${!my_array[@]}"
do
  echo "$i" "${my_array[$i]}"
done

0 Hydrogen
1 Helium
2 Lithium
3 Beryllium

循环数组的另一种方法是获取数组的长度并使用 C 样式循环：

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

# Length of the array
length=${#my_array[@]}

# Array Loop
for (( i=0; i < ${length}; i++ ))
do
  echo $i ${my_array[$i]}
done

0 Hydrogen
1 Helium
2 Lithium
3 Beryllium

添加一个新元素

要向 bash 数组添加新元素并指定其索引，请使用以下格式：

my_array[index_n]="New Element"

这是一个例子：

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## add new element
my_array[9]="Aluminum"

## print all elements
echo "${my_array[@]}"

Hydrogen Helium Lithium Beryllium Aluminum

在不指定索引的情况下向数组添加新元素的另一种方法是使用 += 运算符。您可以添加一个或多个元素：

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## add new elements
my_array+=( Cobalt Nickel )

## print all elements
echo "${my_array[@]}"

Hydrogen Helium Lithium Beryllium Cobalt Nickel

删除元素

要删除单个元素，您需要知道元素索引。可以使用以下 unset 命令删除元素：

unset my_array[index]

我们来看一个例子：

declare -a my_array=( "Hydrogen" "Helium" "Lithium" "Beryllium" )

## remove element
unset my_array[2]

## print all elements
echo "${my_array[@]}"

Hydrogen Helium Beryllium

结论

到目前为止，您应该很好地理解如何使用 bash 数组。我们已经向您展示了如何创建数字索引和关联数组，如何遍历数组，计算数组长度以及添加和删除元素。
