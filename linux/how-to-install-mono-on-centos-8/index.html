在 CentOS 8 上安装配置 mono

2020年3月17日 • 约 700 字 , 预计阅读时间 4 分钟

Mono 是一个基于 ECMA/ISO 标准开发和运行跨平台应用程序的平台，它是 Microsoft .NET 框架的免费开源实现。

本教程描述了如何在 CentOS 8 上安装 Mono 。

先决条件

这些说明假定您以 root 用户或具有 sudo 特权的用户身份登录。

在 Ubuntu 上安装 Mono

在 CentOS 8 上安装 Mono 的最简单和推荐的方法是从 Mono 的存储库中安装它。这是一个相对简单的过程，只需几分钟。

  1. 首先导入存储库的 GPG 密钥：

    sudo rpm --import 'http://pool.sks-keyservers.net/pks/lookup? op=get & search=0x3fa7e0328081bff6a14da29aa6a19b38d3d831ef'

    成功时，不会产生任何输出。

  2. 通过运行以下命令将 Mono 存储库添加到您的系统中：

    dnf config-manager --add-repo https://download.mono-project.com/repo/centos8-stable.repo

    输出应类似于以下内容：

    Adding repo from: https://download.mono-project.com/repo/centos8-stable.repo

  3. 启用存储库后，安装 Mono ：

    sudo dnf install mono-complete

    mono-complete 是一个软件包集合，包含了 Mono 运行时，开发工具和所有库。

  4. 通过运行以下命令打印 Mono 版本来验证安装：

    mono --version

    在撰写本文时， Mono 的最新稳定版本是 6.8.0.105 ：

    Mono JIT compiler version 6.8.0.105 (tarball Tue Feb  4 19:28:42 UTC 2020)
Copyright (C) 2002-2014 Novell, Inc, Xamarin Inc and Contributors. www.mono-project.com
    TLS:           __thread
    SIGSEGV:       altstack
    Notifications: epoll
    Architecture:  amd64
    Disabled:      none
    Misc:          softdebug 
    Interpreter:   yes
    LLVM:          yes(610)
    Suspend:       hybrid
    GC:            sgen (concurrent by default)

现在，您已经在 CentOS 系统上成功安装了 Mono ，然后就可以开始使用它了。

Mono入门

为了确保正确设置所有内容，我们将构建一个 Hello World 程序，该程序将输出经典的 “hello world” 消息。

打开文本编辑器，然后创建一个包含以下内容的 hello.cs 文件：

using System;

public class HelloWorld
{
    public static void Main(string[] args)
    {
        Console.WriteLine ("Hello World!");
    }
}

使用 csc 编译器生成程序：

csc hello.cs

上面的命令将创建一个名为的可执行文件 hello.exe 。

使用以下命令运行程序：

mono hello.exe

输出应如下所示：

Hello, World

如果只想通过键入程序名称来执行程序，则需要添加可执行权限到需要执行的文件：

chmod +x hello.exe

现在，您可以通过键入以下内容来运行 hello.exe 文件了：

./hello.exe

结论

可从官方 Mono 软件包存储库中安装最新的稳定 Mono 发行包。
