如何在Ubuntu 18.04上安装Atom文本编辑器

2019年5月9日 • 约 800 字 , 预计阅读时间 4 分钟

Atom是由GitHub开发的开源跨平台代码编辑器。它有一个内置的包管理器，嵌入式Git控件，智能自动完成，语法高亮和多个窗格。

引擎盖下Atom是一个使用HTML，JavaScript，CSS和Node.js构建在Electron上的桌面应用程序。

在Ubuntu机器上安装Atom的最简单和推荐的方法是启用Atom存储库并通过命令行安装Atom软件包。

虽然本教程是为Ubuntu 18.04编写的，但相同的说明适用于Ubuntu 16.04和任何基于Debian的发行版，包括Debian，Linux Mint和Elementary OS。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

在Ubuntu上安装Atom

执行以下步骤在Ubuntu系统上安装Atom：

  1. 首先更新包列表并通过键入以下内容来安装依赖项：

    sudo apt updatesudo apt install software-properties-common apt-transport-https wget

  2. 接下来，使用以下wget命令导入Atom Editor GPG密钥：

    wget -q https://packagecloud.io/AtomEditor/atom/gpgkey -O- | sudo apt-key add -

    并键入以下命令启用Atom存储库：

    sudo add-apt-repository "deb [arch=amd64] https://packagecloud.io/AtomEditor/atom/any/ any main"

  3. 启用存储库后，使用以下命令安装最新版本的Atom：

    sudo apt install atom

从Atom开始

现在Atom安装在您的Ubuntu系统上，您可以通过键入code或单击Atom图标（Activities -> Atom）从命令行启动它。

您现在可以根据自己的喜好开始安装主题和扩展以及配置Atom。

升级Atom

要在发布新版本时升级Atom安装，可以使用apt软件包管理器正常升级过程：

sudo apt updatesudo apt upgrade

结论

您已在Ubuntu 18.04计算机上成功安装了Atom。要了解有关如何使用Atom的更多信息，从初学者基础知识到高级技术，请访问他们的官方文档页面。
