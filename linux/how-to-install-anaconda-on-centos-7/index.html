如何在CentOS 7上安装Anaconda

May 9, 2019

Anaconda是最受欢迎的python数据科学和机器学习平台，用于大规模数据处理，预测分析和科学计算。Anaconda发行版附带了1,000多个数据包，conda命令行工具和一个名为Anaconda Navigator的桌面图形用户界面。

本教程将介绍在CentOS 7上下载和安装Anaconda for Python 3的步骤。

安装Anaconda

在撰写本文时，Anaconda的最新稳定版本是5.3.1版本。在下载Anaconda安装程序脚本之前，请访问Anaconda下载页面，检查是否有可供下载的新版Anaconda for Python 3。

执行以下步骤在CentOS 7上安装Anaconda：

  1. 下载Anaconda安装脚本。

    导航到该/tmp目录并使用您从“下载”页面复制的链接下载Anaconda安装脚本：

    cd /tmpcurl -O https://repo.anaconda.com/archive/Anaconda3-5.3.1-Linux-x86_64.sh

    下载可能需要一些时间，具体取决于您的连接速度。

    1. 验证脚本的数据完整性。

    使用该sha256sum命令验证脚本校验和：

    sha256sum Anaconda3-5.3.1-Linux-x86_64.sh

    您应该看到如下输出：

    d4c4256a8f46173b675dd6a62d12f566ed3487f932bab6bb7058f06c124bcc27  Anaconda3-5.3.1-Linux-x86_64.sh

    确保从上面的命令打印的哈希与64位Linux页面上的Anaconda和Python 3中可用的哈希相匹配，以获得适当的Anaconda版本。

    https://docs.anaconda.com/anaconda/install/hashes/Anaconda3-5.3.1-Linux-x86_64.sh-hash.html

  2. 运行Anaconda安装脚本

    要启动Anaconda安装过程，请运行安装脚本：

    bash Anaconda3-5.3.1-Linux-x86_64.sh

    您应该看到如下输出：

    Welcome to Anaconda3 5.3.1

In order to continue the installation process, please review the license
agreement.
Please, press ENTER to continue

    按ENTER继续，然后按ENTER滚动许可证。审核完许可证后，系统会要求您批准许可条款：

    Do you accept the license terms? [yes|no]

    键入yes以接受许可证，系统将提示您选择安装位置。

    Anaconda3 will now be installed into this location:
/home/linuxize/anaconda3

    - Press ENTER to confirm the location
    - Press CTRL-C to abort the installation
    - Or specify a different location below

    默认位置适用于大多数用户，按ENTER确认位置并继续安装过程。

    If you get an error saying `bunzip2: command not found`, install the `bzip2` package with: <br />
`sudo yum install bzip2`.

    安装可能需要一些时间，一旦完成，将显示以下输出：

    要激活Anaconda安装PATH，请使用以下命令将Anaconda安装程序添加的新环境变量加载到当前shell会话中：

    source ~/.bashrc

  3. 验证安装

更新Anaconda

更新Anaconda是一个非常简单的过程，首先用以下内容更新conda工具：

conda update conda

提示确认更新时，键入y以继续。

更新conda后，继续进行Anaconda更新：

conda update anaconda

与上一个命令相同，出现提示时，键入y以继续。

您应该定期更新Anaconda安装。

卸载Anaconda

如果要从CentOS系统卸载Anaconda，请按照以下步骤操作：

  1. 删除Anaconda安装目录。

    要删除整个Anaconda安装目录类型：

    rm -rf ~/anaconda3
    1. 编辑PATH环境变量。

    编辑~/.bashrc文件并删除Anaconda安装程序添加的代码：

    在〜/ .bashrc

    # added by Anaconda3 5.3.1 installer
# >>> conda init >>>
# !! Contents within this block are managed by 'conda init' !!
__conda_setup="$(CONDA_REPORT_ERRORS=false '/home/linuxize/anaconda3/bin/conda' shell.bash hook 2> /dev/null)"
if [ $? -eq 0 ]; then
    \eval "$__conda_setup"
else
    if [ -f "/home/linuxize/anaconda3/etc/profile.d/conda.sh" ]; then
        . "/home/linuxize/anaconda3/etc/profile.d/conda.sh"
        CONDA_CHANGEPS1=false conda activate base
    else
        \export PATH="/home/linuxize/anaconda3/bin:$PATH"
    fi
fi
unset __conda_setup
# <<< conda init <<<

  2. 删除隐藏的文件。

    以下命令将删除在用户主目录中创建的隐藏文件和文件夹：

    rm -rf ~/.condarc ~/.conda ~/.continuum

    结论

    现在您已经下载并安装了Anaconda的CentOS系统，您可以查看官方的conda入门指南。
