如何在 Ubuntu 18.04 上安装 Yarn

2019年5月12日 • 约 900 字 , 预计阅读时间 4 分钟

Yarn 是一个与 npm 兼容的 JavaScript 包管理器，可帮助您自动执行安装，更新，配置和删除 npm 包的过程。它的出现是为了解决 npm 的一系列问题，例如通过并行化操作和减少与网络连接相关的错误来加速软件包安装过程。

在本教程中，我们将讨论如何安装如何在 Ubuntu 18.04 系统上使用 Yarn APT 包存储库安装 Yarn 。官方 Yarn 存储库始终如一，并提供最新版本。我们还将介绍基本的 Yarn 命令和选项。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Yarn

按照以下步骤在 Ubuntu 18.04 系统上安装 Yarn ：

  1. 第一步是启用 Yarn 存储库。首先使用以下 curl 命令导入存储库的 GPG 密钥：

    curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -

    键入以下内容，将 Yarn APT 存储库添加到系统的软件存储库列表中：

    echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list

  2. 将存储库添加到系统后，更新包列表并安装 Yarn ，其中包括：

    sudo apt update
sudo apt install yarn

    如果您尚未在系统上安装 Node.js ，则上面的命令将安装它。如果使用 nvm ，可以跳过 Node.js 安装：

    sudo apt install --no-install-recommends yarn

  3. 要验证 Yarn 是否已成功安装，请运行以下命令以打印 Yarn 版本号：

    yarn --version

    在撰写本文时， Yarn 的最新版本是版本 1.10.1 。

    1.10.1

使用 yarn

现在您已经在 Ubuntu 系统上安装了 Yarn ，下一步是探索一些最常见的 Yarn 命令。

创建一个新项目

要创建新的 Yarn 项目，请使用 yarn init 如下所示的命令：

yarn init my_yarn_project

init 脚本会问你几个问题。您可以应答或按 enter 以使用默认值。

yarn init v1.10.1
question name (vagrant): Linuxize
question version (1.0.0): 0.0.1
question description: Testing Yarn
question entry point (index.js): 
question repository url: 
question author: Linuxize
question license (MIT): 
question private: 
success Saved package.json
Done in 20.18s.

完成后，脚本将创建一个 package.json 包含您提供的信息的基本文件。您可以稍后打开并编辑此文件。

添加依赖

如果要在项目中使用其他包，则需要将其添加到项目依赖项中。为此，请使用 yarn add 命令后跟包名：

yarn add [package_name]

上面的命令还将更新 package.json 和 yarn.lock 文件，以便在运行时处理此项目的任何人都 yarn 将获得相同的依赖项。

您还可以指定包版本或包标签：

yarn add [package_name]@[version_or_tag]

升级依赖

要升级依赖项，请使用以下方法之一：

yarn upgrade [package_name]
yarn upgrade [package_name]@[version_or_tag]

上面的命令将根据 package.json 文件中指定的版本范围将项目依赖项更新为其最新版本。

删除依赖

使用 yarn remove 命令后跟包名称来删除依赖项：

yarn remove [package_name]

此命令还将更新项目 package.json 和 yarn.lock 文件。

安装所有项目依赖项

要安装 package.json 文件运行中指定的所有项目依赖项：

yarn

要么

yarn install

结论

我们已经向您展示了如何在 Ubuntu 18.04 机器上安装 yarn 和 yarn 的基本用法。有关 yarn 的更多信息，请访问 Yarn 文档页面。
