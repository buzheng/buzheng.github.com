如何在 Ubuntu 18.04 上安装最新的 Eclipse IDE

2019年4月21日 • 约 500 字 , 预计阅读时间 3 分钟

在本教程中，我们将向您展示如何在 Ubuntu 18.04 计算机上安装最新的 Eclipse IDE 。

Eclipse 是使用最广泛的 Java 集成开发环境(IDE)。它可以通过插件进行扩展，也可以用于其他编程语言的开发，如C ++，JavaScript和 PHP。

Ubuntu 存储库中提供的 Eclipse 安装包(版本 3.8.1)已过时。在 Ubuntu 18.04 上安装最新的 Eclipse IDE 最简单的方法是使用 snappy 打包系统。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

安装 Eclipse

在撰写本文时， Eclipse 的最新稳定版本是 Eclipse 4.8.0 (Photon)。

要在 Ubuntu 系统上安装 Eclipse ，请按照下列步骤操作：

  1. 安装 Java 。

    Eclipse 是一个基于 Java 的应用程序，它需要安装 Java 运行时环境 (JRE) 才能运行。使用下面的命令安装默认的 OpenJDK 包：

    sudo apt install default-jre
    1. 安装 Eclipse 。

    我们将使用 snappy 包装系统安装 Eclipse 。要在系统上下载并安装 Eclipse snap 包，请键入：

    sudo snap install --classic eclipse

    成功安装 Eclipse 后，您应该看到以下输出：

    eclipse 4.8.0 from 'snapcrafters' installed

    启动 Eclipse

    您已经学习了如何在 Ubuntu 18.04 计算机上安装 Eclipse 。您现在可以开始处理 Java 项目了。
