如何在Debian 9上安装R

May 9, 2019

R是一种快速增长的开源编程语言和环境，专门用于统计计算和图形表示。它由R统计计算基金会支持，主要由统计人员和数据挖掘者用于开发统计软件和执行数据分析。

本教程将指导您如何在Debian 9系统上安装R.

先决条件

在继续本教程之前，请确保满足以下先决条件：

  • Debian 9系统至少有1G的RAM。如果您的计算机的RAM少于1GB，则可以创建交换文件。
  • 以具有sudo权限的用户身份登录。

在Debian上安装R.

在撰写本文时，R的最新稳定版本是3.5版。从Debian仓库的R程序包通常是过时的，所以我们将通过添加维护信息库安装R CRAN。

以下步骤描述了如何在Debian 9上安装最新的稳定版R：

  1. 安装通过HTTPS添加新存储库所需的依赖项：

    sudo apt install dirmngr apt-transport-https ca-certificates software-properties-common gnupg2
    1. 通过运行以下命令，启用CRAN存储库并将CRAN GPG密钥添加到系统：
    sudo apt-key adv --keyserver keys.gnupg.net --recv-key 'E19F5F87128899B192B1A2C2AD5F960A256A04AF'sudo add-apt-repository 'deb https://cloud.r-project.org/bin/linux/debian stretch-cran35/'

  2. 添加存储库后，通过键入以下命令更新包列表并安装R包：

    sudo apt updatesudo apt install r-base
    1. 通过运行以下命令验证安装，该命令将打印R版本：
    R --version
    R version 3.5.1 (2018-07-02) -- "Feather Spray"
Copyright (C) 2018 The R Foundation for Statistical Computing
Platform: x86_64-pc-linux-gnu (64-bit)

    从CRAN安装R软件包

    R如此受欢迎的主要原因之一是通过Comprehensive R Archive Network（CRAN）提供的各种软件包。

    如果尚未安装，请安装build-essential包含编译R软件包所需工具的软件包。

    sudo apt install build-essential

出于演示目的，我们将安装一个名为的包stringr，它提供快速，正确的常见字符串操作实现。

以root身份启动时，程序包将全局安装并可供所有系统用户使用。如果您在没有sudo的情况下启动R，则会为您的用户设置个人库。

首先以root身份打开R控制台：

sudo -i R

以下所有命令均在R控制台中执行。

要安装stringr软件包，只需键入：

install.packages("stringr")

安装需要一些时间，一旦完成，加载库：

library(stringr)

创建一个名为的简单字符向量tutorial：

tutorial <- c("How", "to", "Install", "R", "on", "Debian", "9")

运行以下函数，打印字符串的长度：

str_length(tutorial)
[1] 3 2 7 1 2 6 1

您可以在“ 可用的CRAN包名称”中找到更多R包，并安装它们install.packages()。

结论

您已经成功安装了R Debian系统并学习了如何安装R软件包。
