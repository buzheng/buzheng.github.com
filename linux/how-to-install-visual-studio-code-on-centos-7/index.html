如何在CentOS 7上安装Visual Studio代码

2019年5月9日 • 约 700 字 , 预计阅读时间 4 分钟

Visual Studio Code是Microsoft开发的一种开源跨平台代码编辑器。它具有内置的调试支持，嵌入式Git控件，语法高亮，代码完成，集成终端，代码重构和代码片段。

在CentOS计算机上安装Visual Studio Code的最简单和推荐的方法是启用VS Code存储库并通过命令行安装VS Code包。

先决条件

您需要以具有sudo访问权限的用户身份登录才能安装软件包。

在CentOS上安装Visual Studio代码

执行以下步骤在CentOS系统上安装Visual Studio Code：

  1. 首先导入Microsoft GPG密钥：

    sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc

  2. 接下来，创建以下repo文件以启用Visual Studio Code存储库：

    sudo nano /etc/yum.repos.d/vscode.repo

    将以下内容粘贴到文件中：

    /etc/yum.repos.d/vscode.repo

    [code]
name=Visual Studio Code
baseurl=https://packages.microsoft.com/yumrepos/vscode
enabled=1
gpgcheck=1
gpgkey=https://packages.microsoft.com/keys/microsoft.asc

    保存文件并关闭文本编辑器。

  3. 启用存储库后，键入以下命令安装最新版本的Visual Studio代码：

    sudo yum install code

而已。Visual Studio Code已安装在CentOS桌面上，您可以开始使用它。

启动Visual Studio代码

现在，在您的CentOS系统上安装了VS Code，您可以通过键入code或单击VS Code图标（Applications -> Programming -> Visual Studio Code）从命令行启动它。

更新Visual Studio代码

发布新版本时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新Visual Studio Code包：

sudo yum update

结论

您已在CentOS 7计算机上成功安装了VS Code。要了解有关VS Code的更多信息，请访问其官方文档页面。
