Linux type 命令

2019年6月10日 • 约 600 字 , 预计阅读时间 4 分钟

在本教程中，我们将介绍 Linux type 命令。 type 命令用于显示有关命令类型的信息。它将向您展示如果在命令行上键入命令将如何解释。

如何使用 type 命令

type 命令的语法如下：

type [OPTIONS] FILE_NAME...

例如，要查找 wc 命令的类型，请键入以下内容：

type wc

输出将是这样的：

wc is /usr/bin/wc

您还可以为 type 命令提供多个参数：

type sleep head

输出将包含有关 sleep 和 head 命令的信息：

sleep is /bin/sleep
head is /usr/bin/head

命令的类型

该选项 -t 告诉 type 您打印一个单词来描述命令的类型，该命令可以是以下之一：

  • alias (shell 别名)
  • function (shell 功能)
  • builtin (shell 内置)
  • file (磁盘文件)
  • keyword (shell 保留字)

这里有一些例子：

alias

type -t grep

在我的系统 grep 中别名为 grep --color=auto ：

alias

function

type -t rvm

rvm是一个用于安装，管理和使用多个 Ruby 环境的工具(函数)：

function

buildin

type -t echo

echo 是一个内置在 Bash 中的 shell 和其他 shell ，如 Zsh 和 Ksh ：

builtin

file

type -t cut

cut 是一个可执行文件：

builtin

keyword

type -t for

for 是 Bash 中的保留词：

keyword

显示包含该命令的所有位置

要打印所有匹配项，请使用以下 -a 选项：

type -a pwd

输出将显示 pwd 内置的 shell ，但它也可以作为独立的 /bin/pwd 可执行文件变量：

pwd is a shell builtin
pwd is /bin/pwd

使用 -a 选项时，只有在未使用 -p 选项时， type 命令才会包含别名和函数。

其他的命令选项

仅当命令是磁盘上的可执行文件时，该 -p 选项才会强制 type 返回命令的路径：

例如，以下命令将不显示任何输出，因为该 pwd 命令是内置 shell 。

type -p pwd

不像 -p ，大写 -P 选项告诉 type 在磁盘上搜索 PATH 可执行文件，即使该命令不是文件。

type -P pwd
pwd is /bin/pwd

使用该 -f 选项时， type 不会像使用内置命令那样查找 shell 函数。

结论

type 命令将向您显示如何解释在命令行上使用特定命令。
