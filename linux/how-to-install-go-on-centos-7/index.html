如何在 CentOS 7 上安装 Go

April 1, 2019

Go ，通常被称为 golang , 是由 Google 创建的现代开源编程语言。许多流行的应用程序，包括 Kubernetes ， Docker ， Hugo 和 Caddy ，都是用 Go 编写的。

在本教程中，我们将向您展示如何在 CentOS 7 系统上下载和安装 Go 。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

如何安装 Go

在撰写本文时， Go 的最新稳定版本是 1.10.3 版本。在下载 tarball 之前，请访问官方 Go 下载页面检查是否有新版本可用。

按照以下步骤在 CentOS 7 上安装 Go：

  1. 下载 tarball。

    要下载 Go 二进制文件，请使用 wget 或 curl：

    wget https://dl.google.com/go/go1.10.3.linux-amd64.tar.gz
    1. 验证 tarball 。

    下载完成后，使用以下 sha256sum 命令验证 tarball 校验和：

    sha256sum go1.10.3.linux-amd64.tar.gz

    fa1b0e45d3b647c252f51f5e1204aba049cde4af177ef9f2181f43004f901035  go1.10.3.linux-amd64.tar.gz

    确保从上面的命令打印的哈希与下载页面中的哈希匹配。

    1. 提取 tarball 。

    使用 tar 命令将 tarball 解压缩到 /usr/local 目录：

    sudo tar -C /usr/local -xzf go1.10.3.linux-amd64.tar.gz

  2. 调整环境变量 Path 。

    现在我们需要通过调整 PATH 环境变量告诉我们的系统在哪里找到 Go 可执行文件二进制文件。

    我们可以通过将以下行附加到 /etc/profile 文件（用于系统范围的安装）或 $HOME/.bash_profile 文件（用于当前用户安装）来执行此操作：

    export PATH=$PATH:/usr/local/go/bin

    保存文件，并使用以下命令将新 PATH 环境变量加载到当前 shell 会话中：

    source ~/.bash_profile

测试安装

为了测试 Go 是否正确安装，我们将设置一个工作区并构建一个简单的 “Hello world” 程序。

  1. 创建工作区目录

    默认情况下，工作空间目录设置为 $HOME/go ，输入下面的命令创建工作空间目录：

    mkdir ~/go
    1. 创建一个简单的 “Hello World” Go 文件。

    在工作区内创建一个新目录 src/hello

    mkdir -p ~/go/src/hello

    并在该目录中创建一个名为的文件  hello.go， 文件 ~/go/src/hello/hello.go 内容如下：

    package main

import "fmt"

func main() {
    fmt.Printf("Hello, World\n")
}

    您可以在此处详细了解 Go 工作区目录层次结构

    1. 构建 hello.go 文件：

    要构建文件请先切换到 ~/go/src/hello 目录并运行 go build ：

    cd ~/go/src/hello
go build

    上面的命令将建立一个名为 hello 的可执行文件。

  2. 运行可执行文件：

    通过键入以下命令运行可执

    ./hello

    Hello, World

    如果您看到上面的输出，那么您已成功安装 Go 。

结论

现在您已经下载并安装了 Go CentOS 系统，您可以开始开发 Go 项目了。
