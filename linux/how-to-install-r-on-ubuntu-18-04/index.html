如何在 Ubuntu 18.04 上安装 R

May 2, 2019

R 是一种快速增长的开源编程语言和专门用于统计计算和图形表示的自由环境。它由 R 统计计算基金会支持，主要由统计人员和数据挖掘者用于开发统计软件和执行数据分析。

本教程将指导您完成在 Ubuntu 18.04 计算机上安装 R 的步骤。

先决条件

在开始学习本教程之前，您需要一台 Ubuntu 18.04 计算机：

  • 至少 1G 的 RAM 。如果系统的 RAM 少于 1GB ，则可以创建交换文件。
  • 具有 sudo 权限的非 root  用户。

在 Ubuntu 上安装 R

在撰写本文时， R 的最新稳定版本是 3.5 版。通常 Ubuntu 软件库中的 R 程序包不是最新的，所以我们将通过添加由 CRAN维护的软件仓库安装 R。

要在 Ubuntu 18.04 上安装最新的稳定版本的 R ，请按照下列步骤操作：

  1. 安装通过 HTTPS 添加新存储库所需的软件包：

    sudo apt install apt-transport-https software-properties-common
    1. 使用以下命令启用 CRAN 存储库并将 CRAN GPG 密钥添加到系统：
    sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys E298A3A825C0D65DFD57CBB651716619E084DAB9
sudo add-apt-repository 'deb https://cloud.r-project.org/bin/linux/ubuntu bionic-cran35/'

  2. 现在添加了存储库，更新包列表并通过键入以下命令安装 R 包：

    sudo apt update
sudo apt install r-base
    1. 要验证安装是否成功，请运行以下命令，该命令将打印 R 版本：
    R --version

    从 CRAN 安装 R 软件包

    R 如此受欢迎的主要原因之一是 Comprehensive R Archive Network （ CRAN ）提供的各种软件包。

    安装 build-essential 软件包，它包含编译 R 包所需工具。

    sudo apt install build-essential

出于演示目的，我们将安装一个名为 stringr 的包，它提供快速，正确的常见字符串操作实现。

以 root 身份启动时，程序包将全局安装并可供所有系统用户使用。如果您在没有 sudo 的情况下启动 R ，则会为您的用户设置个人库。

首先以 root 身份打开 R 控制台：

sudo -i R

以下所有命令均在 R 控制台中执行。

键入以下命令安装 stringr 包：

install.packages("stringr")

安装需要一些时间，一旦完成，完成，加载库：

library(stringr)

创建一个名为 tutorial 的简单字符向量：

tutorial <- c("How", "to", "Install", "R", "on", "Ubuntu", "18.04")

运行以下函数，打印字符串的长度：

str_length(tutorial)

[1] 3 2 7 1 2 6 5

您可以在通过名称查找可用的 CRAN 包，并通过 install.packages() 安装它们。

结论

您已成功安装 R 您的 Ubuntu 系统并学习如何安装 R 软件包。
