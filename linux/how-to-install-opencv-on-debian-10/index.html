在 Debian 10 Linux 上安装 OpenCV 教程

2020年1月16日 • 约 900 字 , 预计阅读时间 4 分钟

本教程显示了如何在 Debian 10 Buster 上安装 OpenCV 。对于大多数人而言，在 Debian 上安装 OpenCV 的最简单方法是使用 apt 软件包管理工具进行安装。

OpenCV( 开源计算机视觉库)是一个开源计算机视觉库，并具有 C++ ， Python 和 Java 的实现。它可用于广泛的应用程序，包括医学图像分析，拼接街景图像，监视视频，检测和识别人脸，跟踪运动对象，提取 3D 模型等。

OpenCV 可以利用多核处理的优势，并具有 GPU 加速功能以进行实时操作。

从 Debian 存储库安装 OpenCV

可从标准 Debian 存储库中获得 OpenCV Python 模块。在撰写本文时，标准 Debian 存储库包括过时的 OpenCV 3.2 版。

要安装 OpenCV Python 模块，请输入：

sudo apt update

上面的命令将安装运行 OpenCV 所需的所有软件包。

要验证安装，请导入 cv2 模块并打印 OpenCV 版本：

python3 -c "import cv2; print(cv2.__version__)"

3.2.0

如果要使用 Python 2 绑定安装 OpenCV ，请安装 python-opencv 软件包。

从源代码安装 OpenCV

从源代码构建 OpenCV 库是安装 OpenCV 的推荐方法。它将针对您的特定系统进行优化，并且您将完全控制构建选项。

要从源代码安装最新的 OpenCV 版本，请执行以下步骤：

  1. 安装必需和可选的依赖项：

    sudo apt install build-essential cmake git pkg-config libgtk-3-dev \

  2. 使用以下命令克隆 OpenCV 和 OpenCV 贡献存储库：

    mkdir ~/opencv_build &&  cd ~/opencv_build

    在撰写本文时， github 存储库中的默认版本为 4.2.0 。如果要安装较旧版本的 OpenCV ，请在 cd   opencv 和 opencv_contrib 目录中均使用 cd  并运行 git checkout <opencv-version>

  3. 下载完成后，创建一个临时构建目录，并导航至该目录：

    cd ~/opencv_build/opencv

    使用 CMake 设置 OpenCV 构建：

    cmake -D CMAKE_BUILD_TYPE=RELEASE \

    CMake 构建系统完成后，您将看到类似以下内容：

    ...
-- Configuring done
-- Generating done
-- Build files have been written to: /home/pi/opencv_build/opencv/build

  4. 开始编译过程：

    make -j2

    根据您的处理器内核数量修改 -j 标志。如果您不知道处理器的内核数量，可以通过键入 nproc 来查看。

    编译可能要花费几分钟或更长时间，具体取决于您的系统配置。完成后，您将看到类似以下的内容：

    ...
[100%] Linking CXX executable ../../bin/example_tutorial_imgcodecs_imwrite
[100%] Built target example_tutorial_goodFeaturesToTrack_Demo
[100%] Built target example_tutorial_imgcodecs_imwrite

  5. 通过键入以下命令安装 OpenCV ：

    sudo make install

    ...
-- Installing: /usr/local/share/opencv4/samples/python/video_threaded.py
-- Installing: /usr/local/share/opencv4/samples/python/video_v4l2.py
-- Installing: /usr/local/share/opencv4/samples/python/watershed.py

  6. 要检查 OpenCV 是否已成功安装，请输入以下命令，您应该会看到 OpenCV 版本：

    pkg-config --modversion opencv4

    4.2.0

    python3 -c " import cv2; print(cv2.__version__) "

    4.2.0-dev

结论

我们向您展示了在 Debian 10 上安装 OpenCV 的两种不同方法。您选择的方法取决于您的要求和偏好。尽管从 Ubuntu 存储库安装打包版本更容易，但是从源代码构建 OpenCV 可以为您提供更大的灵活性，并且它应该是安装 OpenCV 时的首选。
