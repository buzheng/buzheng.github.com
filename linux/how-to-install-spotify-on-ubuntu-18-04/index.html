如何在Ubuntu 18.04上安装Spotify

May 9, 2019

Spotify是一种数字音乐流媒体服务，可让您即时访问数百万首歌曲，从古老的经典歌曲到最新的热门歌曲。

在本教程中，我们将向您展示如何在Ubuntu 18.04桌面上安装Spotify。相同的步骤适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

先决条件

您需要以具有sudo访问权限的用户身份登录才能在Ubuntu系统上安装软件包。

在Ubuntu上安装Spotify

在Ubuntu Linux上安装Spotify是一个相对简单的过程，只需几分钟。

  1. 首先使用以下命令导入存储库的GPG密钥：

    sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 931FF8E79F0876134EDDBDCCA87FF9DF48BF1C90

    输出应该如下所示：

    gpg: key A87FF9DF48BF1C90: public key "Spotify Public Repository Signing Key <tux@spotify.com>" imported
gpg: Total number processed: 1
gpg:               imported: 1

  2. 将Spotify APT存储库添加到系统的软件存储库列表中：

    echo deb http://repository.spotify.com stable non-free | sudo tee /etc/apt/sources.list.d/spotify.list
    1. 启用存储库后，更新apt源并安装Spotify：
    sudo apt updatesudo apt install spotify-client

而已。Spotify已安装在您的Ubuntu桌面上。

启动Spotify

您可以通过键入spotify或单击Spotify图标（Activities -> Spotify）从终端启动Spotify客户端：

当您第一次启动Spotify应用程序时，将出现如下所示的窗口：

升级Spotify

要在发布新版本时升级Spotify客户端，可以使用apt软件包管理器正常升级过程：

sudo apt updatesudo apt upgrade

结论

您已在Ubuntu 18.04桌面上成功安装Spotify。如果Spotify在您所在的国家/地区不可用，您可以使用VPN来解决这些地理限制。
