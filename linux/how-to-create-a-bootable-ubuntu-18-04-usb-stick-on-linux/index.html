如何在Linux上创建可启动的Ubuntu 18.04 USB记忆棒

May 9, 2019

在本教程中，我们将向您展示如何从Linux终端创建可启动的Ubuntu USB记忆棒。您可以使用此USB记忆棒在任何支持从USB启动的计算机上启动并测试或安装Ubuntu。

先决条件

  • 一个4GB或更大的USB记忆棒驱动器
  • 计算机运行任何Linux发行版
  • Ubuntu ISO文件。访问Ubuntu下载页面，您可以在其中找到Ubuntu Desktop，Ubuntu Server和各种Ubuntu风格的下载链接。您很可能想要下载最新的Ubuntu LST桌面版。

在Linux上创建可启动的Ubuntu 18.04 USB记忆棒

虽然有许多不同的GUI工具允许您将ISO映像闪存到USB驱动器，但在本教程中，我们将使用该dd命令创建可启动的ubuntu 18.04 USB记忆棒。

在Linux上创建可启动的Ubuntu 18.04 USB Stick是一个快速简便的过程，只需按照下面的详细步骤操作即可。

  1. 首先将USB闪存盘插入USB端口。

  2. 使用以下lsblk命令找出USB驱动器的名称：

    lsblk

    输出将如下所示：

    NAME        MAJ:MIN RM   SIZE RO TYPE MOUNTPOINT
sda           8:0    0 465.8G  0 disk 
└─sda1        8:1    0 465.8G  0 part /data
sdx           8:16   1   7.5G  0 disk 
└─sdx1        8:17   1   7.5G  0 part /run/media/linuxize/Kingston
nvme0n1     259:0    0 232.9G  0 disk 
├─nvme0n1p1 259:1    0   512M  0 part /boot
├─nvme0n1p2 259:2    0    16G  0 part [SWAP]
└─nvme0n1p3 259:3    0 216.4G  0 part /

    从上面的输出中可以看出，USB设备/dev/sdx可能会因系统而异。

  3. 大多数Linux发行版在插入时会自动挂载USB闪存驱动器。在闪烁图像之前，请确保未安装USB设备。要执行此操作，请使用umount命令，后跟安装它的目录（安装点）或设备名称：

    sudo umount /dev/sdx1
    1. 最后一步是将Ubuntu ISO映像刷新到USB驱动器。确保/dev/sdx使用驱动器进行更换，并且不要附加分区号。另外，替换/path/to/ubuntu-18.04.2-desktop-amd64.iso为ISO文件的路径。如果您使用Web浏览器下载了该文件，则应将其存储Downloads在您的用户帐户中的文件夹中。
    sudo dd bs=4M if=/path/to/ubuntu-18.04.2-desktop-amd64.iso of=/dev/sdx status=progress oflag=sync

    该命令将在闪烁图像时显示进度条。

    此过程可能需要几分钟，具体取决于ISO文件的大小和USB记忆棒的速度。完成后，您将看到如下内容：

    458+1 records in
458+1 records out
1921843200 bytes (1.9 GB, 1.8 GiB) copied, 147.006 s, 13 MB/s

    就这样！你的USB记忆棒上有一个可启动的Ubuntu。

    结论

    您已经学习了如何从Linux终端创建可启动的Ubuntu USB记忆棒。
