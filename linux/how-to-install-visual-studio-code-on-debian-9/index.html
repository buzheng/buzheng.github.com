如何在Debian 9上安装Visual Studio Code

May 9, 2019

Visual Studio Code是Microsoft开发的免费开源跨平台代码编辑器。它具有内置的调试支持，嵌入式Git控件，语法高亮，代码完成，集成终端，代码重构和代码片段。可以使用扩展来扩展Visual Studio代码功能。

本教程介绍如何使用VS Code存储库中的apt在Debian上安装Visual Studio代码编辑器。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

在Debian上安装Visual Studio代码

完成以下步骤以在Debian系统上安装Visual Studio Code：

  1. 首先输入以下内容来更新软件包索引并安装依赖项：

    sudo apt updatesudo apt install software-properties-common apt-transport-https curl
    1. 使用以下curl命令导入Microsoft GPG密钥：
    curl -sSL https://packages.microsoft.com/keys/microsoft.asc | sudo apt-key add -

    将Visual Studio代码存储库添加到您的系统：

    sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"
    1. 添加存储库后，安装最新版本的Visual Studio代码：
    sudo apt updatesudo apt install code

而已。Visual Studio Code已安装在Debian桌面上，您可以开始使用它。

启动Visual Studio代码

一旦在Debian系统上安装了VS Code，您就可以通过键入code或单击VS Code图标（Activities -> Visual Studio Code）从命令行启动它。

您现在可以根据自己的喜好开始安装扩展和配置VS代码。

更新Visual Studio代码

发布新版本的Visual Studio Code时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新软件包：

sudo apt updatesudo apt upgrade

结论

您已在Debian 9计算机上成功安装了VS Code。您的下一步可能是安装其他组件并自定义您的用户和工作区设置。
