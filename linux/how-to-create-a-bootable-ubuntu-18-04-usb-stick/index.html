如何在Windows上创建可启动的Ubuntu 18.04 USB记忆棒

May 9, 2019

本教程将引导您完成在Windows上创建可启动Ubuntu USB记忆棒的过程。您可以使用此USB记忆棒在任何支持从USB启动的计算机上启动并测试或安装Ubuntu。

先决条件

  • 一个4GB或更大的USB记忆棒驱动器
  • Microsoft Windows XP或更高版本

在Windows上创建可启动的Ubuntu 18.04 USB记忆棒

在Windows上创建可启动的Ubuntu 18.04 USB Stick是一个相对简单的过程，只需按照下面列出的步骤操作即可。

下载Ubuntu ISO文件

要下载Ubuntu ISO文件，请访问Ubuntu下载页面，您可以在其中找到Ubuntu Desktop，Ubuntu Server和各种Ubuntu版本的下载链接。

您很可能想要下载最新的Ubuntu LST桌面版。

安装Etcher

有几种不同的应用程序可供免费使用，它们允许您将ISO映像闪存到USB驱动器。在本教程中，我们将使用Etcher创建一个可启动的ubuntu 18.04 USB记忆棒。

Etcher是一个免费的开源实用程序，用于将图像闪存到SD卡和USB驱动器，并支持Windows，macOS和Linux。

前往balenaEtcher下载页面，下载最新的Etcher for Windows。

下载安装文件后，双击它以启动安装向导。按照安装向导的步骤在Windows桌面上安装Etcher。

将Ubuntu ISO文件刷新到USB记忆棒

使用Etcher创建可启动的Ubuntu USB记忆棒是一项易于执行的任务。

  1. 将USB闪存盘插入USB端口并启动Etcher。

  2. 单击Select image按钮，找到您的Ubuntu .iso文件。如果您使用Web浏览器下载了该文件，则应将其存储Downloads在您的用户帐户中的文件夹中。

  3. 如果只有一个驱动器，Etcher将自动选择USB驱动器。否则，如果连接了多个SD卡或USB记忆棒，请确保在闪烁图像之前选择了正确的USB驱动器。

  4. 单击Flash image按钮，闪烁过程将开始。

    Etcher将在闪烁图像时显示进度条和ETA。

    此过程可能需要几分钟，具体取决于ISO文件的大小和USB记忆棒的速度。完成后，将出现以下屏幕，通知您图像已成功闪烁。

    单击[X]以关闭Etcher窗口。

就这样！你的USB记忆棒上有一个可启动的Ubuntu。

结论

您已经学习了如何在Windows上创建可启动的Ubuntu USB记忆棒。
