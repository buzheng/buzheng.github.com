如何在 Ubuntu 18.04 上安装 Opera 浏览器

April 15, 2019

Opera 是世界上最流行的跨平台 Web 浏览器之一。它是为现代网络构建的快速，易用且安全的浏览器。

Opera 基于与 Chrome 相同的引擎，提供类似的浏览体验，并允许您从 Google 的官方网上商店安装 Chrome 扩展程序。它内置的功能有很多，如免费 VPN ，广告拦截器，电池节电器和加密钱包。

Opera 不是一个开源浏览器，它不包含在默认的 Ubuntu 存储库中。

在本教程中，我们将阐述如何在 Ubuntu 18.04 上安装 Opera Web 浏览器。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Debian 的发行版，包括 Debian ， Linux Mint 和 Elementary OS 。

在 Ubuntu 上安装 Opera

在 Ubuntu Linux 上安装 Opera 是一个相对简单的过程，只需几分钟。

在 Ubuntu 上安装软件包需要 sudo 权限。确保以具有 sudo 权限的用户身份运行以下命令。

  1. 打开终端并使用以下 wget 命令导入 Opera 存储库的 GPG 密钥：

    wget -qO- https://deb.opera.com/archive.key | sudo apt-key add -

    该命令应该输出 OK ，这意味着密钥已成功导入，并且来自此存储库的包将被视为可信任。

    1. 接下来，将 Opera APT 存储库添加到系统的软件存储库列表中：
    echo deb https://deb.opera.com/opera-stable/ stable non-free | sudo tee /etc/apt/sources.list.d/opera.list

  2. 启用存储库后，更新包列表并安装 opera-stable 包：

    sudo apt updatesudo apt install opera-stable

    在安装过程中，系统将提示您是否要创建新的存储库文件。选择 No，因为我们已经将 Opera 存储库添加到系统 APT 源。

    此时， Opera 已安装在您的 Ubuntu 桌面上。

    启动 Opera

    现在 Opera 已安装在您的 Ubuntu 系统上，您可以通过键入 opera 或单击 Opera 图标（ Activities -> Opera ）从命令行启动它。

    当您第一次启动 Opera 时，您将被重定向到默认的 Opera 欢迎页面。

    从这里开始，您可以开始自定义浏览器的设置，主题和外观。

    更新 Opera

    发布新版本时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Opera 软件包：

    sudo apt update
sudo apt upgrade

卸载 Opera

如果要卸载 Opera ，只需删除已安装的软件包，然后使用以下命令禁用存储库：

sudo apt remove --auto-remove opera-stable
sudo rm -f /etc/apt/sources.list.d/opera.list

结论

我们已经向您展示了如何在 Ubuntu 18.04 台式机上安装 Opera 。如果您以前使用过其他浏览器，例如 Chrome 或 Firefox ，则可以将书签和设置导入 Opera 。
