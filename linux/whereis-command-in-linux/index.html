Linux 中的 whereis 命令

2019年10月14日 • 约 800 字 , 预计阅读时间 4 分钟

whereis 是一个命令行实用程序，可让您查找给定命令的二进制文件，源文件和手册页文件的位置。

在本文中，我们将向您展示如何在 Linux 中使用 whereis 命令。

如何使用 whereis 命令

该 whereis 命令的语法如下：

whereis [OPTIONS] FILE_NAME...

不带任何选项使用时， whereis 命令在二进制文件，源文件和手册文件中搜索指定为参数的文件或者命令的位置。

默认情况下， whereis 在环境变量中列出的路径和目录中搜索命令的文件。使用该 -l 选项可找到 whereis 命令搜索的目录。

whereis -l

例如，要获取有关该 bash 命令的信息，请键入以下内容：

whereis bash

bash: /bin/bash /etc/bash.bashrc /usr/share/man/man1/bash.1.gz

上面的输出中 bash: 是您要获取信息的命令， /bin/bash 是二进制文件的路径， /etc/bash.bashrc 源文件以及 /usr/share/man/man1/bash.1.gz 手册页。

如果要搜索的命令不存在， whereis 将仅打印命令名称。

您还可以为 whereis 命令提供多个参数：

whereis netcat uptime

输出将包含有关 netcat 和 uptime 命令的信息：

netcat: /bin/netcat /usr/share/man/man1/netcat.1.gz
uptime: /usr/bin/uptime /usr/share/man/man1/uptime.1.gz

要仅搜索命令二进制文件，请使用该 -p 选项。

例如，要查找 ping 命令的位置，请键入以下内容：

whereis -p ping

ping: /bin/ping

仅搜索命令二进制文件的位置时，最好使用 which 或 type 命令。

要仅搜索源文件，请使用 -s 选项。

whereis -s command

如果源文件存在， whereis 则将打印其位置。

该 -m 选项仅允许您搜索手册文件：

whereis -m command

要限制 whereis 搜索二进制文件的位置请使用 -B 选项，要搜索帮助手册请使用 -M 选项，要搜索源请使用 -S 选项。每个选项均接受以空格分隔的目录的绝对路径列表。目录列表必须由 -f 结束。

例如，要 cp 在 /bin 目录中搜索二进制文件，请输入：

whereis -b -B /bin -f cp

cp: /bin/cp

该 -u 选项指示 whereis 搜索异常条目。对于每种请求类型(二进制，手动和源文件)而言，没有完全相同的一项的文件将被视为异常文件(命令)。

例如，要搜索 /bin 目录中没有手册页或具有多个文档的所有二进制文件，请输入：

cd /bin
whereis -m -u *

该选项 -u 后的通配符 * 表示当前工作目录(/bin)中的所有文件。

结论

该 whereis 实用程序用于查找给定命令的二进制文件，源文件和手册文件。
