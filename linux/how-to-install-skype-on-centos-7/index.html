如何在 CentOS 7 上安装 Skype

2019年3月6日 • 约 500 字 , 预计阅读时间 3 分钟

Skype 是世界上最受欢迎的通信应用程序之一，它允许您免费进行在线音频和视频通话，并供提价格合理的国际通话服务来拨打全球的移动电话和固定电话。

Skype 不是一个开源应用程序，它不包含在 CentOS 存储库中。

本教程介绍如何在 CentOS 7 上安装最新版本的 Skype 。

先决条件

您登录的用户必须具有 sudo 权限才能安装软件包。

在 CentOS 上安装 Skype

执行以下步骤在 CentOS 上安装 Skype 。

下载 Skype

首先打开终端，方法是使用 Ctrl+Alt+T 键盘快捷键或单击终端图标。

使用下面的 wget 命令来下载最新的 Skype 软件的 .rpm 包：

wget https://go.skype.com/skypeforlinux-64.rpm

安装 Skype

下载完成后，使用以具有 sudo 权限的用户身份运行以下命令来安装 Skype ：

sudo yum localinstall ./skypeforlinux-64.rpm

到此位置， Skype 已安装在您的 CentOS 桌面上。

启动 Skype

现在已经安装了 Skype ，您可以从命令行键入 skypeforlinux 启动它或单击 Skype 图标（Applications -> Internet -> Skype）启动它。

当您第一次启动 Skype 时，将出现如下所示的窗口：

您可以从这里使用您的 Microsoft 帐户登录 Skype ，并开始与您的朋友和家人聊天和交谈。

更新 Skype

在安装过程中，官方 Skype 存储库将添加到您的系统中。使用 cat 命令验证文件内容：

cat /etc/yum.repos.d/skype-stable.repo
[deb [arch=amd64] https://repo.skype.com/deb stable main]([skype-stable]
name=skype (stable)
baseurl=https://repo.skype.com/rpm/stable/
enabled=1
gpgcheck=1
gpgkey=https://repo.skype.com/data/SKYPE-GPG-KEY)

这可确保在 Skype 发布新版本时，通过桌面标准软件更新工具更新是，您的 Skype 将自动更新。

结论

在本教程中，您已经学习了如何在 CentOS 7 桌面上安装 Skype 。
