如何在CentOS上列出已安装的软件包

May 9, 2019

在本教程中，我们将向您展示如何在CentOS上列出和过滤已安装的软件包。了解如何在CentOS系统上列出已安装的软件包对于需要在另一台计算机上安装相同软件包或者要重新安装系统的情况非常有用。

我们还将向您展示如何检查是否安装了特定软件包，计算已安装的软件包以及查找已安装软件包的版本。

列出已安装的包与Yum

YUM（Yellow dog Updater）是默认的CentOS包管理器。它可用于从官方和第三方CentOS存储库下载，安装，删除，查询和管理CentOS RPM软件包。

要使用yum列出CentOS系统上已安装的软件包，可以使用以下命令：

sudo yum list installed

它将打印所有已安装软件包的列表，包括有关软件包版本和存储库的信息。

包列表很长，为了更好的可读性，最好将输出传递给less：

sudo yum list installed | less

要确定是否安装了特定软件包，可以使用grep命令过滤输出。

例如，要查明我们的系统上是否安装了解压缩包，我们将运行：

sudo yum list installed | grep unzip
unzip.x86_64    6.0-19.el7    @anaconda

上面的输出显示我们已经在我们的机器上安装了解压缩版本6.0-19。

列出已安装的包含Rpm

在CentOS中，您可以使用rpm带有-q查询包的选项的命令。以下命令将列出所有已安装的软件包：

sudo rpm -qa

不同的是yum，rpm只会显示包名称。

查询（搜索）是否安装了某个软件包会将软件包名称传递给该rpm -q命令。以下命令将显示系统上是否安装了tmux软件包：

sudo rpm -q tmux

如果软件包已安装，您将看到如下内容：

tmux-1.8-4.el7.x86_64

否则，该命令将打印：

package tmux2is not installed

创建所有已安装软件包的列表

要在CentOS系统上创建所有已安装软件包的名称列表并将其保存在名为的文件中packages_list.txt，请运行以下命令：

sudo rpm -qa > packages_list.txt

要在另一台服务器上安装相同的软件包，可以使用cat命令将所有软件包传递到yum：

sudo yum -y install $(cat packages_list.txt)

计算已安装软件包的数量

要了解系统上安装了多少软件包，您可以使用与以前相同的命令，但不是将输出重定向到文件，而是可以将其传送到wc实用程序并计算行数：

sudo rpm -qa | wc -l
603

上面的输出显示系统中安装了603个软件包。

结论

到目前为止，您应该知道如何列出和过滤CentOS系统上已安装的软件包。还有一个名为的工具repoquery包含在yum-utils包中，您可以使用它来获取已安装包的列表。

centos yum rpm
