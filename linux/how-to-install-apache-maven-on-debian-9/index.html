如何在Debian 9上安装Apache Maven

2019年5月9日 • 约 1400 字 , 预计阅读时间 5 分钟

Apache Maven是一个开源项目管理和理解工具，主要用于Java项目。Maven使用项目对象模型（POM），它本质上是一个XML文件，包含有关项目，配置详细信息，项目依赖项等的信息。

在本教程中，我们将向您展示在Debian 9上安装Apache Maven的两种不同方法。

官方Debian存储库包含可与apt程序包管理器一起安装的Maven程序包。这是在Debian上安装Maven的最简单方法，但是包含在存储库中的版本总是在最新版本的Maven之后的几个版本。

要安装最新版本的Maven，请按照本文第二部分提供的说明从我们的官方网站下载Maven。

选择一种最适合您的安装方法。

先决条件

为了能够在Debian系统上安装软件包，您必须以具有sudo权限的用户身份登录。

用Apt在Debian上安装Apache Maven

使用Debian安装Maven apt是一个简单，直接的过程。

  1. 首先，更新包索引：

    sudo apt update

  2. 通过运行以下命令安装Maven：

    sudo apt install maven

  3. 键入以下命令验证安装：

    mvn -version

    输出应该如下所示：

    Apache Maven 3.3.9
Maven home: /usr/share/maven
Java version: 1.8.0_181, vendor: Oracle Corporation
Java home: /usr/lib/jvm/java-8-openjdk-amd64/jre
Default locale: en_US, platform encoding: UTF-8
OS name: "linux", version: "4.9.0-8-amd64", arch: "amd64", family: "unix"

而已。Maven现在安装在你的Debian系统上。

安装最新版本的Apache Maven

以下部分提供了在Debian 9上安装最新Apache Maven版本的详细信息。我们将从其官方网站下载最新版本的Apache Maven。

1.安装OpenJDK

Maven 3.3+需要在您的系统上安装JDK 1.7或更高版本。我们将安装OpenJDK，它是Debian 9中的默认Java开发和运行时。

首先更新包索引：

sudo apt update

输入以下命令安装OpenJDK包：

sudo apt install default-jdk

通过检查其版本来验证Java安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_181"
OpenJDK Runtime Environment (build 1.8.0_181-8u181-b13-2~deb9u1-b13)
OpenJDK 64-Bit Server VM (build 25.181-b13, mixed mode)

如果您更喜欢Oracle Java而不是OpenJDK，请按照以下说明安装Oracle Java。

2.下载Apache Maven

在撰写本文时，最新版本的Apache Maven是3.6.0。在继续下一步之前，您应该检查Maven下载页面以查看是否有更新的版本。

/tmp使用以下wget命令在目录中下载Apache Maven存档：

wget https://www-us.apache.org/dist/maven/maven-3/3.6.0/binaries/apache-maven-3.6.0-bin.tar.gz -P /tmp

下载完成后，在目录中解压缩归档/opt：

sudo tar xf /tmp/apache-maven-*.tar.gz -C /opt

为了更好地控制Maven版本和更新，我们将创建一个maven指向Maven安装目录的符号链接：

sudo ln -s /opt/apache-maven-3.6.0 /opt/maven

稍后如果要升级Maven安装，只需解压缩较新版本并更改符号链接以指向最新版本即可。

3.设置环境变量

接下来，我们需要设置环境变量。为此，请打开文本编辑器并创建一个名为目录mavenenv.sh内的新文件/etc/profile.d/。

sudo nano /etc/profile.d/maven.sh

粘贴以下配置：

/etc/profile.d/maven.sh

export JAVA_HOME=/usr/lib/jvm/default-java
export M2_HOME=/opt/maven
export MAVEN_HOME=/opt/maven
export PATH=${M2_HOME}/bin:${PATH}

保存并关闭文件。此脚本将在shell启动时获取。

键入以下命令使脚本可执行：

sudo chmod +x /etc/profile.d/maven.sh

最后使用以下命令加载环境变量：

source /etc/profile.d/maven.sh

4.验证安装

要验证Maven是否已正确安装，请使用mvn -version将打印Maven版本的命令：

mvn -version

您应该看到如下内容：

Apache Maven 3.6.0 (97c98ec64a1fdfee7767ce5ffb20918da4f719f3; 2018-10-24T18:41:47Z)
Maven home: /opt/maven
Java version: 1.8.0_181, vendor: Oracle Corporation, runtime: /usr/lib/jvm/java-8-openjdk-amd64/jre
Default locale: en_US, platform encoding: UTF-8
OS name: "linux", version: "4.9.0-8-amd64", arch: "amd64", family: "unix"

而已。最新版本的Maven现已安装在您的Debian系统上。

结论

您已经在Debian 9上成功安装了Apache Maven。您现在可以访问官方Apache Maven文档页面并了解如何开始使用Maven。
