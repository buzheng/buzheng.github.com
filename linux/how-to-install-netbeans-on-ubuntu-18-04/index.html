如何在Ubuntu 18.04上安装Netbeans

2019年5月9日 • 约 1000 字 , 预计阅读时间 4 分钟

Netbeans是一个功能齐全的跨平台IDE，可帮助您使用Java，PHP和许多其他语言构建应用程序。

它具有内置的调试支持，嵌入式Git控件，集成了主要的构建自动化工具，如Maven，Ant和Gradle，语法高亮，代码生成器，代码模板，代码重构和丰富的导航功能。

Netbeans可通过插件进行扩展，并且具有完善的API。它还可以用于其他编程语言的开发，例如C / C ++，Groovy，Javadoc，JavaScript等。

本教程介绍如何在Ubuntu 18.04上安装Netbeans IDE。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

先决条件

您需要以具有sudo访问权限的用户身份登录才能在Ubuntu系统上安装软件包。

安装Java OpenJDK

Netbeans要求将Oracle Java over OpenJDK 8或更高版本安装到您​​的系统中。

我们将安装OpenJDK 8.安装非常简单，首先要更新软件包索引：

sudo apt update

输入以下命令安装OpenJDK 8软件包：

sudo apt install openjdk-8-jdk

通过键入以下将打印Java版本的命令来验证安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_191"
OpenJDK Runtime Environment (build 1.8.0_191-8u191-b12-2ubuntu0.18.04.1-b12)
OpenJDK 64-Bit Server VM (build 25.191-b12, mixed mode)

如果您更喜欢Oracle Java而不是OpenJDK，请按照以下说明安装Oracle Java。

安装Netbeans IDE

在撰写本文时，Netbeans的最新稳定版本是10.0版。最简单的方法是在Ubuntu 18.04上安装Netbeans是使用snappy打包系统。

要下载并安装Netbeans snap包，请使用Ctrl+Alt+T键盘快捷键打开终端并键入：

sudo snap install netbeans --classic

此过程可能需要一些时间，具体取决于您的连接速度。成功安装后，您将看到以下输出：

netbeans 10.0 from 'apache-netbeans' installed

而已。Netbeans IDE已安装在您的Ubuntu桌面上。

结论

您已经学习了如何在Ubuntu 18.04桌面上安装Netbeans。您现在可以探索新的IDE并开始处理Java和PHP项目。

要查找有关如何使用NetBeans IDE开发应用程序的更多信息，请访问Netbeans文档页面。
