如何在 Ubuntu 18.04 上安装 Tor 浏览器

2019年7月20日 • 约 900 字 , 预计阅读时间 4 分钟

Tor 浏览器通过 Tor 网络路由您的网络流量，使其成为私有和匿名的。当您使用 Tor 浏览器时，您正在查看的网站的连接会被加密并受到保护，不受网络监控和流量分析的影响。

在本教程中，我们将解释如何在 Ubuntu 18.04 上安装 Tor Web 浏览器。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。

在 Ubuntu 上安装 Tor 浏览器

在 Ubuntu Linux 上安装 Tor 浏览器是一个相对简单的过程，只需几分钟。

以具有 sudo 权限的用户身份执行以下命令。

  1. 首先安装 apt-transport-https 通过 HTTPS 访问存储库所需的软件包：

    sudo apt update
sudo apt install apt-transport-https
    1. 使用以下 wget 命令将 Tor Apt 存储库 GPG 密钥导入系统：
    wget -q -O - https://deb.torproject.org/torproject.org/A3C4F0F979CAA22CDBA8F512EE8CBC9E886DDD89.asc | sudo apt-key add -

    上面的命令应该输出 OK ，这意味着密钥已成功导入，并且来自此存储库的包将被视为可信。

  2. 启用 Tor 存储库：

    echo "deb https://deb.torproject.org/torproject.org $(lsb_release -cs) main" | sudo tee -a /etc/apt/sources.list

    $(lsb_release -cs) 将打印 Ubuntu 代号。如果你有 Ubuntu 版本  18.04 ，命令将打印 bionic 。

    1. 启用存储库后，通过键入以下 apt 命令更新程序包列表并安装最新版本的 Tor 浏览器：
    sudo apt update
sudo apt install tor deb.torproject.org-keyring torbrowser-launcher

现在， Tor 浏览器已安装在您的 Ubuntu 桌面上。

使用 Tor 浏览器

使用 Tor 浏览器有一些缺点。通过 Tor 浏览 Web 比常规 Internet 连接慢，并且一些主要网站阻止 Tor 用户。同样在某些国家/地区， Tor 是非法的或被政府强制执行的防火墙阻止。

您可以通过键入 torbrowser-launcher 或单击 Tor Browser 图标(Activities -> Tor Browser)从命令行启动 Tor 浏览器。

当您第一次启动 Tor 浏览器时，它将下载所有依赖项。

接下来，您将看到 Tor 网络设置窗口。

大多数用户只需点击“连接”按钮而无需进一步配置。审查 Tor 或使用代理的国家/地区的用户可以在此处查找有关如何配置网络的信息。

单击“连接”按钮后，浏览器将建立与 Tor 网络的连接并启动。

默认情况下， Tor 浏览器捆绑了多个安全扩展，例如 HTTPS Everywhere 和 NoScript ，并且不保留任何浏览历史记录。

强烈建议不要安装其他附加组件或插件，因为它可能会绕过 Tor 或损害您的隐私。

更新 Tor 浏览器

发布新版本时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Tor Browser 软件包：

sudo apt update
sudo apt upgrade

卸载 Tor 浏览器

如果要卸载 Tor 浏览器，只需使用以下命令删除已安装的软件包：

sudo apt remove --auto-remove tor deb.torproject.org-keyring

结论

我们已经向您展示了如何在 Ubuntu 18.04 桌面上安装 Tor 浏览器 。您现在可以安全，私密地浏览网页，并访问本地互联网服务提供商可能已阻止的网站。
