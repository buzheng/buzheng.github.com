如何在Debian 9上安装和配置Nagios

May 9, 2019

Nagios是一种流行的开源监控软件。它可以保留整个IT基础架构的清单，并确保您的网络，服务器，应用程序，服务和流程正常运行。如果关键基础架构组件发生故障，Nagios会发送通知警报。

本教程介绍如何在Debian 9上安装和配置最新版本的Nagios Core。

先决条件

您需要以具有sudo访问权限的用户身份登录才能安装软件包。

在Debian上安装Nagios

以下步骤介绍如何从源安装最新版本的Nagios Core。

首先安装编译和运行Nagios所需的以下软件包：

sudo apt update && sudo apt upgradesudo apt install autoconf gcc libc6 make wget unzip apache2 php libapache2-mod-php7.0 libgd-devsudo apt install libmcrypt-dev libssl-dev bc gawk dc build-essential libnet-snmp-perl gettext

1.下载Nagios

导航到该/usr/src目录并使用wget从项目Github存储库下载最新版本的Nagios ：[](/linux/wget-command-examples/)

cd /usr/src/sudo wget https://github.com/NagiosEnterprises/nagioscore/archive/nagios-4.4.3.tar.gz

下载完成后，解压缩tar文件：

sudo tar zxf nagios-*.tar.gz

在继续执行后续步骤之前，请确保键入以下内容以切换到Nagios源目录：

cd nagioscore-nagios-*/

2.编译Nagios

通过运行configure脚本启动构建过程：

sudo ./configure --with-httpd-conf=/etc/apache2/sites-enabled

成功完成后，您将看到以下消息：

Review the options above for accuracy.  If they look okay,
type 'make all' to compile the main program and CGIs.

运行make命令以启动构建过程：

sudo make all

编译可能需要一些时间，具体取决于您的系统。构建过程完成后，您将看到如下内容：

*** Compile finished ***
...
Enjoy.

3.创建Nagios用户和组

要创建新的系统nagios用户和组类型：

sudo make install-groups-users

将Apache www-data用户添加到nagios组：

sudo usermod -a -G nagios www-data

4.安装Nagios二进制文件

运行以下命令以安装Nagios二进制文件，CGI和HTML文件：

sudo make install

您应该看到以下输出：

*** Main program, CGIs and HTML files installed ***

5.创建外部命令目录

通过键入以下内容创建外部命令目录并设置适当的权限：

sudo make install-commandmode
*** External command directory configured ***

6.安装Nagios配置文件

安装示例Nagios配置文件：

sudo make install-config
*** Config files installed ***

Remember, these are *SAMPLE* config files.  You'll need to read
the documentation for more information on how to actually define
services, hosts, etc. to fit your particular needs.

7.安装Apache配置文件

以下命令将创建Apache配置文件：

sudo make install-webconf
*** Nagios/Apache conf file installed ***

确保启用了Apache重写和cgi模块：

sudo a2enmod rewritesudo a2enmod cgi

8.创建系统单元文件

以下命令将安装systemd单元文件，并使Nagios服务能够在引导时启动。

sudo make install-daemoninit
...
*** Init script installed ***

9.创建用户帐户

为了能够访问Nagios Web界面，您需要创建一个管理员用户。

在本教程中，我们将创建一个名为的用户nagiosadmin：

sudo htpasswd -c /usr/local/nagios/etc/htpasswd.users nagiosadmin

系统将提示您输入并确认用户密码。

New password: 
Re-type new password: 
Adding password for user nagiosadmin

重新启动Apache服务以使更改生效：

sudo systemctl restart apache2

10.配置防火墙

防火墙将保护您的服务器免受不必要的流量。

如果您的服务器上没有配置防火墙，您可以查看我们的指南，了解如何在Debian上使用ufw设置防火墙

输入以下命令打开Apache端口：

sudo ufw allow Apache

安装Nagios插件

切换回/usr/src目录并从项目Github存储库下载最新版本的Nagios插件：

cd /usr/src/sudo wget -O nagios-plugins.tar.gz https://github.com/nagios-plugins/nagios-plugins/archive/release-2.2.1.tar.gz

下载完成后，键入以下命令提取tar文件：

sudo tar zxf nagios-plugins.tar.gz

切换到插件源目录：

cd nagios-plugins-release-2.2.1

逐个运行以下命令来编译和安装Nagios插件：

sudo ./tools/setupsudo ./configuresudo makesudo make install

启动Nagios

现在你已经在你的Debian服务器上安装了Nagios，启动它：

sudo systemctl start nagios

要验证Nagios是否正在运行，请检查服务状态：

sudo systemctl status nagios

输出应如下所示，表示Nagios服务处于活动状态并正在运行。

访问Nagios Web界面

要访问Nagios Web界面，请打开您喜欢的浏览器并键入服务器的域名或公共IP地址，然后执行以下操作/nagios：

http(s)://your_domain_or_ip_address/nagios

结论

您已在Debian系统上从源代码成功安装了最新的Nagios版本。

您现在应该访问Nagios文档并了解有关如何配置和使用Nagios的更多信息。
