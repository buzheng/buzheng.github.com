Linux 中的 rm 命令

2019年10月9日 • 约 900 字 , 预计阅读时间 4 分钟

rm 是用于删除文件和目录的命令行实用程序。它是每个 Linux 用户都应该熟悉的基本命令之一。

在本指南中，我们将通过示例和最常见的选项来解释如何使用 rm 命令。

如何使用 rm 命令

rm (删除)命令的常规语法如下：

rm [OPTIONS]... FILE...

默认情况下，不带任何选项执行时， rm 不删除目录，也不提示用户是否继续删除给定的文件。

要删除单个文件，请使用 rm 命令，后跟文件名作为参数：

rm filename

如果您对父目录没有写权限，则会收到 “Operation not permitted” 错误。

如果该文件不受写保护，它将被删除，恕不另行通知。成功时，该命令不会产生任何输出，并且返回零。

删除写保护文件时，该命令将提示您进行确认，如下所示：

rm: remove write-protected regular empty file 'filename'?

键入 y 并单击 Enter 以删除文件。

该 -f 选项告诉 rm 从不提示用户并忽略不存在的文件和参数。

rm -f filename

如果要获取有关要删除的内容的信息，请使用 -v(详细)选项：

rm -v filename
removed 'filename'

删除多个文件

与 unlink 命令不同，rm 允许您一次删除多个文件。为此，请将文件名作为以空格分隔的参数传递：

rm filename1 filename2 filename3

您可以使用正则表达式来匹配多个文件。例如，要删除 .png 当前目录中的所有文件，请键入：

rm *.png

使用正则表达式时，请在运行 rm 命令之前使用 ls 命令列出文件始终是一个好主意，这样您就可以看到哪些文件将被删除。

删除目录(文件夹)

要删除一个或多个空目录，请使用以下 -d 选项：

rm -d dirname

rm -d 在功能上与 rmdir 命令相同。

要以递归方式删除非空目录及其中的所有文件，请使用 -r (递归)选项：

rm -r dirname

删除前提示

该 -i 选项告诉 rm 您在删除每个给定文件之前提示用户：

rm -i filename1 filename2

确认请输入 y ，然后按 Enter ：

rm: remove regular empty file 'filename1'?
rm: remove regular empty file 'filename2'?

当删除三个以上的文件或递归删除目录时，要获得整个操作的单个提示，请使用以下 -I 选项：

rm -i filename1 filename2 filename3 filename4

系统将要求您确认删除所有给定的文件和目录：

rm: remove 4 arguments?

rm -rf

如果给定目录或目录中的文件受写保护，该 rm 命令将提示您确认操作。要删除目录而不需要提示，请使用 -f 选项：

rm -rf dirname

该 rm -rf 命令非常危险，应格外小心！

结论

我们已经向您展示了如何使用 Linux 的 rm 命令从 Linux 系统中删除文件和目录。

删除重要文件或目录时要格外小心，因为一旦删除文件，就无法轻松恢复它。
