如何在 Ubuntu 18.04 上安装 Go

March 20, 2019

Go 是由 Google 创建的现代开源编程语言。许多流行的应用程序，包括 Kubernetes ， Docker 和 Grafana ，都是用 Go 编写的。

本教程将指导您完成在 Ubuntu 18.04 计算机上下载和安装 Go 的步骤。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Go

按照以下步骤在 Ubuntu 18.04 上安装 Go ：

下载 Go 源码包

在撰写本文时，最新的稳定版 Go 是版本 1.11.2 。在下载 Go 源码包之前，请访问官方 Go 下载页面并检查是否有新版本可用。

要下载 Go 二进制文件，您可以使用 wget 或 curl ：

wget https://dl.google.com/go/go1.11.2.linux-amd64.tar.gz

验证 Go 源码包

要验证 Go 源码包的校验和，可以使用以下 sha256sum 命令：

sha256sum go1.11.2.linux-amd64.tar.gz

输出看起来像这样

1dfe664fa3d8ad714bbd15a36627992effd150ddabd7523931f077b3926d736d  go1.11.2.linux-amd64.tar.gz

确保从上面的命令打印的哈希与下载页面中的哈希匹配。

提取 Go 源码包

将 Go 源码包使用 · 解压缩到 /usr/local 目录：

sudo tar -C /usr/local -xzf go1.11.2.linux-amd64.tar.gz

调整 PATH 环境变量

为了让我们的系统知道在哪里找到 Go 可执行二进制文件，我们需要调整 PATH 环境变量。我们可以通过将以下行附加到 /etc/profile 文件（用于系统范围的安装）或 $HOME/.profile 文件（用于当前用户安装）来执行此操作：

export PATH=$PATH:/usr/local/go/bin

保存文件，并将新的 PATH 环境变量加载到当前的 shell 会话中：

source ~/.profile

验证 Go 安装

通过打印 Go 版本验证安装

go version

输出应该如下所示：

go version go1.11.2 linux/amd64

Go 入门

我们将建立一个工作区并构建一个简单的 Hello world 程序，它将简单地打印出经典的 “hello world” 消息。

  1. 默认情况下使用 $HOME/go 作为工作空间目录，输入以下命令创建目录：

    mkdir ~/go
    1. 在工作区内创建一个新目录 src/hello
    mkdir -p ~/go/src/hello

    在该目录中，创建一个名为 hello.go 文件，内容如下：

    package main

import "fmt"

func main() {
    fmt.Printf("Hello, World\n")
}

    您可以在此处详细了解 Go 工作区目录层次结构

    1. 切换到 ~/go/src/hello 目录并运行 go build 命令来构建文件：
    cd ~/go/src/hello
go build

    上面的命令将建立一个名为可执行文件 hello 。

  2. 您只需执行以下命令即可运行可执行文件：

    ./hello
    Hello, World

    如果您看到上面的输出，那么您已成功安装 Go 。

结论

现在您已经在 Ubuntu 系统上下载并安装了 Go ，您可以开始开发 Go 项目了。
