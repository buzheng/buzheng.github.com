如何在Ubuntu 18.04上安装Kodi

May 9, 2019

Kodi（以前称为XBMC）是一个免费的开源跨平台媒体播放器和娱乐中心，可让您组织和播放流媒体，如视频，播客，音乐，互联网以及本地和网络存储。

您可以通过从官方Kodi存储库和非官方第三方存储库安装新的加载项和皮肤来增强Kodi功能。

在本教程中，我们将向您介绍如何在Ubuntu 18.04上安装Kodi。相同的说明适用于Ubuntu 16.04和任何其他基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

在Ubuntu上安装Kodi

包含在Ubuntu存储库中的Kodi版本总是落后于最新的Kodi意识到的。在撰写本文时，Kodi的最新稳定版本是17版Krypton。

我们将从他们的官方存储库安装Kodi 17。它不需要技术知识，安装和配置媒体服务器不应超过10分钟。

按照以下步骤在您的Ubuntu系统上安装Kodi：

  1. 首先更新包列表并通过键入以下内容来安装依赖项：

    sudo apt updatesudo apt install software-properties-common apt-transport-https
    1. 通过发出以下命令将Kodi APT存储库添加到系统的软件存储库列表：
    sudo add-apt-repository ppa:team-xbmc/ppa

    提示时按Enter继续：

    Official Team Kodi stable releases
More info: https://launchpad.net/~team-xbmc/+archive/ubuntu/ppa
Press [ENTER] to continue or Ctrl-c to cancel adding it
    1. 启用Kodi存储库后，更新apt软件包列表并安装最新版本的Kodi：
    sudo apt updatesudo apt install kodi

而已！此时，您已在Ubuntu 18.04系统上成功安装了Kodi。

开始Kodi

现在Kodi安装在您的Ubuntu系统上，您可以通过键入kodi或单击Kodi图标（Activities -> Kodi）从命令行启动它：

当您第一次启动Kodi时，会出现如下窗口：

从这里开始，您可以通过安装新的插件和添加媒体库来开始自定义您的Kodi实例。

要退出Kodi，请单击左上角的“关闭电源”按钮或按CTRL+END。

更新Kodi

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新Kodi软件包：

sudo apt updatesudo apt upgrade

卸载Kodi

如果要卸载Kodi，只需删除已安装的软件包，然后使用以下命令禁用存储库：

sudo apt remove --auto-remove kodisudo add-apt-repository --remove ppa:team-xbmc/ppa

然后键入以下命令删除Kodi配置目录：

rm -r ~/.kodi/

结论
