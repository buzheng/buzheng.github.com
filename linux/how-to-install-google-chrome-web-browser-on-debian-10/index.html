如何在 Debian 10 Linux 上安装 Google Chrome Web 浏览器

2019年8月28日 • 约 700 字 , 预计阅读时间 4 分钟

谷歌浏览器是世界上最受欢迎的网络浏览器。它是为现代网络构建的快速，直观且安全的浏览器。

Chrome 不是一个开源浏览器，它不包含在 Debian 存储库中。它基于 Chromium ，这是一个开源浏览器，包含在默认的 Debian Buster 存储库中。

在本教程中，我们将介绍在 Debian 10 Linux 上安装 Google Chrome 网络浏览器的步骤。

先决条件

您需要以具有 sudo 访问权限的用户身份登录才能在 Debian 系统上安装软件包。

在 Debian 上安装 Google Chrome

执行以下步骤在 Debian 10 系统上安装 Google Chrome ：

1.下载谷歌浏览器

使用 Ctrl+Alt+T 键盘快捷键或单击终端图标打开终端。

使用以下 wget 命令下载最新的 Google Chrome 的 .deb 包：

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

2.安装 Google Chrome

下载完成后，安装 Google Chrome   apt ：

sudo apt install ./google-chrome-stable_current_amd64.deb

出现提示时，输入您的用户密码，安装将继续。

启动 Google Chrome

您可以通过单击浏览器图标 (Activities -> Google Chrome) 或在命令行键入 google-chrome 来启动 Google Chrome  。

首次启动 Google Chrome 浏览器时，系统会询问您是否要将 Google Chrome 设置为默认浏览器：

根据您的喜好选择，然后单击 OK 继续。

谷歌浏览器将打开后，您将看到默认的 Chrome 欢迎页面。

您可以在此处使用自己的 Google 帐户登录 Chrome ，以便在所有设备上同步书签，历史记录，密码和其他设置。

更新 Google Chrome

在安装过程中，官方 Google Chrome apt 存储库将添加到您的系统中。使用该 cat 命令验证是否已创建存储库文件：

cat /etc/apt/sources.list.d/google-chrome.list
### THIS FILE IS AUTOMATICALLY CONFIGURED ###
# You may comment out this entry, but any other modifications may be lost.
deb [arch=amd64] http://dl.google.com/linux/chrome/deb/ stable main

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Google Chrome 软件包：

sudo apt update
sudo apt upgrade

结论

我们向您展示了如何在 Debian Buster 桌面系统上安装 Google Chrome。如果您之前使用过其他浏览器，例如 Firefox 或 Opera ，则可以将书签和设置导入 Chrome 。
