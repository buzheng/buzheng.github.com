如何在 Debian 9 上安装 Vagrant

February 28, 2019

Vagrant 是一个用于构建和管理虚拟机环境的开源命令行工具。默认情况下， Vagrant 可以在 VirtualBox ， Hyper-V 和 Docker 之上配置计算机，除此之外还可以通过 Vagrant 插件系统安装许多其他提供程序，如 Libvirt (KVM)， VMware 和 AWS 。

对开发开发人员来说， Vagrant 主要帮助他们轻松地设置与生产环境相匹配的开发环境。

本教程介绍如何在 Debian 9 机器上安装 Vagrant 。我们将使用 VirtualBox 提供程序，它是 Vagrant 的默认提供程序。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

在继续本教程之前，请确保您已满足以下先决条件：

  • 以具有 sudo 权限的用户身份登录
  • 安装了 VirtualBox

在 Debian 上安装 Vagrant

Debian 的存储库中提供的 Vagrant 包已经非常过时了，我们将从官方 Vagrant 网站下载并安装最新版本的 Vagrant 。

在撰写本文时，Vagrant 的最新稳定版本是 2.2.3 版。在继续执行后续步骤之前，您应该在 Vagrant 下载页面检查是否有更新的版本。

首先更新包列表：

sudo apt update

使用以下 curl 命令下载 Vagrant 包：

curl -O https://releases.hashicorp.com/vagrant/2.2.3/vagrant_2.2.3_x86_64.deb

一旦 .deb 文件被下载通过键入以下命令进行安装：

sudo apt install ./vagrant_2.2.3_x86_64.deb

要验证安装是否成功，请运行以下命令，该命令将打印Vagrant版本：

vagrant --version

输出应该如下所示：

Vagrant 2.2.3

Vagrant 入门

现在 Vagrant 已经安装在你的 Debian 机器上了，接下来我们就看看如何创建一个最常见的用例开发环境。

第一步是创建一个目录，该目录将作为项目根目录并保存 Vagrantfile 文件。Vagrantfile 是一个 Ruby 文件，描述了如何配置和准备虚拟机。

创建项目目录并切换到它：

mkdir ~/my-first-vagrant-project
cd ~/my-first-vagrant-project

下一步是使用 vagrant init 命令初始化一个新的 Vagrantfile ，并指定我们希望使用的 box

box 是 Vagrant 环境的包格式，并且是特定于提供者的。您可以在 Vagrant box 目录页面上找到公开可用的 Vagrant box 列表。

在这个例子中，我们将使用该 centos/7 box。运行以下命令初始化新的 Vagrantfile：

vagrant init centos/7
A `Vagrantfile` has been placed in this directory. You are now
ready to `vagrant up` your first virtual environment! Please read
the comments in the Vagrantfile as well as documentation on
`vagrantup.com` for more information on using Vagrant.

您可以打开Vagrantfile，阅读注释并根据需要进行调整。

现在我们可以运行 vagrant up 命令，根据 Vagrantfile 创建和配置虚拟机。

vagrant up
==> default: Configuring and enabling network interfaces...
default: SSH address: 192.168.121.74:22
default: SSH username: vagrant
default: SSH auth method: private key
==> default: Rsyncing folder: /home/linuxize/Vagrant/my-first-vagrant-project/ => /vagrant

从上面的输出可以看出，Vagrant还将项目目录安装 /vagrant 在虚拟机中，这样您就可以在主机上处理项目的文件。

要进入虚拟机，只需运行：

vagrant ssh

您可以使用以下命令停止虚拟机：

vagrant halt

如果运行的计算机正在运行,下面的命令将停止该计算机并且销毁在计算机初始化期间创建的所有资源：

vagrant destroy

结论

您已经学会了如何在 Debian 9 机器上安装和使用 Vagrant 。我们还向您展示了如何创建基本开发环境。

要查找有关 Vagrant 的更多信息，请访问官方 Vagrant 文​​档页面 。
