如何在Ubuntu 18.04上安装PyCharm

2019年5月9日 • 约 1000 字 , 预计阅读时间 4 分钟

PyCharm是一个用于Python和Web开发的全功能IDE 。它具有内置调试支持，Docker和Vagrant支持，嵌入式Git控件，Anaconda支持，语法突出显示，代码完成，ssh终端，代码重构和丰富的导航功能。

PyCharm可通过插件扩展，并支持Python Web开发框架，如Django和Flask。它还可以用于其他编程语言的开发，例如JavaScript，SQL，HTML，CSS，AngularJS，Node.js等。

PyCharm有两个版本，社区和专业版。您可以在此处找到有关这两个版本之间差异的更多信息。

在本教程中，我们将介绍如何在Ubuntu 18.04上安装PyCharm Community Edition。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

安装PyCharm

最简单的方法是在Ubuntu 18.04上安装PyCharm是使用snappy打包系统。

要下载并安装PyCharm快照包，请使用Ctrl+Alt+T键盘快捷键打开终端并键入：

sudo snap install pycharm-community --classic

下载可能需要一些时间，具体取决于您的连接速度。成功安装PyCharm后，将显示以下输出：

pycharm-community 2018.2.4 from 'jetbrains' installed

结论

您已经学习了如何在Ubuntu 18.04计算机上安装PyCharm。您现在可以探索新的IDE并开始处理Python项目。

虽然本教程是为Ubuntu 18.04编写的，但相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Linux Mint，Kubuntu和Elementary OS。
