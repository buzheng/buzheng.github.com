如何在Debian 9上安装Google Chrome Web浏览器

May 9, 2019

谷歌浏览器是世界上最受欢迎的网络浏览器。它是为现代网络构建的快速，直观和安全的浏览器。

Chrome不是一个开源浏览器，它不包含在Debian存储库中。谷歌浏览器基于Chromium，这是一个开源浏览器，可在Debian存储库中找到。

在本教程中，我们将介绍如何在Debian 9上安装Google Chrome网络浏览器。

先决条件

您需要以具有sudo访问权限的用户身份登录才能在Debian系统上安装软件包。

在Debian上安装Google Chrome

完成以下步骤以在Debian系统上安装Google Chrome：

1.下载谷歌浏览器

使用Ctrl+Alt+T键盘快捷键或单击终端图标打开终端。

运行以下wget命令以下载最新的Google Chrome .deb包：

wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

2.安装Google Chrome

下载完成后，键入以下内容安装Google Chrome：

sudo apt install ./google-chrome-stable_current_amd64.deb

系统将要求您输入用户密码。上面的命令将安装Chrome和所有依赖项。

启动Google Chrome

现在您已经在Debian系统上安装了Google Chrome，您可以通过输入google-chrome或点击Google Chrome图标（Activities -> Google Chrome）从命令行启动它。

当您首次启动Google Chrome时，系统会显示以下窗口，询问您是否要将Google Chrome设置为默认浏览器，并将使用统计信息和崩溃报告发送给Google：

根据您的偏好选择，然后单击OK继续。

谷歌浏览器将打开，您将看到默认的Chrome欢迎页面。

在此处，您可以使用自己的Google帐户登录Chrome，以便在所有设备上同步书签，历史记录，密码和其他设置。

正在更新Google Chrome

在安装过程中，官方Google存储库将添加到您的系统中。您可以使用cat命令验证文件内容：

cat /etc/apt/sources.list.d/google-chrome.list
### THIS FILE IS AUTOMATICALLY CONFIGURED ###
# You may comment out this entry, but any other modifications may be lost.
deb [arch=amd64] http://dl.google.com/linux/chrome/deb/ stable main

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新Google Chrome软件包：

sudo apt updatesudo apt upgrade

结论

在本教程中，我们向您展示了如何在Debian 9台式机上安装Google Chrome。如果您之前使用过其他浏览器，例如Firefox或Opera，则可以将书签和设置导入Chrome。

chrome browser debian
