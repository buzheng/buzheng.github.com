如何在 Ubuntu 18.04 上安装 Anaconda

2019年4月24日 • 约 1200 字 , 预计阅读时间 5 分钟

Anaconda 是最受欢迎的 python 数据科学和机器学习平台，用于大规模数据处理，预测分析和科学计算。 Anaconda 发行版附带了 1,000 多个数据包， conda 命令行工具和一个名为 Anaconda Navigator 的桌面图形用户界面。

本教程将指导您完成在 Ubuntu 18.04 上下载和安装 Anaconda Python Distribution 的步骤。

安装 Anaconda

在撰写本文时， Anaconda 的最新稳定版本是 5.2 版本。在下载 Anaconda 安装程序脚本之前，请访问 Anaconda 下载页面，检查是否有可供下载的新版 Anaconda for Python 3 。

按照以下步骤在 Ubuntu 18.04 上安装 Anaconda ：

  1. 下载 Anaconda 安装脚本。

    切换到 tmp 目录并使用 wget 或 curl 下载 Anaconda 安装脚本：

    cd /tmp
curl -O https://repo.anaconda.com/archive/Anaconda3-5.2.0-Linux-x86_64.sh

    下载可能需要一些时间，具体取决于您的连接速度。

  2. 验证脚本的数据完整性。

    使用该 sha256sum 命令验证脚本校验和：

    sha256sum Anaconda3-5.2.0-Linux-x86_64.sh

    您应该看到如下输出：

    09f53738b0cd3bb96f5b1bac488e5528df9906be2480fe61df40e0e0d19e3d48  Anaconda3-5.2.0-Linux-x86_64.sh

    确保从上面的命令打印的哈希值与Anaconda with Python 3 on 64-bit Linux page 上的哈希相匹配，以获得适当的 Anaconda 版本。

    https://docs.anaconda.com/anaconda/install/hashes/Anaconda3-5.2.0-Linux-x86_64.sh-hash.html

  3. 运行 Anaconda 安装脚本

    要启动 Anaconda 安装过程，请运行安装脚本：

    bash Anaconda3-5.2.0-Linux-x86_64.sh

    您应该看到如下输出：

    Welcome to Anaconda3 5.2.0

In order to continue the installation process, please review the license
agreement.
Please, press ENTER to continue

    按 ENTER 继续，然后按 ENTER 滚动许可证。审核完许可证后，系统会要求您同意许可条款：

    Do you approve the license terms? [yes|no]

    键入 yes 以接受许可证，系统将提示您选择安装位置。

    Anaconda3 will now be installed into this location:
/home/buzheng/anaconda3

    - Press ENTER to confirm the location
    - Press CTRL-C to abort the installation
    - Or specify a different location below

    默认位置适用于大多数用户，按 ENTER 确认位置并继续安装过程。

    安装可能需要一些时间，一旦完成，将显示以下输出：

    Installation finished.
Do you wish the installer to prepend the Anaconda3 install location
to PATH in your /home/buzheng/.bashrc ? [yes|no]

    如果要使用 conda 命令类型 yes 按 ENTER ，您将看到以下输出：

    Appending source /home/buzheng/anaconda3/bin/activate to /home/buzheng/.bashrc
A backup will be made to: /home/buzheng/.bashrc-anaconda3.bak

For this change to become active, you have to open a new terminal.

Thank you for installing Anaconda3!

    安装程序还会询问您是否要下载并安装 Visual Studio Code 。

    Anaconda is partnered with Microsoft! Microsoft VSCode is a streamlined
code editor with support for development operations like debugging, task
running and version control.

To install Visual Studio Code, you will need:
    - Administrator Privileges
    - Internet connectivity

Visual Studio Code License: https://code.visualstudio.com/license

Do you wish to proceed with the installation of Microsoft VSCode? [yes|no]

    您可以在网站上找到有关 Visual Studio 的更多信息。如果要安装 Visual Studio Code 类型，请 yes 键入 no 。

    要激活 Anaconda 安装，请使用以下命令将 Anaconda 安装程序添加的新环境变量 PATH 加载到当前 shell 会话中：

    source ~/.bashrc

  4. 验证安装

    您可以使用该 conda 命令验证 Anaconda 安装。例如，要显示有关当前 conda 安装的信息，请输入：

更新 Anaconda

更新 Anaconda 是一个非常简单的过程，首先用以下命令更新 conda 工具：

conda update conda

提示确认更新时，键入 y 以继续。

更新 conda 后，继续进行 Anaconda 更新：

conda update anaconda

与上一个命令相同，出现提示时，键入 y 以继续。

您应该定期更新 Anaconda 安装。

卸载 Anaconda

如果要从 Ubuntu 系统卸载 Anaconda ，请按照以下步骤操作：

  1. 删除 Anaconda 安装目录。

    要删除整个 Anaconda 安装目录类型：

    rm -rf ~/anaconda3

  2. 编辑 PATH 环境变量。

    编辑 ~/.bashrc 文件并从 PATH 环境变量中删除 Anaconda 目录：

    # added by Anaconda3 installer
export PATH="/home/buzheng/anaconda3/bin:$PATH"

  3. 删除隐藏的文件。

    以下命令将删除在用户主目录中创建的隐藏文件和文件夹：

    rm -rf ~/.condarc ~/.conda ~/.continuum

结论

现在您已经在您的 Ubuntu 系统下载并安装了 Anaconda ，您可以查看官方入门指南。
