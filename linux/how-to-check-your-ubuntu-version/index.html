如何查看你的 Ubuntu 版本

March 27, 2019

当您第一次登录 Ubuntu 系统时，在进行任何工作之前，最好查看一下该机器上运行的 Ubuntu 版本。

新的 Ubuntu 版本每 6 个月发布一次，而 LTS (长期支持) 版本每两年发布一次。桌面和服务器上支持 LTS 版本 5 年，其他标准版本支持 9 个月。

在本教程中，我们将向您展示如何查看您的 Ubuntu 版本。有几种方法可以查看您正在运行的 Ubuntu 版本，包括可以从命令行或图形界面中进行查看。

如何从命令行查看 Ubuntu 版本

实用程序 lsb_release 显示有关 Linux 发行版的 LSB (Linux Standard Base) 信息。

查看 Ubuntu 版本的首选方法是使用 lsb_release 显示有关 Linux 发行版的 LSB ( Linux 标准库)信息的实用程序。无论您运行的是哪个桌面环境或 Ubuntu 版本，此方法都可以使用。

按照以下步骤从命令行查看 Ubuntu 版本：

  1. 使用 Ctrl+Alt+T 键盘快捷键或单击终端图标打开终端。

  2. 使用该 lsb_release -a 命令显示 Ubuntu 版本。

    lsb_release -a
    No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 18.04 LTS
Release:    18.04
Codename:   bionic

    您的 Ubuntu 版本将显示在 Description 行中。从上面的输出可以看出我使用的是 Ubuntu 18.04 LTS 。

    如果您想只显示 Description 行，而不是打印所有上述信息，通过 -d 参数来指定只显示 Ubuntu 版本。

    lsb_release -d

    输出应类似于以下内容：

    Description:    Ubuntu 18.04 LTS

或者，您也可以使用下文中的命令查看您的 Ubuntu 版本。

使用 /etc/issue 文件查看 Ubuntu 版本

下面的 cat 命令将显示 /etc/issue 包含系统标识文本的内容：

cat /etc/issue

输出结果如下所示：

Ubuntu 18.04 LTS \n \l

使用 /etc/os-release 文件查看 Ubuntu 版本

/etc/os-release 是一个包含操作系统标识数据的文件，只能在运行 systemd 的较新 Ubuntu 版本上找到。

只有拥有 Ubuntu 16.04 或更高版本时，此方法才有效：

cat /etc/os-release

输出结果如下所示：

NAME="Ubuntu"
VERSION="18.04 LTS (Bionic Beaver)"
ID=ubuntu
ID_LIKE=debian
PRETTY_NAME="Ubuntu 18.04 LTS"
VERSION_ID="18.04"
HOME_URL="https://www.ubuntu.com/"
SUPPORT_URL="https://help.ubuntu.com/"
BUG_REPORT_URL="https://bugs.launchpad.net/ubuntu/"
PRIVACY_POLICY_URL="https://www.ubuntu.com/legal/terms-and-policies/privacy-policy"
VERSION_CODENAME=bionic
UBUNTU_CODENAME=bionic

使用 hostnamectl 命令查看 Ubuntu 版本

hostnamectl 是一个允许您设置主机名的命令，但您也可以使用它来查看您的 Ubuntu 版本。

此命令仅适用于 Ubuntu 16.04 或更高版本：

hostnamectl
   Static hostname: linuxize
         Icon name: computer-vm
           Chassis: vm
        Machine ID: f1ce51f447c84509a86afc3ccf17fa24
           Boot ID: 2b3cd5003e064382a754b1680991040d
    Virtualization: kvm
  Operating System: Ubuntu 18.04 LTS
            Kernel: Linux 4.15.0-22-generic
      Architecture: x86-64

查看 Gnome 桌面中的 Ubuntu 版本

Gnome 是 Ubuntu 18.04 (Bionic Beaver) 中的默认桌面环境。以前版本的 Ubuntu 使用 Unity 作为默认桌面环境。

如果您使用的是 Gnome ，请按照以下步骤查看您的 Ubuntu 版本：

  1. 单击 设置 图标，打开系统设置窗口，如下图所示：

  2. 在系统设置窗口中，单击 详情 选项卡：

    您的 Ubuntu 版本将显示在橙色的 Ubuntu 徽标下。
