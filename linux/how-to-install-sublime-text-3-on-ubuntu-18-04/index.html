如何在 Ubuntu 18.04 上安装 Sublime Text 3

2019年5月18日 • 约 600 字 , 预计阅读时间 4 分钟

Sublime Text 是用于 Web 和软件开发的最广泛使用的文本和源代码编辑器之一。它非常快，并且具有许多开箱即用的强大功能。您还可以通过安装新插件和创建自定义设置来增强其功能。

在本教程中，我们将向您展示如何在 Ubuntu 18.04 计算机上安装 Sublime Text 3 。任何基于 Debian 的发行版都应该使用相同的步骤。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Sublime 文本

要在 Ubuntu 系统上安装 Sublime Text 3 ，请按照下列步骤操作：

  1. 更新 apt 程序包列表并安装从 https 源获取程序包所需的依赖项：

    sudo apt update
sudo apt install apt-transport-https ca-certificates curl software-properties-common

  2. 使用以下 curl 命令导入存储库的 GPG 密钥：

    curl -fsSL https://download.sublimetext.com/sublimehq-pub.gpg | sudo apt-key add -

    通过键入以下内容，将 Sublime Text APT 存储库添加到系统的软件存储库列表中：

    sudo add-apt-repository "deb https://download.sublimetext.com/ apt/stable/"

  3. 启用存储库后，使用以下命令更新 apt 源并安装 Sublime Text 3 ：

    sudo apt updatesudo apt install sublime-text

现在， Sublime Text 已安装在您的 Ubuntu 桌面上。

Sublime Text 是一个专有应用程序。它可以免费下载和评估使用。但是，如果您持续使用许可证，则需要购买许可证。

要查找有关 Sublime Text 的更多信息，请访问 Sublime Text 官方文档页面。
