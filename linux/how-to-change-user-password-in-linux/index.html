如何在 Linux 中更改用户密码

March 26, 2019

本教程将向您展示如何在 Linux 中更改用户密码。我们还将介绍如何强制用户在下次登录时更改密码。本教程中提供的示例应适用于任何 Linux 发行版。

介绍

要在 Linux 中更改用户帐户的密码，可以使用该 passwd 命令。加密用户的密码以及其他密码相关信息存储在 /etc/shadow 文件中。

作为普通用户，您只能更改自己的密码。 root 用户和具有 sudo 权限的用户可以更改其他用户的密码，并定义如何使用或更改该用户的密码。

更改密码时，请确保使用强大且唯一的密码。拥有强大的密码是保护帐户最重要的事情。强密码通常至少包含 16 个字符，至少使用一个大写字母，一个小写字母，一个数字和一个特殊字符。

更改当前用户的密码

要更改用户的帐户密码，请运行 passwd 不带任何选项的命令：

passwd
Changing password for buzheng.
(current) UNIX password:
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully

系统将提示您输入当前密码。如果密码正确，则会要求您输入并确认新密码。

输入密码时，屏幕上不会显示密码。

下次登录系统时，请使用新密码。

更改另一个用户的密码

正如我们在介绍中提到的，只有 root 用户和具有 sudo 访问权限的用户可以更改另一个用户帐户的密码。以下示例假定您以具有 sudo 权限的用户身份登录。

要更改其他用户帐户的密码，请在 passwd 命令后跟用户名。例如，要更改名为的用户的密码 buzheng ，请运行以下命令：

sudo passwd buzheng

系统将提示您输入并确认新密码：

Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully

强制用户在下次登录时更改密码

默认情况下，密码设置为永不过期。如果要在下次登录时强制用户更改其密码，请在 passwd 的命令后面带上 --expire 标记，后面再带上用户名：

sudo passwd --expire buzheng

上面的命令将立即使户帐户的密码到期。

下次用户尝试使用旧密码登录时，将显示一条消息，强制他们更改密码：

ssh buzheng@192.168.121.209
WARNING: Your password has expired.
You must change your password now and login again!
Changing password for buzheng.
(current) UNIX password:
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
Connection to 192.168.121.209 closed.

用户设置新密码后，将关闭连接。

结论

在本教程中，您学习了如何更改用户密码以及如何设置密码到期。出于安全原因，建议您定期更新密码，并为每个帐户使用唯一的密码。

您可以在 Linux passwd 手册页上找到有关 passwd 命令的更多信息。
