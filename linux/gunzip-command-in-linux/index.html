Linux 中的 gunzip 命令

2019年10月12日 • 约 600 字 , 预计阅读时间 4 分钟

Gunzip 是用于解压缩 Gzip 文件的命令行工具。

Gzip 是最流行的压缩算法之一，可以减小文件大小并保留原始文件模式，所有权和时间戳。

按照约定，使用 Gzip 压缩的文件以 .gz 或 .z 扩展名给出。

在本教程中，我们将解释如何使用该 gunzip 命令。

用 gunzip 解压缩文件

该 gunzip 命令的常规语法如下：

gunzip [OPTION]... [FILE]...

在大多数 Linux 发行版中，例如 Ubuntu ， CentOS 和 Debian ， gunzip 是该 gzip -d 命令的 bash 脚本包装器。

所有 gzip 命令行选项均适用 gunzip 。

要使用 gunzip 来解压缩 .gz 文件，请将压缩文件名作为参数传递：

gunzip filename.gz

该命令会将压缩的文件恢复为其原始名称，所有者，模式和时间戳。

默认情况下，一旦解压缩， gunzip 将删除压缩文件。使用该 -k 选项保留文件：

gunzip -k filename.gz

要在终端上写入输出，请使用 -c 选项。这使您可以保留压缩文件，并可以选择将其解压缩到另一个位置：

gunzip -c filename.gz > /directory/path/filename

该 gunzip 命令还接受多个文件作为参数：

gunzip file1.gz file2.gz file3.gz

要递归解压缩给定目录中的所有文件，请使用以下 -r 选项：

gunzip -r directory

列出压缩文件的内容

与该 -l 选项一起使用时， gunzip 显示有关给定压缩文件的信息：

gunzip -l filename.gz

输出将包括未压缩的文件名，压缩和未压缩的大小以及压缩率：

         compressed        uncompressed  ratio uncompressed_name
                146                 141   9.2% filename

要获得更详细的输出，请使用以下 -v 选项：

gunzip -lv filename

method  crc     date  time           compressed        uncompressed  ratio uncompressed_name
defla 4a4a3fb5 Aug 29 15:40                 146                 141   9.2% filename

结论

该 gunzip 命令允许您解压缩 .gz 文件。

有关该 gunzip 命令的更多信息，请访问 Gnu gzip 文档页面。
