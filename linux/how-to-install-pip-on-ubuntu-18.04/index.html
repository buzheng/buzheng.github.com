如何在 Ubuntu 18.04 上安装 Pip

March 15, 2019

Pip 是一个包管理工具，它简化了用 Python 编写的软件包（例如那些 Python 包索引（ PyPI ）中的软件包）的安装和管理。Ubuntu 18.04 默认情况下没有安装 Pip ，但安装非常简单。

在本教程中，我们将向您展示如何在 Ubuntu 18.04 上使用 apt 包管理器安装 Python Pip 。我们还将向您介绍使用 pip 安装和管理 Python 包的基础知识。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

为 Python 3 安装 pip

Ubuntu 18.04 默认安装 Python 3, 作为默认的 Python 版本。完成以下步骤为 Python 3 安装 pip (pip3)：

  1. 首先使用以下命令更新包列表：

    sudo apt update
    1. 使用以下命令为 Python 3 安装 pip ：
    sudo apt install python3-pip

    上面的命令还会安装 构建 Python 模块所需的所有依赖项。

  2. 安装完成后，通过查看 pip 版本验证安装：

    pip3 --version

    版本号可能会有所不同，但它看起来像这样：

    pip 9.0.1 from /usr/lib/python3/dist-packages (python 3.6)

为 Python 2 安装 pip

默认情况下， Ubuntu 18.04 中未安装 Python 2 。要安装 Python 2 和为 Python 2 安装 pip ，请完成以下步骤：

  1. 运行以下命令更新程序包索引：

    sudo apt update
    1. 安装 Python 2 的 pip：
    sudo apt install python-pip

    上面的命令将安装 Python2 ， Pip 以及构建 Python 模块所需的所有依赖项。

  2. 通过打印 pip 版本号验证安装：

    pip --version

    版本号可能会有所不同，但它看起来像这样：

    pip 9.0.1 from /usr/lib/python2.7/dist-packages (python 2.7)

如何使用 Pip

在安装全局的 python 模块时，强烈建议使用 apt 包管理器安装提供的 python 模块，因为它们是经过测试的可以在 Ubuntu 系统上正常运行的模块。

只有在程序包管理器中没有可用的程序包时，才应使用 pip 安装全局的 Python 模块。

在大多数情况下，您应该仅在虚拟环境中使用 pip 。 Python 虚拟环境允许您为特定项目在一个独立位置安装 Python 模块，而不是全局安装。这样您就不必担心影响其他 Python 项目。

在本节中，我们将向您展示一些有用的基本 pip 命令。使用 pip ，我们可以从 PyPI 安装软件包，版本控制，本地项目和分发文件，但在大多数情况下，您只需要从 PyPI 安装软件包。

要查看所有 pip 命令和选项的列表，请输入：

pip3 --help

您可以使用 pip <command> --help 获取有关特定命令的更多信息。例如，要获取有关 install 命令的更多信息，请输入：

pip install --help

使用 Pip 安装软件包

假设您要安装一个名为的软件包 scrapy ，它可用于从网站上抓取和提取数据。

要安装最新版本的软件包，您需要运行以下命令：

pip3 install scrapy

要安装特定版本的软件包，您将发出以下命令：

pip3 install scrapy==1.5

如果您使用 Python 2 则替换 pip3 为 pip 。

使用 Pip 从 requirement.txt 安装软件包

requirement.txt 是一个文本文件，其中包含运行特定 Python 项目所需的 pip 包列表及其版本。

使用以下命令安装 requirements.txt 文件中指定的包列表：

pip3 install -r requirements.txt

列出已安装的包

要列出所有已安装的 pip 包，请使用以下命令：

pip3 list

使用 Pip 升级包

要将已安装的软件包升级到最新版本，请使用以下命令：

pip3 install --upgrade package_name

使用 Pip 卸载软件包

要卸载程序包运行：

pip3 uninstall package_name

结论

在本教程中，您学习了如何在 Ubuntu 机器上安装 pip 以及如何使用 pip 管理 Python 包。有关 pip 的更多信息，请访问 pip 用户指南页面。
