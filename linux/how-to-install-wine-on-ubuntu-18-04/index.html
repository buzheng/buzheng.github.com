如何在 Ubuntu 18.04 上安装和使用 Wine

2019年4月9日 • 约 1100 字 , 预计阅读时间 5 分钟

Wine 是一个开源兼容层，允许您在类 Unix 操作系统（如 Linux ， FreeBSD 和 macOS ）上运行 Windows 应用程序。Wine 就是 Wine，不是模拟器。它是一个接口，将 Windows 系统调用转换为 Linux 和其他基于 Unix 的操作系统使用的等效 POSIX 调用。

Wine 并不总是在 Linux 上运行 Windows 程序的最佳选择。并非所有 Windows 应用程序都能在 Wine 中运行，即使它们确实如此，它们可能也不会像通常那样运行。数据库 Wine AppDB 包含了它已经过测试，证实在 Wine 工作的应用程序列表。

您还可以使用 VirtualBox 或 VMware 等虚拟化工具，但它们需要更多系统资源，并且需要安装 Windows。

本教程介绍如何在 Ubuntu 18.04 上安装 Wine 。我们将向您展示如何安装稳定发行版 3.0 和最新版本的 Wine 4.0 。相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Linux Mint 和 Elementary OS 。

先决条件

为了能够在 Ubuntu 系统上安装新软件包，您必须以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Wine 3.0

Wine 包包含在默认的 Ubuntu 存储库中，可以使用 apt 包管理器轻松安装。这是在 Ubuntu 上安装 Wine 的最简单方法，但是存储库中包含的版本可能不是最新版本的 Wine 。

首先更新包列表：

sudo apt update

接下来，键入以下命令安装 Wine ：

sudo apt install wine64

安装完成后，使用 wine --version 命令打印 Wine 版本：

wine --version

在撰写本文时， Ubuntu 18.04 存储库中提供的当前版本的 Wine 是 3.0 。

wine-3.0 (Ubuntu 3.0-1ubuntu1)

在 Ubuntu 上安装 Wine 4.0

在本节中，我们将提供有关如何在 Ubuntu 18.04 上安装 Wine 4.0 的分步说明。

首先，使用以下 wget 命令将 WineHQ 存储库的 GPG 密钥导入您的系统：

wget -qO- https://dl.winehq.org/wine-builds/winehq.key | sudo apt-key add -

通过运行以下命令添加 WineHQ 存储库：

sudo apt install software-properties-common
sudo apt-add-repository 'deb http://dl.winehq.org/wine-builds/ubuntu/ bionic main'

键入以下命令安装 Wine 4.0 软件包：

sudo apt-get install --install-recommends winehq-stable

键入以下命令验证 Wine 安装：

wine --version

输出应该如下所示：

wine-4.0

配置 Wine

要配置 Wine，请在终端中运行命令 winecfg，该命令将安装 Mono 和 Gecko 并设置 Wine 环境：

winecfg

将出现一个对话框，询问您是否要安装 mono 。

单击 “Install” 按钮，安装将开始。完成后，将出现一个新对话框，提示您安装 Gecko 。再次单击 “Install” 按钮。

在 Wine 配置对话框中，您可以配置各种 Wine 设置。在大多数情况下，默认设置应该没问题完成后，关闭对话框。

安装 Notepad ++

现在已经安装并配置了 Wine ，我们将向您展示如何在 Ubuntu 上安装 Windows 应用程序。

在本教程中，我们将安装 Notepad++，它是 Windows 用户中最受欢迎的文本编辑器之一。启动浏览器并从 Notepad++ 下载页面下载 Notepad++ 安装文件。

下载完成后，右键单击 .exe 文件并选择 “Open With Wine Windows Program Loader”，开始安装。

将启动安装向导，选择语言并以与在 Windows 计算机上安装任何应用程序相同的方式安装 Notepad++ 。

Windows 应用程序安装在 ~/.wine/drive_c/ 目录中。

要启动 Notepad++ 编辑器，请导航到 ~/.wine/drive_c/Program Files (x86)/Notepad++ 目录。

然后双击该 notepad++.exe 文件。

结论

您已在 Ubuntu 18.04 桌面上成功安装并配置了 Wine 。我们还向您展示了如何在 Wine 下安装和启动 Windows 应用程序。
