如何在 Ubuntu 18.04 上安装 Visual Studio 代码

2019年5月24日 • 约 600 字 , 预计阅读时间 4 分钟

Visual Studio Code 是 Microsoft 开发的一种开源跨平台代码编辑器。它具有内置的调试支持，内置了 Git 控件，语法高亮，代码完成，集成终端，代码重构和代码片段。

在 Ubuntu 机器上安装 Visual Studio Code 的最简单和推荐的方法是启用 VS Code 存储库并通过命令行安装 VS Code 包。

虽然本教程是为 Ubuntu 18.04 编写的，但 Ubuntu 16.04 可以使用相同的步骤。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Visual Studio 代码

要在 Ubuntu 系统上安装 Visual Studio 代码，请按照下列步骤操作：

  1. 首先，通过键入以下内容来更新软件包索引并安装依赖项：

    sudo apt update
sudo apt install software-properties-common apt-transport-https wget

  2. 接下来，使用以下 wget 命令导入 Microsoft GPG 密钥：

    wget -q https://packages.microsoft.com/keys/microsoft.asc -O- | sudo apt-key add -

    并键入以下命令启用 Visual Studio Code 存储库：

    sudo add-apt-repository "deb [arch=amd64] https://packages.microsoft.com/repos/vscode stable main"

  3. 启用存储库后，使用以下命令安装最新版本的 Visual Studio 代码：

    sudo apt install code

现在，Visual Studio Code 已安装在您的 Ubuntu 桌面上，您可以开始使用它。

启动 Visual Studio 代码

现在，在您的 Ubuntu 系统上安装了 VS Code ，您可以通过键入 code 或单击 VS Code 图标（Activities -> Visual Studio Code）从命令行启动它。

您现在可以根据自己的喜好开始安装扩展和配置 VS 代码。

更新 Visual Studio 代码

发布新版本时，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Visual Studio Code 包：

sudo apt update
sudo apt upgrade

结论

您已在 Ubuntu 18.04 计算机上成功安装了 VS Code 。您的下一步可能是安装其他组件并自定义您的用户和工作区设置。

要了解有关 VS Code 的更多信息，请访问其官方文档页面。
