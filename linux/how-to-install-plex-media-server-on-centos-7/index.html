如何在 CentOS 7 上安装 Plex Media Server

April 10, 2019

Plex 是一种流媒体服务器，可将您的所有视频，音乐和照片集合在一起，并随时随地将其传输到您的设备。

在本教程中，我们将向您展示如何在 CentOS 7 上安装和配置 Plex Media Server。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

安装 Plex Media Server

在 CentOS 7 上安装和管理 Plex Media Server 的推荐方法是使用 Plex 官方存储库。它不需要技术知识，安装和配置媒体服务器不应超过 20 分钟。

以下步骤描述了如何在 CentOS 系统上安装 Plex Media Server ：

  1. 添加 Plex 存储库

    要启用 Plex 存储库，请打开文本编辑器并在 /etc/yum.repos.d/ 目录中创建一个 新的 YUM 存储库配置文件plex.repo：

    文件 /etc/yum.repos.d/plex.repo 的内容:

    [PlexRepo]
name=PlexRepo
baseurl=https://downloads.plex.tv/repo/rpm/$basearch/
enabled=1
gpgkey=https://downloads.plex.tv/plex-keys/PlexSign.key
gpgcheck=1
    1. 安装 Plex

    使用以下命令安装最新版本的 Plex Media Server ：

    sudo yum install plexmediaserver

    安装完成后，启动 plexmediaserver 服务并使用以下命令使其在系统引导时启动：

    sudo systemctl start plexmediaserver.service
sudo systemctl enable plexmediaserver.service
    1. 验证安装

    要验证 Plex 服务是否正在运行，请执行以下操作：

    sudo systemctl status plexmediaserver
    ● plexmediaserver.service - Plex Media Server for Linux
   Loaded: loaded (/usr/lib/systemd/system/plexmediaserver.service; enabled; vendor preset: disabled)
   Active: active (running) since Sat 2018-07-21 22:22:22 UTC; 12s ago
 Main PID: 13940 (Plex Media Serv)
   CGroup: /system.slice/plexmediaserver.service

    调整防火墙规则

    现在 Plex 已安装并正在运行，您需要配置防火墙以允许 Plex Media Server 特定端口上的流量。

    如果您的系统未启用防火墙，则可以跳过本节。

    打开您选择的文本编辑器并创建以下 Firewalld 服务文件 /etc/firewalld/services/plexmediaserver.xml：

    <?xml version="1.0" encoding="utf-8"?>
<service version="1.0">
<short>plexmediaserver</short>
<description>Plex TV Media Server</description>
<port port="1900" protocol="udp"/>
<port port="5353" protocol="udp"/>
<port port="32400" protocol="tcp"/>
<port port="32410" protocol="udp"/>
<port port="32412" protocol="udp"/>
<port port="32413" protocol="udp"/>
<port port="32414" protocol="udp"/>
<port port="32469" protocol="tcp"/>
</service>

保存文件并键入以下命令应用新的防火墙规则：

sudo firewall-cmd --add-service=plexmediaserver --permanent
sudo firewall-cmd --reload

最后检查新防火墙规则是否成功应用于：

sudo firewall-cmd --list-all
public (active)
  target: default
  icmp-block-inversion: no
  interfaces: eth0
  sources:
  services: ssh dhcpv6-client plexmediaserver
  ports:
  protocols:
  masquerade: no
  forward-ports:
  source-ports:
  icmp-blocks:
  rich rules:

配置 Plex 媒体服务器

创建用于保存媒体文件的目录：

sudo mkdir -p /opt/plexmedia/{movies,series}

Plex Media Server 作为用户 plex 运行，该用户需要具有对媒体文件和目录的读取和执行权限。要设置正确的所有权，请运行以下命令。

sudo chown -R plex: /opt/plexmedia

您可以选择任何位置存储媒体文件，只需确保设置正确的权限即可。

打开浏览器，键入 http://YOUR_SERVER_IP:32400/web ，您将看到一个设置向导，它将指导您通过 Plex 配置：

要使用 Plex Media Server ，您需要创建一个帐户。

按下 Google ， Facebook 按钮使用第三方账户或按下电子邮件按钮创建免费的 Plex 帐户。如果您想要访问高级功能，可以购买 Plex Pass 计划。

注册后，您将被重定向到如下页面，其中包含有关 Plex 如何工作的信息，如下所示：

单击 Got it 按钮。

在下一个屏幕上输入您的 Plex 服务器名称， 选中该复选框 Allow me to access my media outside my home，然后单击 Next 。

下一步是添加媒体库。单击 Add Library 按钮。

弹出窗口显示时，选择电影作为库类型并单击 Next 。

在下一步中，单击 Browse for media folder 并添加包含 Movies 媒体文件的目录的路径，在我们的示例中是 /opt/plexmedia/movies 。

单击 Add 按钮，然后单击 Add Library 。

您可以根据需要添加任意数量的库。

单击 Next ，然后 Done 您将被重定向到 Plex Web 仪表板。

现在您已经完成了设置向导，您可以开始探索 Plex 选项及其可以执行的所有操作。

结论

您已经学习了如何在 CentOS 7 计算机上安装 Plex Media Server 以及如何创建媒体库。

您现在可以在 Android ， iPhone ，智能电视， Xbox ， Roku 或任何其他受支持的设备上安装 Plex 应用程序。您可以在 Plex 下载页面上找到支持的应用程序和设备列表，也可以从设备的应用程序商店中安装该应用程序。

您还应该查看官方的 Plex 快速入门指南和 Plex 文档页面。
