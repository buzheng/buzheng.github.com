如何在Ubuntu上安装Deb包

2019年5月9日 • 约 1400 字 , 预计阅读时间 5 分钟

Deb是所有基于Debian的发行版使用的安装包格式。可以使用命令行或图形包管理器安装和卸载Deb软件包。

在本教程中，我们将解释如何.deb在Ubuntu 18.04上安装debian 软件包。相同的说明适用于Ubuntu 16.04和任何基于Debian的发行版，包括Debian，Linux Mint和Elementary OS。

在你开始之前

Ubuntu存储库包含数千个deb软件包，可以从Ubuntu软件中心安装。如果您更喜欢从命令行安装，则可以使用apt命令行实用程序。

但是有些应用程序未包含在Ubuntu或任何第三方存储库中。这些应用程序必须从开发人员网站下载并手动安装。

从非官方来源安装deb软件包时要格外小心。

正在下载deb软件包

出于演示目的，在本教程中，我们将下载并安装TeamViewer deb文件。TeamViewer是一款一体化解决方案，适用于远程支持，桌面共享，在线会议以及计算机之间的文件传输。

启动Web浏览器并导航到TeamViewer for Linux 下载页面。点击Ubuntu和Debian下载链接下载deb软件包。

如果您更喜欢使用终端，可以使用wget或下载deb文件curl：

wget --no-check-certificate https://download.teamviewer.com/download/linux/teamviewer_amd64.deb

curl -k -O -L https://download.teamviewer.com/download/linux/teamviewer_amd64.deb

从命令行安装deb软件包

从命令行安装deb软件包时，您可以使用多种工具。在下面的章节中，我们将向您展示如何使用apt，gdebi以及dpkg实用程序安装deb包。

使用apt安装deb软件包

APT（高级软件包工具）是软件包管理系统的命令行界面。它是在Ubuntu 14.04中引入的，它结合了最常用的命令apt-get和apt-cache。

要安装本地deb软件包，apt需要提供deb软件包的完整路径。如果程序包位于当前工作目录中，而不是使用绝对路径，则可以./在程序包名称之前添加前缀。否则apt将尝试从Ubuntu的存储库中检索并安装该软件包。

sudo apt install ./teamviewer_amd64.deb

系统将提示您键入Y以继续：

...
0 upgraded, 84 newly installed, 0 to remove and 64 not upgraded.
Need to get 21.0 MB of archives.
After this operation, 66.3 MB of additional disk space will be used.
Do you want to continue? [Y/n]

该apt软件包管理器将解决并安装所有的包依赖关系。

使用gdebi安装deb软件包

gdebi是一个安装本地deb包的工具。它在Ubuntu中默认不安装，但您可以从Ubuntu软件中心或使用以下命令安装它：

sudo apt install gdebi

您现在可以gdebi通过键入以下内容来安装deb软件包

sudo gdebi teamviewer_amd64.deb

...
Do you want to install the software package? [y/N]:

y在提示时键入，gdebi将解析并安装deb包及其所有依赖项。

使用dpkg安装deb软件包

dpkg是基于Debian的系统的低级包管理器。使用-i（或--install）选项安装deb软件包dpkg。

sudo dpkg -i teamviewer_amd64.deb

与apt和不同gdebi，dpkg并不解决依赖关系。如果在安装deb软件包时遇到任何依赖性错误，可以使用以下apt命令来解析和安装所有软件包依赖项：

sudo apt install -f

使用GUI安装deb软件包

如果您更喜欢使用图形界面，只需下载deb文件并双击打开即可。

这将打开默认的发行版图形软件中心

单击“安装”按钮，将打开“身份验证”对话框。

输入管理密码，然后单击“验证”按钮。

安装可能需要一些时间，具体取决于文件大小及其依赖性。安装deb软件包后，Ubuntu软件中心内的Install按钮将更改为Remove。

多数情况下，应用程序已安装在您的系统上，您可以从应用程序菜单启动它。

结论

在本教程中，您学习了如何安装本地deb软件包。
