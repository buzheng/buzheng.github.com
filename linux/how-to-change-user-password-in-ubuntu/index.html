如何在Ubuntu中更改用户密码

May 9, 2019

定期更新密码并为每个帐户使用唯一密码始终是个好主意。

作为Ubuntu的普通用户，您只能更改自己的密码。root用户和具有sudo权限的用户可以更改其他用户的密码，并定义如何使用或更改密码。

本教程介绍如何从命令行或通过Ubuntu GUI更改Ubuntu中的用户密码。

Ubuntu从命令行更改密码

在Ubuntu和其他Linux发行版中，您可以使用该passwd命令更改用户帐户的密码。

要更改用户帐户密码，请运行passwd不带任何选项的命令：

passwd

系统将提示您输入当前密码。如果密码正确，则会要求您输入并确认新密码。

Changing password for linuxize.
(current) UNIX password: 
Enter new UNIX password: 
Retype new UNIX password: 
passwd: password updated successfully

键入时，屏幕上不显示密码。

下次登录Ubuntu计算机时，请使用新密码。

更改另一个用户的密码

您登录的用户必须具有sudo权限才能更改或重置其他用户的密码。

要更改其他用户帐户的密码，请运行passwd命令后跟用户名。例如，要更改名为的用户的密码sansa，请运行以下命令：

sudo passwd sansa

系统将提示您输入并确认新密码：

Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully

Ubuntu从GUI更改密码

如果您不喜欢命令行，则可以通过GUI重置用户密码。

Gnome是Ubuntu 18.04（Bionic Beaver）中的默认桌面环境。以前版本的Ubuntu使用Unity作为默认桌面环境。

请按照以下步骤更改您的用户密码：

  1. 单击“设置”图标打开系统“设置”窗口，如下图所示。

  2. 在系统设置窗口中，单击“用户”选项卡。

    If you want to change other users passwords you need to Unlock the panel which requires sudo (administrator) privileges.

  3. 单击·····旁边的输入，打开“更改密码”窗口Password。

  4. 输入您当前的密码，然后输入并验证新密码。

    To automatically generate a random password, click on the double gear icon, next to `New Password`.

  5. 单击Change以保存新密码。

而已！您已成功更改密码。

结论

在本教程中，您学习了如何在Ubuntu中更改用户密码。

更改密码时，请确保使用强大且唯一的密码。拥有强密码是您帐户安全性的最重要方面。强密码通常至少包含16个字符，至少使用一个大写字母，一个小写字母，一个数字和一个特殊字符。

password Ubuntu terminal
