如何在 Linux 中压缩文件和目录

March 19, 2019

ZIP 是使用最广泛的归档文件格式，支持无损数据压缩。

ZIP 文件是包含一个或多个压缩文件或目录的数据容器。压缩的 zip 文件占用较少的磁盘空间，可以比未压缩文件更快地从一台机器传输到另一台机器。借助于适当的工具，可以在 Windows ， macOS 和 Linux 等所有操作系统轻松提取 ZIP 文件。

在本教程中，我们将向您展示如何使用 zip 实用程序通过命令行在 Linux 系统中压缩(压缩)文件和目录。

Zip 命令

Zip 是一个帮助您创建 ZIP 存档的实用程序。

zip 命令采用以下语法形式：

zip OPTIONS ARCHIVE_NAME FILES

要在特定目录中创建 ZIP 存档，用户需要具有该目录的写入权限。 ZIP 文件不支持 Linux 样式的所有权信息，并且所有提取的文件将由运行该命令的用户拥有。如果要保留文件所有者和权限，请使用 tar 命令。

默认情况下，大多数 Linux 发行版中都没有安装 zip 实用程序，但您可以使用发行版的软件包管理器轻松安装它。

在 Ubuntu 和 Debian 上安装 Zip

sudo apt install zip

在 CentOS 和 Fedora 上安装 Zip

sudo yum install zip

如何压缩文件和目录

要压缩一个或多个文件，请指定要添加存档的文件，多个文件使用空格分隔，如下所示：

zip archivename.zip filename1 filename2 filename3
adding: filename1 (deflated 63%)
adding: filename2 (stored 0%)
adding: filename3 (deflated 38%)

默认情况下， zip 命令打印添加到存档的所有文件的名称和压缩方法。我们将在本指南后面解释压缩方法和级别。

如果您指定了 zip 存档名称，那么将自动添加 .zip 后缀名，除非存档名称包含点。 zip archivename.zip filename 创建的存档的名称与 zip archivename filename 相同。

如果要不需要打印 zip 命令的输出，请使用以下 `-q` 选项：

zip -q archivename.zip filename1 filename2 filename3

通常，您在创建目录的压缩文件时，需要包括子目录的内容。而 -r 选项允许您递归遍历整个目录结构。使用 zip 压缩目录时：

zip -r archivename.zip directory_name

您还可以在同一压缩文件中添加多个文件和目录：

zip -r archivename.zip directory_name1 directory_name2 file1 file1

压缩方法和级别

zip 默认压缩方法是 deflate 。如果 zip 实用程序确定无法压缩文件，则只需将文件存储在存档中，而不使用 store 方法对其进行压缩。在大多数 Linux 发行版中， zip 实用程序还支持 bzip2 压缩方法。

要指定压缩方法，请使用该 -Z 选项。

zip -r -Z bzip2 archivename.zip directory_name
...
adding: sub_dir/ (stored 0%)
adding: sub_dir/file1 (bzipped 52%)
adding: sub_dir/file2 (bzipped 79%)

zip 命令允许您使用前缀为短划线的数字 0 到 9 指定压缩级别。默认压缩级别为 -6 。使用 -0 时所有文件将无需压缩即可存储。 -9 将强制 zip 命令对所有文件使用最佳压缩。

例如，要使用压缩级别 -9 ， 您可以键入以下内容：

zip -9 -r archivename.zip directory_name

压缩级别越高， zip 进程占用的 CPU 越多，完成时间就越长。

创建受密码保护的 ZIP 文件

如果您有需要存储在存档中的敏感信息，可以使用以下 -e 选项对其进行加密：

zip -e  archivename.zip directory_name

系统将提示您输入并验证存档密码：

Enter password:
Verify password:

创建拆分 Zip 文件

想象一下这样的场景，您希望将 zip 存档存储在文件托管服务上，文件大小上载限制为 1GB ， ZIP 存档为 5GB 。

您可以使用 -s 选项后跟指定的大小来创建新的拆分的 Zip 文件。单位可以是 k (千字节)， m (兆字节)， g (千兆字节) 或 t (兆兆字节)。

zip -s 1g -r archivename.zip directory_name

上面的命令将在达到指定的大小限制后继续在集合中创建新的存档。

archivename.zip
archivename.z01
archivename.z02
archivename.z03
archivename.z04

ZIP 示例

把当前目录中的所有文件，压缩进名为 archivename.zip 的 ZIP 存档，。

zip archivename *

与上面相同，包括隐藏文件(以点开头的文件)：

zip archivename .* *

创建一个名为 archivename.zip 的 ZIP 存档，其中包含当前目录中的所有 MP3 文件，不压缩文件。

zip -0 archivename *.mp3

结论

在本教程中，您学习了如何使用 zip 命令创建 ZIP 存档。我们还向您展示了 zip 命令的最常见选项。

如果您想了解更多关于 Unzip 的信息，请访问 Zip Man 页面。
