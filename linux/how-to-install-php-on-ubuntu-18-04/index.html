如何在 Ubuntu 18.04 上安装 PHP

2019年3月28日 • 约 1100 字 , 预计阅读时间 5 分钟

在本教程中，我们将介绍在 Ubuntu 18.04 上安装发行版中的默认的 PHP 7.2 并将其与 Nginx 和 Apache 集成所需的步骤。我们还将向您展示如何安装 PHP 7.1 和 7.3 。

大多数流行的 PHP 框架和应用程序都与 PHP 7.2 兼容，包括 WordPress ， Laravel ， Drupal 和 Nextcloud 。

先决条件

在开始本教程之前，请确保以具有 sudo 权限的用户身份登录。

使用 Apache 安装PHP 7.2

如果您使用 Apache 作为 Web 服务器来安装 PHP 和 Apache PHP 模块，请运行以下命令：

sudo apt install php libapache2-mod-php

安装软件包后，重新启动 Apache 服务：

sudo systemctl restart apache2

在 Nginx 中配置 PHP 7.2

与 Apache 不同，Nginx没有内置的处理 PHP 文件的支持，所以我们需要安装一个单独的应用程序，如 PHP FPM （fastCGI 进程管理器），用它来处理 PHP 文件。

要安装 PHP 和 PHP FPM 包，请运行以下命令：

sudo apt install php-fpm

安装软件包后，您可以使用以下命令检查 PHP FPM 服务的状态：

systemctl status php7.2-fpm

     Docs: man:php-fpm7.2(8)

接下来编辑 Nginx 服务器块并添加以下行，以便 Nginx 可以处理 PHP 文件：

server {

    # . . . other code

    location ~ \.php$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/run/php/php7.2-fpm.sock;
    }
}

不要忘记重新启动 Nginx 服务，以使新配置生效：

sudo systemctl restart nginx

安装 PHP 扩展

要扩展 PHP 的核心功能，您可以安装一些额外的扩展。 PHP 扩展以包的形式提供，可以通过以下方式轻松安装：

sudo apt install php-[extname]

例如，如果要安装 MySQL 和 GD PHP 扩展，则可以运行以下命令：

sudo apt install php-mysql php-gd

安装新的 PHP 扩展后，请不要忘记重新启动 Apache 或 PHP FPM 服务，具体取决于您的设置。

测试 PHP

要测试您的 Web 服务器是否已正确配置，能否正确处理 PHP 文件，请使用以下代码在目录 /var/www/html 中创建名为 info.php 的新文件 /var/www/html/info.php：

<?php

phpinfo();

保存文件，打开您选择的浏览器并访问 http://your_server_ip/info.php

在 Ubuntu 18.04 上安装 PHP 7.3

PHP 7.3 是 PHP 的最新稳定版本。执行以下步骤在 Ubuntu 18.04 上安装 PHP 7.3 。

  1. 首先启用 Ondrej PHP 存储库：

    sudo apt install software-properties-commonsudo add-apt-repository ppa:ondrej/php

  2. 安装 PHP 7.3 和一些最常见的 PHP 模块：

    sudo apt install php7.3 php7.3-common php7.3-opcache php7.3-cli php7.3-gd php7.3-curl php7.3-mysql

  3. 要验证安装，请运行以下命令以打印 PHP 版本：

    php -v

在 Ubuntu 18.04 上安装 PHP 7.1

仅当您要安装与 PHP 7.2 不兼容的应用程序时才使用 PHP 7.1 。

按照以下步骤安装 PHP 7.1 ：

  1. 键入以下命令启用 Ondrej PHP 存储库：

    sudo apt install software-properties-commonsudo add-apt-repository ppa:ondrej/php

  2. 安装 PHP 7.1 和几个最常见的 PHP 模块：

    sudo apt install php7.1 php7.1-common php7.1-opcache php7.1-mcrypt php7.1-cli php7.1-gd php7.1-curl php7.1-mysql

  3. 键入以下命令验证安装：

    php -v

    PHP 7.1.20-1+ubuntu18.04.1+deb.sury.org+1 (cli) (built: Jul 25 2018 10:07:09) ( NTS )

  with Zend OPcache v7.1.20-1+ubuntu18.04.1+deb.sury.org+1, Copyright (c) 1999-2018, by Zend Technologies

结论

您已在 Ubuntu 18.04 服务器上成功安装了 PHP 。您还可以阅读有关如何使用 Composer 安装和设置 PHP 项目的信息。
