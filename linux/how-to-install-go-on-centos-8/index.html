在 CentOS 8 上安装 go 语言

2019年11月22日 • 约 700 字 , 预计阅读时间 4 分钟

Go (通常称为 golang) 是 Google 创建的一种现代开源编程语言，可让您构建可靠且高效的应用程序。

Go 编写了许多流行的应用程序，例如 Kubernetes ， Docker ， Prometheus 和 Terraform 。

本教程说明了如何在 CentOS 8 系统上下载和安装 Go 。

在 CentOS 8 上下载并安装 Go

在撰写本文时， Go 的最新稳定版本为 1.13.4 。在下载压缩包之前，请访问官方的 Go  下载页面，并检查是否有可用的新版本。

请按照以下步骤在 CentOS 8 上下载并安装 Go ：

  1. 使用 wget 或 curl 下载 Go 二进制文件：

    wget https://dl.google.com/go/go1.13.4.linux-amd64.tar.gz

  2. 下载归档文件后，请输入以下内容来验证 tarball 校验和：

    sha256sum go1.13.4.linux-amd64.tar.gz

    确保 sha256sum 命令打印的哈希值与下载页面中的哈希值匹配。

    692d17071736f74be04a72a06dab9cac1cd759377bd85316e52b2227604c004c  go1.13.4.linux-amd64.tar.gz

  3. 使用以下 tar 命令将压缩包解压缩到 /usr/local 目录中：

    sudo tar -C /usr/local -xf go1.13.4.linux-amd64.tar.gz

    上面的命令必须以 root 用户或具有 sudo 特权的用户身份运行。

  4. 通过调整 $PATH 环境变量告诉系统在哪里可以找到 Go 可执行二进制文件。

    您可以通过将以下行添加到 /etc/profile 文件(对于系统范围的安装)或 $HOME/.bash_profile 文件(对于当前用户安装)来实现：

    export PATH=$PATH:/usr/local/go/bin

    保存文件，并使用以下 source 命令将新的 PATH 环境变量加载到当前的 shell 会话中：

    source ~/.bash_profile

至此， Go 已经安装在您的 CentOS 系统上。

测试安装

为了测试 Go 是否正确安装，我们将设置一个工作区并构建一个简单的 “Hello world” 程序。

  1. 工作区目录的位置由 GOPATH 环境变量指定。默认情况下，它设置为 $HOME/go 。要创建目录，请运行以下命令：

    mkdir ~/go

  2. 在工作空间内创建一个新目录 src/hello ：

    mkdir -p ~/go/src/hello

    在该目录中创建一个名 hello.go 为的文件：

    nano ~/go/src/hello/hello.go

    将以下代码粘贴到 ~/go/src/hello/hello.go 文件中：

    package main

import "fmt"

func main() {
    fmt.Printf("Hello, World\n")
}

  3. 导航到 ~/go/src/hello 目录并运行 go build 以构建代码：

    cd ~/go/src/hello
go build

    上面的命令将构建一个名为 hello 的可执行文件。

  4. 通过键入以下命令来运行可执行文件：

    ./hello

    如果看到以下输出，则说明您已经成功安装了 Go 。

    Hello, World

结论

现在，您已经下载并安装了 Go ，现在可以开始编写 Go 代码了。
