如何在 Debian 9 上安装 TeamViewer

2019年6月17日 • 约 600 字 , 预计阅读时间 4 分钟

TeamViewer 是一种跨平台解决方案，可用于远程控制，桌面共享，在线会议和计算机之间的文件传输。

TeamViewer 是专有的计算机软件，它不包含在默认的 Debian 存储库中。本教程描述了如何在 Debian 9 上安装 TeamViewer 。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

在 Debian 上安装 TeamViewer

在 Debian 系统上安装 TeamViewer 是一个非常简单的过程，只需完成以下步骤即可。

1.下载 TeamViewer

使用 Ctrl+Alt+T 键盘快捷键或单击终端图标打开终端。

运行以下 wget 命令以下载最新的 TeamViewer 的 .deb 包：

wget https://download.teamviewer.com/download/linux/teamviewer_amd64.deb

2.安装 TeamViewer

要安装下载的 TeamViewer   .deb 包，请键入以下命令：

sudo apt install ./teamviewer_amd64.deb

出现提示时 Do you want to continue ? [Y/n] ，键入 Y 以继续安装。

启动 TeamViewer

您可以通过键入 teamviewer 或单击“活动”菜单中的图标从命令行启动 TeamViewer 应用程序。

首次启动 TeamViewer 时，将显示类似于以下内容的窗口。单击 License Agreement 按钮接受许可协议中的条款。

应用程序将启动，您将看到类似以下内容。

此时，您在 Debian 桌面上安装了 TeamViewer 。您现在可以连接到您的朋友或客户计算机，并提供即时技术帮助。

更新 TeamViewer

在安装过程中，官方 TeamViewer 存储库将添加到您的系统中。您可以使用 cat 命令验证文件内容：

cat /etc/apt/sources.list.d/teamviewer.list
...
deb http://linux.teamviewer.com/deb stable main
# deb http://linux.teamviewer.com/deb preview main
...

这可确保在通过桌面标准软件更新工具发布新版本时， TeamViewer 安装将自动更新。

结论

在本教程中，我们向您展示了如何在 Debian 9 台式机上安装 TeamViewer 。使用 TeamViewer ，您可以连接到任何远程计算机，并像在坐在机器前面一样对其进行操作。
