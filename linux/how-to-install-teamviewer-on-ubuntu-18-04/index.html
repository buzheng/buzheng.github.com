如何在Ubuntu 18.04上安装TeamViewer

May 9, 2019

TeamViewer是一种跨平台解决方案，可用于计算机之间的远程控制，桌面共享和文件传输。

TeamViewer是专有的计算机软件，它不包含在Ubuntu存储库中。本教程介绍如何在Ubuntu 18.04上安装TeamViewer。相同的说明适用于Ubuntu 16.04和任何其他基于Debian的发行版，包括Debian，Linux Mint和Elementary OS。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

在Ubuntu上安装TeamViewer

在撰写本文时，TeamViewer的最新稳定版本是版本14.0.12762。

执行以下步骤在Ubuntu系统上安装TeamViewer：

1.下载TeamViewer

使用Ctrl+Alt+T键盘快捷键或单击终端图标打开终端。

.deb使用以下wget命令下载最新的TeamViewer 包：

wget https://download.teamviewer.com/download/linux/teamviewer_amd64.deb

2.安装TeamViewer

.deb以具有sudo权限的用户身份发出以下命令来安装TeamViewer 程序包：

sudo apt install ./teamviewer_amd64.deb

在提示符下Do you want to continue? [Y/n]，键入Y以继续安装。

启动TeamViewer

现在TeamViewer已安装在您的Ubuntu系统上，您可以通过键入teamviewer或单击TeamViewer图标（Activities -> TeamViewer）从命令行启动它：

首次启动TeamViewer时，将显示类似于以下内容的窗口。单击License Agreement按钮接受许可协议中的条款。

TeamViewer将打开，您将看到类似以下内容。

恭喜，此时您已在Ubuntu桌面上安装了TeamViewer。您现在可以连接到您的朋友或客户机器并立即提供帮助。

更新TeamViewer

在安装过程中，官方TeamViewer存储库将添加到您的系统中。您可以使用cat命令验证文件内容：

cat /etc/apt/sources.list.d/teamviewer.list
...
deb http://linux.teamviewer.com/deb stable main
# deb http://linux.teamviewer.com/deb preview main
...

这可确保在通过桌面标准软件更新工具发布新版本时，TeamViewer安装将自动更新。

结论

在本教程中，我们向您展示了如何在Ubuntu 18.04台式机上安装TeamViewer。
