如何在CentOS 7上安装Apache Maven

May 9, 2019

Apache Maven是一个免费的开源项目管理和理解工具，主要用于Java项目。Maven使用项目对象模型（POM），它本质上是一个XML文件，包含有关项目，配置详细信息，项目依赖项等的信息。

在本教程中，我们将向您展示在CentOS 7上安装Apache Maven的两种不同方法。

官方CentOS存储库包含可与yum程序包管理器一起安装的Maven程序包。这是在CentOS上安装Maven的最简单方法，但是存储库中包含的版本可能落后于最新版本的Maven。

要安装最新版本的Maven，请按照本文第二部分提供的说明从我们的官方网站下载Maven。

选择最适合您的设置和环境的安装方法。

先决条件

您登录的用户必须具有sudo权限才能安装软件包。

使用Yum在CentOS上安装Apache Maven

使用CentOS 7安装Maven yum是一个简单，直接的过程。

  1. 通过在终端中键入以下命令来安装Maven：

    sudo yum install maven
    1. 键入以下mvn -version命令验证安装：
    mvn -version

    输出应该如下所示：

    Apache Maven 3.0.5 (Red Hat 3.0.5-17)
Maven home: /usr/share/maven
Java version: 1.8.0_191, vendor: Oracle Corporation
Java home: /usr/lib/jvm/java-1.8.0-openjdk-1.8.0.191.b12-0.el7_5.x86_64/jre
Default locale: en_US, platform encoding: UTF-8
OS name: "linux", version: "3.10.0-862.3.2.el7.x86_64", arch: "amd64", family: "unix"

    而已。Maven现已安装在您的CentOS系统上，您可以开始使用它。

    安装最新版本的Apache Maven

    以下部分提供了有关如何在CentOS 7上安装最新Apache Maven版本的分步说明。我们将从其官方网站下载最新版本的Apache Maven。

    1.安装OpenJDK

    Maven 3.3+需要安装JDK 1.7或更高版本。我们将安装OpenJDK，它是CentOS 7中的默认Java开发和运行时。

    输入以下命令安装OpenJDK包：

    java-1.8.0-openjdk

通过运行以下命令验证是否已成功安装Java：

java -version

输出应该如下所示：

openjdk version "1.8.0_191"
OpenJDK Runtime Environment (build 1.8.0_191-b12)
OpenJDK 64-Bit Server VM (build 25.191-b12, mixed mode)

如果您更喜欢Oracle Java而不是OpenJDK，请按照以下说明安装Oracle Java。

2.下载Apache Maven

在撰写本文时，最新版本的Apache Maven是3.6.0。在继续下一步之前，您应该检查Maven下载页面以查看是否有更新的版本。

首先/tmp使用以下wget命令在目录中下载Apache Maven ：

wget https://www-us.apache.org/dist/maven/maven-3/3.6.0/binaries/apache-maven-3.6.0-bin.tar.gz -P /tmp

下载完成后，在目录中解压缩归档/opt：

sudo tar xf /tmp/apache-maven-3.6.0.tar.gz -C /opt

为了更好地控制Maven版本和更新，我们将创建一个maven指向Maven安装目录的符号链接：

sudo ln -s /opt/apache-maven-3.6.0 /opt/maven

要升级Maven安装，只需解压缩较新版本并更改符号链接即可指向它。

3.设置环境变量

接下来，我们需要设置环境变量。打开文本编辑器并创建一个名为目录mavenenv.sh内的新文件/etc/profile.d/。

sudo nano /etc/profile.d/maven.sh

粘贴以下行：

/etc/profile.d/maven.sh

export JAVA_HOME=/usr/lib/jvm/jre-openjdk
export M2_HOME=/opt/maven
export MAVEN_HOME=/opt/maven
export PATH=${M2_HOME}/bin:${PATH}

保存并关闭文件。此脚本将在shell启动时获取。

键入以下命令使脚本可执行：

sudo chmod +x /etc/profile.d/maven.sh

键入以下命令加载环境变量：

source /etc/profile.d/maven.sh

4.验证安装

要验证是否已安装Maven，请使用mvn -version将打印Maven版本的命令：

mvn -version

您应该看到如下内容：

Apache Maven 3.6.0 (97c98ec64a1fdfee7767ce5ffb20918da4f719f3; 2018-10-24T18:41:47Z)
Maven home: /opt/maven
Java version: 1.8.0_191, vendor: Oracle Corporation, runtime: /usr/lib/jvm/java-1.8.0-openjdk-1.8.0.191.b12-0.el7_5.x86_64/jre
Default locale: en_US, platform encoding: UTF-8
OS name: "linux", version: "3.10.0-862.3.2.el7.x86_64", arch: "amd64", family: "unix"

而已。最新版本的Maven现已安装在您的CentOS系统上。

结论

您已经在CentOS 7上成功安装了Apache Maven。您现在可以访问官方Apache Maven文档页面并了解如何开始使用Maven。
