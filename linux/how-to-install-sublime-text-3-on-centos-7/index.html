如何在CentOS 7上安装Sublime Text 3

May 9, 2019

Sublime Text是一个专有的跨平台多功能文本和源代码编辑器，用于Web和软件开发。它非常快，并且具有许多开箱即用的强大功能。您可以通过安装新插件和创建自定义设置来增强其功能。

本教程概述了在CentOS 7 Linux系统上安装Sublime Text 3的步骤。

先决条件

您登录的用户必须具有管理员级权限才能安装软件包。

在CentOS上安装Sublime文本

按照以下步骤在CentOS 7系统上安装Sublime Text 3：

  1. 首先导入官方Sublime Text存储库的GPG密钥：

    sudo rpm -v --import https://download.sublimetext.com/sublimehq-rpm-pub.gpg
    1. 使用以下命令将Yum存储库添加到系统中：
    sudo yum-config-manager --add-repo https://download.sublimetext.com/rpm/stable/x86_64/sublime-text.repo
    adding repo from: https://download.sublimetext.com/rpm/stable/x86_64/sublime-text.repo
grabbing file https://download.sublimetext.com/rpm/stable/x86_64/sublime-text.repo to /etc/yum.repos.d/sublime-text.repo
repo saved to /etc/yum.repos.d/sublime-text.repo
    1. 启用存储库后，键入以下命令安装Sublime Text 3：
    sudo yum install sublime-text

而已。Sublime Text已安装在CentOS桌面上。

Sublime Text是一个专有应用程序。它可以免费下载和评估。但是，如果您持续使用许可证，则需要购买许可证。评估没有强制执行时间限制。

结论

您已在CentOS 7计算机上成功安装了Sublime Text。您的下一步可能是安装Sublime Text Package Control并根据您的需要开始自定义新编辑器。

要查找有关Sublime Text的更多信息，请访问Sublime Text官方文档页面。
