如何在 CentOS 7 上安装 Java

2019年3月8日 • 约 1200 字 , 预计阅读时间 5 分钟

Java 是世界上最流行的编程语言之一，用于构建不同类型的应用程序和系统。

本教程描述了如何在 CentOS 7 上安装 Java 的各种版本和实现。我们将向您展示如何安装 OpenJDK 以及 Oracle Java 。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

Java 的版本和实现

Java 分为三个不同版本：标准版（SE），企业版（EE）和微型版（ME）。本教程介绍 Java SE （标准版）版。

CentOS 7 中有两个不同的 Java 包， Java Runtime Environment （JRE）和 Java Development Kit （JDK）。如果您只想运行 Java 程序，那么您需要仅仅是 JRE ，如果您是 Java 开发人员，那么您将需要 JRE 和包含开发/调试工具和库的 JDK 。

Java 还有两种不同的实现: OpenJDK 和 Oracle Java ，它们之间几乎没有区别，只是 Oracle Java 有一些额外的商业功能。

如果您不确定要安装哪个 Java 实现和版本，则一般建议是安装到 OpenJDK 8 JDK ，这是当前的 LTS 版本的 Java 。

先决条件

在开始本教程之前，请确保以具有 sudo 权限的用户身份登录。

安装 OpenJDK

OpenJDK 是 Java 平台的开源实现，也是 CentOS 7 中的默认 Java 开发和运行时。安装 OpenJDK 的过程很简单。

安装 OpenJDK 8 JDK

在撰写本文时， OpenJDK 8 是 Java 的当前 LTS 版本以及要安装的推荐版本。您可以通过键入以下命令使用 yum 安装它：

sudo yum install java-1.8.0-openjdk-devel

安装 OpenJDK 8 JRE

您可以通过键入以下命令使用 yum 安装 OpenJDK 8 JRE ：

sudo yum install java-1.8.0-openjdk

JRE 是 JDK 的子集，如果您已经安装了 JDK 软件包，则无需安装 JRE 软件包。

安装 OpenJDK 7 JDK

如果您的应用程序需要在 CentOS 7 服务器上安装较旧的 Java SE 7 ，请通过键入以下命令使用 yum 安装 OpenJDK 7 JDK 软件包：

sudo yum install java-1.7.0-openjdk-devel

安装 OpenJDK 7 JRE

如果您的应用程序需要 OpenJDK 7 JRE ，则可以通过键入以下命令使用 yum 来安装：

sudo yum install java-1.7.0-openjdk

安装 Oracle Java

在本节中，我们将介绍安装 Oracle Java 的步骤。 Oracle 软件包只能从其官方网站下载。

我们将使用 curl 下载所需的 Java 包。如果您尚未在系统上安装 curl ，则可以通过键入以下命令进行安装：

sudo yum install curl

在编写本文时，最新的 Oracle Java 版本是 8u162 （Java 8）和 9.0.4 （Java 9）。你如果想要安装最新版本安装请跳转到下载页面，接受许可协议，将 .rpm 软件包的下载链接复制到下面的 curl 命令中进行更改。

安装 Oracle Java 9 JRE

curl -L -b "oraclelicense=a" -O http://download.oracle.com/otn-pub/java/jdk/9.0.4+11/c2514751926b4512b076cc82f959763f/jre-9.0.4_linux-x64_bin.rpm
sudo yum localinstall jre-9.0.4_linux-x64_bin.rpm

安装 Oracle Java 9 JDK

curl -L -b "oraclelicense=a" -O http://download.oracle.com/otn-pub/java/jdk/9.0.4+11/c2514751926b4512b076cc82f959763f/jdk-9.0.4_linux-x64_bin.rpm
sudo yum localinstall jdk-9.0.4_linux-x64_bin.rpm

安装 Oracle Java 8 JRE

curl -L -b "oraclelicense=a" -O http://download.oracle.com/otn-pub/java/jdk/8u162-b12/0da788060d494f5095bf8624735fa2f1/jre-8u162-linux-x64.rpm
sudo yum localinstall jre-8u162-linux-x64.rpm

安装 Oracle Java 8 JDK

curl -L -b "oraclelicense=a" -O http://download.oracle.com/otn-pub/java/jdk/8u162-b12/0da788060d494f5095bf8624735fa2f1/jdk-8u162-linux-x64.rpm
sudo yum localinstall jdk-8u162-linux-x64.rpm

设置默认版本

您可以使用以下命令检查默认 Java ：

java -version

java version "9.0.4"
Java(TM) SE Runtime Environment (build 9.0.4+11)
Java HotSpot(TM) 64-Bit Server VM (build 9.0.4+11, mixed mode)

如果服务器上安装了多个 Java 版本，则可以使用 alternatives 系统实用程序更改默认版本：

sudo alternatives --config java

输出应类似于以下内容：

There are 4 programs which provide 'java'.

  Selection    Command
-----------------------------------------------
   1           /usr/java/jdk1.8.0_162/jre/bin/java
   2           java-1.8.0-openjdk.x86_64 (/usr/lib/jvm/java-1.8.0-openjdk-1.8.0.161-0.b14.el7_4.x86_64/jre/bin/java)
   3           java-1.7.0-openjdk.x86_64 (/usr/lib/jvm/java-1.7.0-openjdk-1.7.0.171-2.6.13.0.el7_4.x86_64/jre/bin/java)
*+ 4           /usr/java/jdk-9.0.4/bin/java

Enter to keep the current selection[+], or type selection number:

要更改默认 Java 版本，只需在提示时输入对应的数字并按下 Enter 。

卸载 Java

如果您出于任何原因想要卸载 Java 软件包，则可以使用 yum 像卸载任何其他软件包一样将其卸载。

例如，如果要卸载 java-1.8.0-openjdk-devel 软件包，只需运行：

sudo yum remove java-1.8.0-openjdk-devel

结论

现在您已经学习了如何在 CentOS 服务器上安装和管理不同的 Java 版本，下一步可能是安装运行在 Java 上的众多应用程序之一，例如 Tomcat ， JBoss / WildFly ， Jetty ， Glassfish ， WebLogic ， Cassandra ， Jenkins ，Gradle .. 等等。
