如何在 Ubuntu 18.04 上安装 Chromium 浏览器

June 16, 2019

Chromium 是一款快速，稳定，安全的浏览器，专为现代网络而打造。它是许多浏览器的基础，包括世界上最广泛使用的 Google Chrome 浏览器。

Chromium 和谷歌 Chrome 之间的主要区别在于 Chromium 是开源的。与 Chrome 不同， Chromium 不包含专有的网络编解码器，也不会向 Google 发送崩溃报告和使用情况跟踪。

如果您更喜欢开源，那么 Chromium 是一个很好的选择。

在本教程中，我们将解释如何在 Ubuntu 18.04 上安装 Chromium Web 浏览器。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。

先决条件

要安装软件包，您必须以具有 sudo 权限的用户身份登录。

在 Ubuntu 上安装 Chromium

Chromium 在默认的 Ubuntu 存储库中可用，安装过程非常简单。

使用 Ctrl+Alt+T 键盘快捷键或单击终端图标打开终端，然后运行以下命令安装铬包：

sudo apt install chromium-browser

系统将提示您输入密码并确认是否要继续安装。

就这样， Chromium 已安装在您的 Ubuntu 系统上。

启动 Chromium

您可以通过键入 chromium-browser 或单击 Chromium 图标 (Activities → Chromium) 从命令行启动 Chromium 浏览器：

您可以根据图标的颜色区分 Google Chrome 和 Chromium 。后者是蓝色的。

当您第一次启动浏览器时，您将看到默认的 Chromium 欢迎页面。

在此处，您可以登录自己的 Google 帐户，在所有设备上同步书签，历史记录，密码和其他设置。

更新 Chromium

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Chromium ：

sudo apt update
sudo apt upgrade

结论

在本教程中，我们向您展示了如何在 Ubuntu 18.04 台式机上安装 Chromium 。

如果您以前使用过其他浏览器，例如 Chrome 或 Opera ，则可以将书签和设置导入 Chromium 。
