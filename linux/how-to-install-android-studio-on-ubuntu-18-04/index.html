如何在 Ubuntu 18.04 上安装 Android Studio

2019年2月27日 • 约 900 字 , 预计阅读时间 4 分钟

Android Studio 是一个功能齐全的跨平台IDE，可帮助您构建各种类型的 Android 设备上的应用程序。它基于 JetBrains 的 IntelliJ IDEA，包括你进行 Android 应用开发所需的一切。

Android Studio 构建系统由 Gradle 提供支持，允许您从单个项目为不同设备创建多个构建变体。

本教程介绍如何在 Ubuntu 18.04 上安装 Android Studio 。相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Kubuntu，Linux Mint 和 Elementary OS。

先决条件

您需要以具有sudo访问权限的用户身份登录才能在 Ubuntu 系统上安装软件包。

安装Java OpenJDK

Android Studio 需要将运行在 OpenJDK 8 或更高版本的 Oracle Java 安装到您​​的系统中。

我们将安装 OpenJDK 8 。安装过程非常简单，首先要更新系统的软件包索引：

sudo apt update

输入以下命令安装 OpenJDK 8 软件包：

sudo apt install openjdk-8-jdk

通过键入以下将输出 Java 版本的命令来验证安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_191"
OpenJDK Runtime Environment (build 1.8.0_191-8u191-b12-2ubuntu0.18.04.1-b12)
OpenJDK 64-Bit Server VM (build 25.191-b12, mixed mode)

如果您更喜欢 Oracle Java 而不是 OpenJDK ，请安装 Oracle Java。

安装Android Studio

在撰写本文时，Android Studio 的最新稳定版本是 3.3.1.0 版本。在 Ubuntu 18.04 上安装 Android Studio 的最简单的方法是使用 snappy 打包系统。

要下载并安装 Android Studio snap 包，请使用 Ctrl+Alt+T 键盘快捷键打开终端并键入：

sudo snap install android-studio --classic

安装完成后，您将看到以下输出：

android-studio 3.3.1.0 from Snapcrafters installed

到此为止，Android Studio 已安装在您的 Ubuntu 桌面上。

启动Android Studio

您可以通过键入 android-studio 终端或单击 Android Studio 图标（Activities -> Android Studio）来启动 Android Studio 。

当您第一次启动 Android Studio 时，会出现如下所示的窗口，要求您从以前的安装中导入 Android Studio 设置：

单击该 OK 按钮后，将出现“安装向导”窗口。单击 Next 按钮以启动初始配置和安装后步骤。

接下来，系统将提示您为 Android Studio 选择所需的设置类型。您很可能想要选择“标准”选项。

在下一步，您可以选择 UI 主题，安装向导将下载并安装所需的SDK组件。此过程可能需要一些时间，具体取决于您的连接速度。

结论

您已经学习了如何在 Ubuntu 18.04 桌面上安装 Android Studio 。您现在可以浏览新的 IDE 并开始处理您的 Android 项目。

要查找有关如何使用Android Studio开发应用程序的更多信息，请访问 Android Studio 文档页面。
