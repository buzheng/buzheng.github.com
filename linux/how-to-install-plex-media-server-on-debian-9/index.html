如何在 Debian 9 上安装 Plex 媒体服务器

March 8, 2019

Plex 是一种流媒体服务器，可让您随时随地组织视频，音乐和照片集，并将其流式传输到您的所有设备。

本教程介绍如何在 Debian 9 上安装 Plex 媒体服务器。

先决条件

您登录的用户必须具有 sudo 权限才能进行安装。

安装 Plex 媒体服务器

按照以下步骤在 Debian 系统上安装 Plex 媒体服务器：

  1. 因为官方 Debian 软件包仓库中不提供 Plex 媒体服务器，所以我们要使用 Plex 官方仓库。首先使用以下 curl 命令导入仓库的 GPG 密钥：

    curl https://downloads.plex.tv/plex-keys/PlexSign.key | sudo apt-key add -

    通过以下的命令将 Plex APT 仓库添加到系统的软件仓库列表：

    echo deb https://downloads.plex.tv/repo/deb ./public main | sudo tee /etc/apt/sources.list.d/plexmediaserver.list

  2. 启用 Plex 仓库后，更新 apt 程序包列表并安装最新版本的 Plex 媒体服务器：

    sudo apt install apt-transport-https
sudo apt update
sudo apt install plexmediaserver
    1. 安装成功后，Plex 服务将自动启动。请使用以下命令，来验证安装是否成功，该命令会打印 Plex 服务器状态：
    sudo systemctl status plexmediaserver

    输出应该如下所示：

  Loaded: loaded (/lib/systemd/system/plexmediaserver.service; enabled; vendor preset: enabled)
  Active: active (running) since Thu 2019-03-07 06:35:51 CST; 17min ago

    Tasks: 127 (limit: 4915)

    调整防火墙规则

    现在您已在服务器上安装并运行 Plex ，您需要确保将防火墙配置为允许 Plex 媒体服务器特定端口上的流量。

    如果您的系统上没有运行防火墙，则可以跳过此部分。

    以下步骤的前提是假定您用 UFW 来管理防火墙：

    打开您选择的文本编辑器并创建 UFW 应用程序配置文件： /etc/ufw/applications.d/plexmediaserver

    [plexmediaserver]
title=Plex Media Server (Standard)
description=The Plex Media Server
ports=32400/tcp|3005/tcp|5353/udp|8324/tcp|32410:32414/udp

[plexmediaserver-dlna]
title=Plex Media Server (DLNA)
description=The Plex Media Server (additional DLNA capability only)
ports=1900/udp|32469/tcp

[plexmediaserver-all]
title=Plex Media Server (Standard + DLNA)
description=The Plex Media Server (with additional DLNA capability)
ports=32400/tcp|3005/tcp|5353/udp|8324/tcp|32410:32414/udp|1900/udp|32469/tcp

保存文件并更新配置文件列表：

sudo ufw app update plexmediaserver

通过以下命令来应用防火墙规则：

sudo ufw allow 22
sudo ufw allow plexmediaserver-all

最后检查新防火墙规则是否成功应用于：

sudo ufw status verbose
Status: active
Logging: on (low)
Default: deny (incoming), allow (outgoing), disabled (routed)
New profiles: skip

To                         Action      From
--                         ------      ----
22/tcp                     ALLOW IN    Anywhere
32400/tcp (plexmediaserver-all) ALLOW IN    Anywhere
3005/tcp (plexmediaserver-all) ALLOW IN    Anywhere
5353/udp (plexmediaserver-all) ALLOW IN    Anywhere
8324/tcp (plexmediaserver-all) ALLOW IN    Anywhere

1900/udp (plexmediaserver-all) ALLOW IN    Anywhere
32469/tcp (plexmediaserver-all) ALLOW IN    Anywhere

配置 Plex 媒体服务器

在启动 Plex 设置向导之前，我们先创建存储 Plex 媒体文件的目录：

sudo mkdir -p /opt/plexmedia/{movies,series}

Plex 媒体服务器作为用户 plex 运行，该用户必须具有对媒体文件和目录的读取和执行权限。请运行以下命令设置正确的所有权。

sudo chown -R plex: /opt/plexmedia

您可以选择任何目录来存储媒体文件，只需确保设置正确的权限即可。

现在我们可以继续进行服务器配置了。打开浏览器，输入 http://YOUR_SERVER_IP:32400/web ，您将看到以下屏幕：

要使用 Plex 媒体服务器，您需要创建一个帐户。

选择 Google 账户， Facebook账户 或电子邮件按钮创建免费的 Plex 帐户。如果您想要访问高级功能，可以购买 Plex Pass 计划。

注册后，您将被重定向到页面，其中包含有关 Plex 如何工作的信息，如下所示：

单击 Got it 按钮。

在下一个屏幕上输入您的 Plex 服务器名称，选中该复选框 Allow me to access my media outside my home ，然后单击 Next 。

下一步是添加媒体库。单击 Add Library 按钮。

弹出窗口显示时，选择 Movies 作为库类型并单击 Next 。

在下一步中，单击 Browse for media folder 并添加包含 Movies 媒体文件的目录的路径（我们的示例中使用 /opt/plexmedia/movies） 。

单击 Add 按钮，然后单击 Add Library 。

您可以根据需要添加任意数量的库。

单击 Next ，继续点击 Done， 您将被重定向到 Plex Web 仪表板。

现在您已经完成了设置向导，后续您可以开始探索 Plex 选项及其可以执行的所有操作。

更新 Plex 媒体服务器

发布新版本后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Plex 媒体服务器软件包：

sudo apt update
sudo apt upgrade

结论

您已经学习了如何在 Debian 9 计算机上安装 Plex 媒体服务器以及如何创建媒体库。

您现在可以在 Android ， iPhone ，智能电视， Xbox ， Roku 或任何其他受支持的设备上安装 Plex 应用程序。您可以在 Plex 下载页面上找到支持的应用程序和设备列表，也可以从设备的应用程序商店中安装该应用程序。

您还应该查看官方的 Plex 快速入门指南和 Plex 文档页面。
