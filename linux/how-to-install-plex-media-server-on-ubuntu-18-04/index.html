如何在 Ubuntu 18.04 上安装 Plex Media Server

March 26, 2019

在本教程中，我们将引导您在 Ubuntu 18.04 上安装 Plex 媒体服务器 。

Plex 是一种流媒体服务器，可让您随时随地组织视频，音乐和照片集，并将其流式传输到您的所有设备。

虽然本教程是为 Ubuntu 18.04 Bionic Beaver 编写的，但 Debian 和 Ubuntu 16.04 Xenial Xerus 可以使用相同的步骤。

先决条件

您需要以具有 sudo 访问权限的用户身份登录才能在 Ubuntu 系统上安装软件包。

安装 Plex Media Server

在 Ubuntu 18.04 上安装和管理 Plex Media Server 的最简单方法是使用 Plex 官方软件仓库。它不需要技术知识，安装和配置媒体服务器不应超过 20 分钟。

请按照以下步骤在 Ubuntu 系统上安装 Plex Media Server ：

  1. 首先使用以下 curl 命令导入软件仓库的 GPG 密钥：

    curl https://downloads.plex.tv/plex-keys/PlexSign.key | sudo apt-key add -

    运行以下命令将 Plex APT 软件仓库添加到系统的软件软件仓库列表：

    echo deb https://downloads.plex.tv/repo/deb public main | sudo tee /etc/apt/sources.list.d/plexmediaserver.list

  2. 启用 Plex 软件仓库后，更新 apt 程序包列表并安装最新版本的 Plex Media Server ：

    sudo apt install apt-transport-https
sudo apt update
sudo apt install plexmediaserver
    1. 要验证 Plex 服务是否正在运行，请键入：
    sudo systemctl status plexmediaserver

    输出应该如下所示：

    ● plexmediaserver.service - Plex Media Server for Linux

    调整防火墙规则

    现在您已在服务器上安装并运行 Plex ，您需要确保将防火墙配置为允许 Plex Media Server 特定端口上的流量。

    如果您的系统上没有运行防火墙，则可以跳过本节。

    以下步骤假定您使用 UFW 管理防火墙：

    打开您喜欢的文本编辑器并创建 UFW 应用程序配置文件 /etc/ufw/applications.d/plexmediaserver ：

    [plexmediaserver]
title=Plex Media Server (Standard)
description=The Plex Media Server
ports=32400/tcp|3005/tcp|5353/udp|8324/tcp|32410:32414/udp

[plexmediaserver-dlna]
title=Plex Media Server (DLNA)
description=The Plex Media Server (additional DLNA capability only)
ports=1900/udp|32469/tcp

[plexmediaserver-all]
title=Plex Media Server (Standard + DLNA)
description=The Plex Media Server (with additional DLNA capability)
ports=32400/tcp|3005/tcp|5353/udp|8324/tcp|32410:32414/udp|1900/udp|32469/tcp

保存文件并更新配置文件列表：

sudo ufw app update plexmediaserver

应用新防火墙规则：

sudo ufw allow plexmediaserver-all

最后检查新防火墙规则是否成功应用于：

sudo ufw status verbose

配置 Plex 媒体服务器

在启动 Plex 设置向导之前，让我们创建存储 Plex 媒体文件的目录：

sudo mkdir -p /opt/plexmedia/{movies,series}

Plex Media Server 使用用户 plex 运行，该用户必须具有对媒体文件和目录的读取和执行权限。要设置正确的所有权，请运行以下命令。

sudo chown -R plex: /opt/plexmedia

您可以选择任何位置来存储媒体文件，只需确保设置正确的权限即可。

现在我们可以继续进行服务器配置了。打开浏览器，输入 http://YOUR_SERVER_IP:32400/web ，您将看到以下屏幕：

要使用 Plex Media Server ，您需要创建一个帐户。

按 Google ， Facebook 或电子邮件按钮创建免费的 Plex 帐户。如果您想要访问高级功能，可以购买 Plex Pass 计划。

注册后，您将被重定向到页面，其中包含有关 Plex 如何工作的信息，如下所示：

单击 Got it 按钮。

在下一个屏幕上输入您的 Plex 服务器名称， Allow me to access my media outside my home 选中该复选框，然后单击 Next 。

下一步是添加媒体库。单击 Add Library 按钮。

弹出窗口显示时，选择电影作为库类型并单击 Next 。

在下一步中，单击 Browse for media folder 并添加包含 Movies 媒体文件目录的路径,在我们的示例中用的是 /opt/plexmedia/movies 。

单击 Add 按钮，然后单击 Add Library 。

您可以根据需要添加任意数量的库。

单击 Next ，然后 Done 您将被重定向到 Plex Web 仪表板。

现在您已经完成了设置向导，您可以开始探索 Plex 选项及其可以执行的所有操作。

更新 Plex 媒体服务器

当有新版本发布后，您可以通过桌面标准软件更新工具或在终端中运行以下命令来更新 Plex Media Server 软件包：

sudo apt update
sudo apt upgrade

结论

您已经学习了如何在 Ubuntu 18.04 计算机上安装 Plex Media Server 以及如何创建媒体库。

您现在可以在 Android ， iPhone ，智能电视， Xbox ， Roku 或任何其他受支持的设备上安装 Plex 应用程序。您可以在 Plex 下载页面上找到支持的应用程序和设备列表，也可以从设备的应用程序商店中安装该应用程序。

您还应该查看官方的 Plex 快速入门指南和 Plex 文档页面。
