如何在 Ubuntu 18.04 上安装 OpenCV

April 3, 2019

本教程介绍如何在 Ubuntu 18.04 上安装 OpenCV 。

OpenCV （开源计算机视觉库）是一个开源计算机视觉库，具有 C++ ， Python 和 Java 的绑定。它用于大量的应用，包括医学图像分析，拼接街景图像，监控视频，检测和识别面部，跟踪移动物体，提取 3D 模型等等。

OpenCV 能充分发挥多核处理器的效能，并具有 GPU 加速功能，可实现实时操作。

我们将向您展示在 Ubuntu 上安装 OpenCV 的两种不同方法。您可以选择最适合您的方法进行安装。

从 Ubuntu 软件仓库安装 OpenCV

OpenCV 包可从 Ubuntu 18.04 软件仓库库获得。在撰写本文时，软件仓库中的版本是 3.2 ，可能并不总是最新版本。

要从 Ubuntu 18.04 软件仓库安装 OpenCV ，请按照下列步骤操作：

  1. 安装 OpenCV

    刷新包索引并通过键入以下命令安装 OpenCV 包：

    sudo apt update
sudo apt install python3-opencv

    上面的命令将安装运行 OpenCV 所需的所有包。

    1. 验证 OpenCV 安装

    要验证安装，我们将导入 cv2 模块并打印 OpenCV 版本：

    python3 -c "\
import cv2
print(cv2.__version__)"
    3.2.0

    Ubuntu 18.04 LTS 中的默认 Python 版本是 3.6 版。如果要使用 python 2 绑定安装 OpenCV ，请安装该 python-opencv 软件包。

    从源代码安装 OpenCV

    从源代码构建 OpenCV 库是安装 OpenCV 的推荐方法。它将针对您的特定系统进行优化，您可以完全控制构建选项。

    要从源代码安装最新的 OpenCV 版本，请按照下列步骤操作：

    1. 安装必要的包

    以下命令将安装所有必需和可选的依赖项：

    sudo apt install build-essential cmake git pkg-config libgtk-3-dev
sudo apt install libavcodec-dev libavformat-dev libswscale-dev libv4l-dev libxvidcore-dev libx264-dev
sudo apt install libjpeg-dev libpng-dev libtiff-dev gfortran openexr libatlas-base-dev
sudo apt install python3-dev python3-numpy libtbb2 libtbb-dev libdc1394-22-dev

  2. 克隆 OpenCV 源代码

    创建一个目录，该目录将保存软件仓库并使用以下命令克隆 OpenCV 和 OpenCV contrib 软件仓库：

    mkdir ~/opencv_build && cd ~/opencv_build
git clone https://github.com/opencv/opencv.git
git clone https://github.com/opencv/opencv_contrib.git

    在撰写本文时， github 软件仓库中的默认版本是 4.0.0 版。如果要安装旧版本的 OpenCV ，导航进入 opencv 和 opencv_contrib 目录，并运行 git checkout <opencv-version>

    1. 使用 CMake 配置 OpenCV

    下载完成后，创建一个临时构建目录，然后切换到该目录：

    cd ~/opencv_build/opencv
mkdir build && cd build

    使用 CMake 设置 OpenCV 构建：

    cmake -D CMAKE_BUILD_TYPE=RELEASE \
    -D CMAKE_INSTALL_PREFIX=/usr/local \
    -D INSTALL_C_EXAMPLES=ON \
    -D INSTALL_PYTHON_EXAMPLES=ON \
    -D OPENCV_GENERATE_PKGCONFIG=ON \
    -D OPENCV_EXTRA_MODULES_PATH=~/opencv_build/opencv_contrib/modules \
    -D BUILD_EXAMPLES=ON ..

    1. 编译 OpenCV

    请根据您的处理器修改 -j 标志。如果您不知道处理器的核心数量，可以通过键入 nproc 来找到它。我的系统有 8 个核心，所以我使用的是 -j8 旗帜。

    开始编译过程：

    make -j8

    编译可能需要几分钟或更长时间，具体取决于您的系统配置。完成后，您将看到如下内容：

  3. 安装 OpenCV

    编译过程完成后，安装 OpenCV ：

    sudo make install

    1. 验证 OpenCV 安装

    要检查 OpenCV 是否已成功安装，请键入以下命令，您应该看到 OpenCV 版本：

    pkg-config --modversion opencv4
    4.0.1
    python3 -c "\
import cv2
print(cv2.__version__)"
    4.0.1-dev

    结论

    我们向您展示了在 Ubuntu 18.04 服务器上安装 OpenCV 的两种不同方法。您可以根据您的要求和偏好选择适合于您的方法。虽然从 Ubuntu 软件仓库安装打包版本更容易，而从源代码构建 OpenCV 可以提供更大的灵活性，它应该是您安装 OpenCV 时的第一个选择。

ubuntu python opencv
