如何在Debian 9上安装和使用FFmpeg

May 9, 2019

FFmpeg是一个免费的开源命令行工具，用于转码多媒体文件。它包含一组共享的音频和视频库，如libavcodec，libavformat和libavutil。使用FFmpeg，您可以在各种视频和音频格式之间进行转换，设置采样率并调整视频大小。

本教程介绍如何在Debian 9上安装FFmpeg。

先决条件

您需要以具有sudo权限的用户身份登录才能在Debian系统上安装软件包。

在Debian上安装FFmpeg

官方Debian存储库包含可以与apt软件包管理器一起安装的FFmpeg软件包。在撰写本文时，Debian 9存储库中提供的当前版本的FFmpeg是3.2.12。

以下步骤描述了如何在Debian 9上安装FFmpeg：

  1. 首先更新包列表：

    sudo apt update
    1. 运行以下命令安装FFmpeg软件包：
    sudo apt install ffmpeg

  2. 通过运行ffmpeg -version将打印FFmpeg版本的命令来验证安装：

    ffmpeg -version

    输出应如下所示：

    ffmpeg version 3.2.12-1~deb9u1 Copyright (c) 2000-2018 the FFmpeg developers
built with gcc 6.3.0 (Debian 6.3.0-18+deb9u1) 20170516

    要打印所有可用的FFmpeg编码器和解码器，您可以使用：

    ffmpeg -encodersffmpeg -decoders

    而已。FFmpeg现已安装在您的系统上，您可以开始使用它。

    Debian存储库中包含的版本总是落后于最新版本的FFmpeg。如果要安装最新版本的FFmpeg，则需要从源代码构建FFmpeg工具。

    FFmpeg示例

    在本节中，我们将向您展示如何使用该ffmpeg工具的一些示例。

    基本转换

    使用ffmpeg您转换音频和视频文件时，无需指定输入和输出格式。自动检测输入文件格式，并从指定的文件扩展名中猜出输出格式。

    • 将视频文件从mov转换为mp4：
    ffmpeg -i input.mov output.mp4

  • 将音频文件从mp3转换为flac：

    ffmpeg -i input.mp3 output.flac

    指定编解码器

    要指定编解码器，请使用该-c选项。编解码器可以是任何支持的解码器/编码器的名称或copy仅复制输入流的特殊值。

    • 使用libvpx视频编解码器和libvorbis音频编解码器将视频文件从mp4转换为webm ：
    ffmpeg -i input.mp4 -c:v libvpx -c:a libvorbis output.webm

  • 将音频文件从mp3转换为使用libopus编解码器编码的ogg 。

    ffmpeg -i input.mp3 -c:a libopus output.ogg

    结论

    您已经学习了如何在Debian 9上安装FFmpeg。您现在可以访问官方的FFmpeg文档页面，了解如何使用FFmpeg进行转换以及您的视频和音频文件。
