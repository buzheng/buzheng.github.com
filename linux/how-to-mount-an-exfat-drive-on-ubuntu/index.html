如何在 Ubuntu Linux 上安装 exFAT 驱动器

2019年4月24日 • 约 600 字 , 预计阅读时间 4 分钟

exFAT（扩展文件分配表），有些人也将 exFAT 称为 FAT64，是一种专有的 Microsoft 文件系统，针对闪存设备（如 SD 卡和 USB 闪存驱动器）进行了优化。它旨在取代旧的 32 位 FAT32 文件系统，该系统无法存储大于 4 GB 的文件。

所有最新版本的 Windows 和 macOS 操作系统都支持 exFAT 文件系统。与大多数其他主要 Linux 发行版一样， Ubuntu 默认情况下不支持专有的 exFAT 文件系统。

如果您使用 Ubuntu 作为您的操作系统，可能在某些时候您在尝试安装 exFAT 格式的 USB 驱动器时会遇到问题。

当您插入 USB 驱动器时，系统将显示如下消息：

Error mounting /dev/sdb1 at /media/linuxize/USB: unknown file system ‘exfat’

在本教程中，我们将解释如何在 Ubuntu 18.04 桌面上启用 exFAT 支持。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。

如何在 Ubuntu Linux 上安装 exFAT 驱动器

为了能够在 Ubuntu 上安装 exFAT 文件系统，您需要安装免费的 FUSE exFAT 模块和工具，它为类 Unix 系统提供全功能的 exFAT 文件系统实现。

在安装软件包之前，请确保在系统上启用了 Universe 存储库。使用 Ctrl+Alt+T 键盘快捷键或单击终端图标打开终端并键入：

sudo add-apt-repository universe

启用存储库后，更新包索引并使用以下命令安装 exfat-fuse 和 exfat-utils 包：

sudo apt update
sudo apt install exfat-fuse exfat-utils

而已！您现在可以打开文件管理器并单击 USB 磁盘进行安装。

结论

您已经学习了如何在 Ubuntu 18.04 计算机上启用对 exFAT 文件系统的支持。

插入时 USB 驱动器将自动安装，但如果自动安装失败，则需要手动安装驱动器。
