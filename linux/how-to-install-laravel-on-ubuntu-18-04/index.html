如何在Ubuntu 18.04上安装Laravel

2019年5月9日 • 约 1100 字 , 预计阅读时间 5 分钟

Laravel是一个开源的PHP Web应用程序框架，具有富有表现力的优雅语法。Laravel允许您轻松构建可扩展且灵活的Web应用程序，RESTful API和电子商务解决方案。

借助内置功能，例如路由，身份验证，会话，缓存和单元测试，Laravel是许多PHP开发人员的首选框架。

在本教程中，我们将向您展示如何在Ubuntu 18.04系统上安装Laravel。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Linux Mint，Kubuntu和Elementary OS。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

将系统软件包更新到最新版本：

sudo apt update && sudo apt upgrade

安装PHP

PHP 7.2是Ubuntu 18.04中的默认PHP版本，完全支持并推荐用于Laravel 5.7。

运行以下命令以安装PHP和所有必需的PHP模块：

sudo apt install php7.2-common php7.2-cli php7.2-gd php7.2-mysql php7.2-curl php7.2-intl php7.2-mbstring php7.2-bcmath php7.2-imap php7.2-xml php7.2-zip

安装Composer

Composer是PHP的依赖管理器，我们将使用它来下载Laravel核心并安装所有必需的Laravel组件。

要全局安装composer，请使用curl下载Composer安装程序并将文件移动到以下/usr/local/bin目录：

curl -sS https://getcomposer.org/installer | sudo php -- --install-dir=/usr/local/bin --filename=composer

通过打印composer版本验证安装：

composer --version

输出应该如下所示：

Composer version 1.8.0 2018-12-03 10:31:16

安装Laravel

在撰写本文时，Laravel的最新稳定版本是版本5.7。

运行Composer create-project命令以在my_app目录中安装Laravel ：

composer create-project --prefer-dist laravel/laravel my_app

上面的命令将获取所有必需的php包。该过程可能需要几分钟，如果成功，输出结束应如下所示：

Package manifest generated successfully.
> @php artisan key:generate --ansi
Application key set successfully.

此时，您已在Ubuntu系统上安装了Laravel。

通过Composer安装时，Laravel将自动创建一个名为的文件.env。此文件包括自定义配置变量，包括数据库凭据。您可以在此处阅读有关如何配置Laravel的更多信息。

您可以通过导航到Laravel项目目录并执行artisan serve命令来启动开发服务器：

cd ~/my_appphp artisan serve

输出看起来像这样：

Laravel development server started: <http://127.0.0.1:8000>

Laravel可以使用SQLite，PostgreSQL，MongoDB或MySQL / MariaDB数据库来存储其所有数据。

如果要使用Laravel Mix编译资产，则需要安装Node.js和Yarn。

结论

恭喜，您已在Ubuntu 18.04计算机上成功安装了Laravel 5.7。您现在可以开始开发应用程序了。
