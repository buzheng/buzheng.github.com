如何在 Ubuntu 18.04 上安装 CouchDB

March 9, 2019

CouchDB 是 Apache Software Foundation 维护的免费开源 NoSQL 数据库。

CouchDB 服务器将其数据存储在命名数据库中，以 JSON 结构存储文档。每个文档可以包含许多字段和附件。字段可以包括文本，数字，列表，布尔值等。CouchDB 提供了一个 RESTful HTTP API ，通过 API 您可以完成读取，创建，编辑和删除数据库文档。

在本教程中，我们将介绍在 Ubuntu 18.04 上从 CouchDB 官方仓库安装最新版本的 CouchDB 的过程。

先决条件

在继续本教程之前，请确保以具有 sudo 权限的用户身份登录。

启用 CouchDB 仓库

首先，使用以下命令将 CouchDB GPG 密钥添加到系统中：

curl -L https://couchdb.apache.org/repo/bintray-pubkey.asc | sudo apt-key add -

导入密钥后，添加 CouchDB 仓库：

echo "deb https://apache.bintray.com/couchdb-deb bionic main" | sudo tee -a /etc/apt/sources.list

在 Ubuntu 上安装 CouchDB

更新包列表并安装 CouchDB ：

sudo apt update
sudo apt install couchdb

在安装过程中，系统将询问您是否要以独立模式或群集模式安装 CouchDB 。我们将在单服务器独立模式下安装 CouchDB 。

接下来，您可以选择设置将 CouchDB 绑定到的网络接口的 IP 地址。对于单服务器设置，请保留默认值 127.0.0.0 。如果要配置群集，请输入 0.0.0.0 ，这样 CouchDB 会绑定到所有的网络接口的 IP 地址。

在下一个提示中设置管理员密码。强烈建议设置密码，使 CouchDB 退出不安全的管理模式（CouchDB 称这种方式为 “Admin Party”，任何人都有有管理员权限，可以为所欲为的做任何事情）。如果将此字段留空，则不会创建管理员用户。

确认密码后， CouchDB 继续安装。

验证 CouchDB 安装

要验证安装是否已成功完成，请运行以下 curl 命令，该命令将以 JSON 格式打印 CouchDB 数据库信息：

curl http://127.0.0.1:5984/

输出将如下所示：

{  
   "couchdb":"Welcome",
   "version":"2.3.0",
   "git_sha":"07ea0c7",
   "uuid":"1d2074b5eb428c30240e0c7384036acf",
   "features":[  
      "pluggable-storage-engines",
      "scheduler"
   ],
   "vendor":{  
      "name":"The Apache Software Foundation"
   }
}

为了可读性，输出已格式化。

如果您更喜欢 GUI ，则可以通过 url: http://DOMAIN_OR_IP_ADDRESS:5984/_utils/ 访问 CouchDB 基于 Web 的界面 Fauxton。

结论

您已经学习了如何安装 CouchDB Ubuntu 18.04 ， 接下来您可以访问 Apache CouchDB 文档并查找有关此主题的更多信息。
