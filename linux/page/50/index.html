如何在 Ubuntu 18.04 上安装 TensorFlow

TensorFlow 是由 Google 构建的机器学习的免费开源平台。它被许多组织使用，包括 Twitter ， PayPal ，英特尔，联想和空中客车。 TensorFlow 可以在系统级别安装，还可以使用 pip 在 Python 虚拟环境中安装或者在 Docker 容器中安装。如果是出于学习目的，最好在 Python 虚拟环境中安装 TensorFlow 。这样，您可以在一台计算机上拥有多个不同的独立 Python 环境，并在每个项目的基础上安装特定版本的模块，而不必担心它会影响您的其他项目。 本教程介绍如何在 Ubuntu 18.04 上安装 TensorFlow 。

如何在 Ubuntu 18.04 上安装 VirtualBox

VirtualBox 是一种开源的跨平台虚拟化软件，允许您同时运行多个客户操作系统（虚拟机）。 在本教程中，我们将向您展示如何在 Ubuntu 18.04 计算机上安装 VirtualBox 的两种不同方法。第一种方法描述了从 Oracle 软件仓库安装 VirtualBox 6.0 所需的步骤，而第二种方法将向您展示如何从官方 Ubuntu 软件仓库安装最新版本的 VirtualBox 。 Ubuntu multiverse 软件仓库中提供的 VirtualBox 包可能并不总是最新版本。如果您希望始终拥有最新版本的 VirtualBox ，那么最好的方法是从 Oracle 软件仓库安装 VirtualBox 。 相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Linux Mint 和 Elementary OS 。

如何在 Debian 9 上安装 Node.js 和 npm

Node.js 是一个开源的跨平台 JavaScript 运行时环境，构建于 Chrome 的 JavaScript 引擎之上，提供了在服务器端执行 JavaScript 代码的能力。它主要用于构建服务器端应用程序，但它作为全栈和前端解决方案也非常受欢迎。 npm 是 Node.js 的默认包管理器，也是世界上最大的软件注册仓库。 在本教程中，我们将介绍如何在 Debian 9 上安装 Node.js 和 npm 。我们将向您展示在 Debian 上安装 Node.js 的两种不同方法。您只需要选择最适合您的安装方法。

如何查看你的 Ubuntu 版本

当您第一次登录 Ubuntu 系统时，在进行任何工作之前，最好查看一下该机器上运行的 Ubuntu 版本。 新的 Ubuntu 版本每 6 个月发布一次，而 LTS (长期支持) 版本每两年发布一次。桌面和服务器上支持 LTS 版本 5 年，其他标准版本支持 9 个月。 在本教程中，我们将向您展示如何查看您的 Ubuntu 版本。有几种方法可以查看您正在运行的 Ubuntu 版本，包括可以从命令行或图形界面中进行查看。

如何在 CentOS 7 上安装 Apache Cassandra

Apache Cassandra 是一个开源的 NoSQL 数据库，没有单点故障，在不影响性能的情况下提供线性可扩展性和高可用性。在 Cassandra 中，记录的结构与关系数据库中的表，行和列类似。 Apache Cassandra 被许多组织使用，包括 Apple， NetFlix， eBay 和 Easou 。 本教程介绍如何在 CentOS 7 上安装 Apache Cassandra。
