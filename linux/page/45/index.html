如何在 Debian 9 上安装 Java

在本教程中，我们将介绍如何在 Debian 9 上安装 Java. Java 是用于构建不同类型的应用程序和系统的最流行的编程语言之一。使用 Java 开发的应用程序更易于扩展，更加灵活且更容易维护。 有两个不同的 Java 包， Java Runtime Environment (JRE)和 Java Development Kit (JDK)。如果您只想运行 Java 程序，那么您需要 JRE ，如果您是 Java 开发人员，那么您将需要 JDK ，其中包括了 JRE 和开发/调试工具和库。 Java 有两种不同的实现: OpenJDK 和 Oracle Java ，它们之间几乎没有区别。不同的只是 Oracle Java 有一些额外的商业功能。 如果您不确定要使用哪个 Java 实现和版本，一般建议使用 Debian 9 上默认提供的 OpenJDK 版本。

创建 Linux Swap 交换区文件

Swap 交换区是磁盘上的空间，当物理 RAM 内存已满时使用。当 Linux 系统用完 RAM 时，非活动页面会从 RAM 移动到 Swap 交换区。 Swap 交换区可以采用专用交换分区或交换文件的形式。在大多数情况下，当在虚拟机上运行 Linux 时，交换分区不存在，因此我们唯一的选择是创建交换文件。 本教程在使用 Ubuntu 18.04 和 CentOS 7 的 Linux 系统上进行了测试，但它适用于任何其他 Linux 发行版。
