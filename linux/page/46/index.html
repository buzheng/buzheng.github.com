创建 Linux Swap 交换区文件

Swap 交换区是磁盘上的空间，当物理 RAM 内存已满时使用。当 Linux 系统用完 RAM 时，非活动页面会从 RAM 移动到 Swap 交换区。 Swap 交换区可以采用专用交换分区或交换文件的形式。在大多数情况下，当在虚拟机上运行 Linux 时，交换分区不存在，因此我们唯一的选择是创建交换文件。 本教程在使用 Ubuntu 18.04 和 CentOS 7 的 Linux 系统上进行了测试，但它适用于任何其他 Linux 发行版。
