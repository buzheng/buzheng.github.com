如何在Ubuntu 18.04上安装PyCharm

PyCharm是一个用于Python和Web开发的全功能IDE 。它具有内置调试支持，Docker和Vagrant支持，嵌入式Git控件，Anaconda支持，语法突出显示，代码完成，ssh终端，代码重构和丰富的导航功能。

PyCharm可通过插件扩展，并支持Python Web开发框架，如Django和Flask。它还可以用于其他编程语言的开发，例如JavaScript，SQL，HTML，CSS，AngularJS，Node.js等。

PyCharm有两个版本，社区和专业版。您可以在此处找到有关这两个版本之间差异的更多信息。

在本教程中，我们将介绍如何在Ubuntu 18.04上安装PyCharm Community Edition。

如何在Ubuntu 18.04上安装Python 3.7

Python是世界上最流行的编程语言之一。凭借其简单易学的语法，Python是初学者和经验丰富的开发人员的绝佳选择。Python是一种多功能的编程语言，您可以使用它来完成您想要的任何操作，编写小脚本，构建游戏，开发网站，创建机器学习算法，分析数据等等。

Python 3.7是Python语言的最新主要版本，它包含许多新功能，例如类型注释的推迟评估，对数据类和上下文变量的支持，对模块属性的访问的自定义等等。

本教程描述了在Ubuntu 18.04上安装Python 3.7的两种方法：使用deadsnakes PPA中的标准apt工具，并从源代码构建。[](https://launchpad.net/~deadsnakes/+archive/ubuntu/ppa)

相同的步骤适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

如何在Ubuntu 18.04上安装VMware Workstation Player

VMware是一种成熟稳定的虚拟化解决方案，允许您在一台计算机上运行多个独立的操作系统。

借助VMware Workstation Player，您可以创建和运行自己的虚拟机，并评估从VMware解决方案交换中心提供的许多软件供应商中作为虚拟设备分发的软件。

VMware Workstation Player不是开源的，仅供个人非商业用途免费使用。如果您正在寻找开源虚拟化平台，那么您应该尝试使用Oracle的VirtualBox。

本教程介绍如何在Ubuntu 18.04上安装VMware Workstation Player。相同的说明适用于Ubuntu 16.04和任何其他基于Debian的发行版，包括Debian，Linux Mint和Elementary OS。
