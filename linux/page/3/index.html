Apache Cassandra 是一个免费的开源 NoSQL 数据库，没有单点故障。它可以在不影响性能的前提下提供线性可扩展性和高可用性。 Apache Cassandra 被许多拥有大型活跃数据集的公司使用，包括 Reddit ， NetFlix ， Instagram 和 Github 。

如何在 CentOS 7 上安装 Asterisk

Asterisk 是最受欢迎和广泛采用的开源 PBX 平台，为 IP PBX 系统，会议服务器和 VoIP 网关提供支持。它被世界各地的个人，小企业，大型企业和政府使用。

Asterisk 功能包括语音邮件，待机音乐，电话会议，呼叫排队，呼叫录音，交互式语音响应等等。

Linux cd 命令

在 Linux 的命令行上工作时， cd 命令是最基本和最常用的命令之一。 cd 命令代表“更改目录”，用于更改 Linux 和其他类 Unix 操作系统中的当前工作目录。当前工作目录是用户当前工作的目录。每次与命令提示符交互时，您都在目录中工作。

在本教程中，我们将向您展示如何使用 cd 命令导航计算机的目录树。

如何在 Ubuntu 18.04 上安装和配置 Gogs

Gogs 是一个用 Go 编写的自托管开源 git 服务器。它包括存储库文件编辑器，项目问题跟踪，并且内置维基。

Gogs 是一个轻量级应用程序，内存占用空间更小，可以安装在低功耗系统上。如果您正在寻找 Gitlab 的替代品，而且您不需要 Gitlab 提供的所有功能，那么您一定要尝试 Gogs 。

本教程介绍了在 Ubuntu 18.04 上安装和配置 Gogs 的步骤。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Ubuntu 的发行版。

如何在 Ubuntu 18.04 上安装 MongoDB

MongoDB 是一个免费的开源文档数据库。它属于一个名为 NoSQL 的数据库系列，它与传统的基于表的 SQL 数据库（如 MySQL 和 PostgreSQL ）不同。

在 MongoDB 中，数据存储在灵活的、类 JSON 的文档中，其中字段可能因文档而异。它不需要预定义的模式，数据结构可以随时间更改。

在本教程中，我们将介绍从官方 MongoDB 存储库在 Ubuntu 18.04 计算机上安装和配置最新版 MongoDB Community Edition 的过程。
