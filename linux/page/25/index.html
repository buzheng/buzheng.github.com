如何在Ubuntu 18.04上安装和配置Nagios

Nagios是最受欢迎的开源监控系统之一。Nagios会保留整个IT基础架构的清单，并确保您的网络，服务器，应用程序，服务和流程正常运行。

如果失败或次优，Nagios将通过各种方法发送通知警报。

本教程将指导您完成在Ubuntu 18.04上安装和配置最新版Nagios Core所需的步骤。

Ubuntu软件存储库中提供了旧版本的Nagios（版本3），可以轻松安装apt install nagios3。

如何在Ubuntu 18.04上安装和配置Zabbix

Zabbix是最受欢迎的开源监控软件工具之一。Zabbix从您的网络设备，系统和应用程序收集指标，并确保它们正常运行。如有任何问题，Zabbix将通过各种方法发送通知警报。

如何在Ubuntu 18.04上添加交换空间

交换是磁盘上的空间，当物理RAM内存已满时使用。当Linux系统用完RAM时，非活动页面会从RAM移动到交换空间。

交换空间可以采用专用交换分区或交换文件的形式。在大多数情况下，在虚拟机上运行Ubuntu时，不存在交换分区，因此唯一的选择是创建交换文件。

本教程介绍了在Ubuntu 18.04系统上添加交换文件所需的步骤。
