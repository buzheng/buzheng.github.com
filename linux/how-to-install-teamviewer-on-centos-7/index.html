如何在CentOS 7上安装TeamViewer

2019年5月9日 • 约 900 字 , 预计阅读时间 4 分钟

TeamViewer是一种跨平台解决方案，可用于计算机之间的远程控制，桌面共享和文件传输。TeamViewer是专有的计算机软件，它不包含在CentOS存储库中。

在本教程中，我们将介绍在CentOS 7上安装TeamViewer所需的步骤。

先决条件

您需要以具有sudo访问权限的用户身份登录才能在CentOS系统上安装软件包。

在CentOS上安装TeamViewer

在CentOS Linux上安装VirtualBox是一个相对简单的过程，只需几分钟：

  1. 首先将Oracle VirtualBox存储库的GPG密钥导入您的系统：

    sudo rpm --import  https://dl.tvcdn.de/download/linux/signature/TeamViewer2017.asc

  2. TeamViewer仅依赖于EPEL软件存储库中可用的软件包。如果系统上未启用存储库，请键入以下命令启用它：

    sudo yum install epel-release

  3. .rpm使用以下wget命令下载最新的TeamViewer 包：

    wget https://download.teamviewer.com/download/linux/teamviewer.x86_64.rpm

  4. 以具有sudo权限的用户身份发出以下命令来安装TeamViewer程序包：

    sudo yum localinstall ./teamviewer.x86_64.rpm

    出现提示时Is this ok [y/d/N]，键入y以继续安装。

此时，TeamViewer已安装并可以使用。您现在可以连接到您的朋友或客户机器并提供帮助。

启动TeamViewer

您可以通过键入teamviewer或单击TeamViewer图标（Applications -> Internet -> TeamViewer）从命令行启动TeamViewer应用程序：

首次启动TeamViewer时，将显示类似于以下内容的窗口。单击License Agreement按钮接受许可协议中的条款。

TeamViewer将打开，您将看到类似以下内容。

更新TeamViewer

在安装过程中，官方TeamViewer存储库将添加到您的系统中。您可以使用cat命令验证文件内容：

cat /etc/yum.repos.d/teamviewer.repo

[teamviewer]
name=TeamViewer - $basearch
baseurl=http://linux.teamviewer.com/yum/stable/main/binary-$basearch/
gpgkey=http://linux.teamviewer.com/pubkey/TeamViewer2017.asc
gpgcheck=1
enabled=1
type=rpm-md
failovermethod=priority

这可确保在通过桌面标准软件更新工具发布新版本时，TeamViewer安装将自动更新。

结论

在本教程中，您学习了如何在CentOS 7桌面计算机上安装TeamViewer。
