如何在 Debian 9 上安装 Gradle

2019年3月5日 • 约 800 字 , 预计阅读时间 4 分钟

Gradle 是一个通用的构建工具，主要用于 Java 项目，结合了 Ant 和 Maven 的最佳功能。与使用 XML 进行脚本编写的前辈不同，Gradle使用 Groovy，这是一种动态的，面向对象的Java平台编程语言，用于定义项目和构建脚本。

在本教程中，我们将解释如何在 Debian 9 上安装 Gradle。

先决条件

您需要以具有 sudo 访问权限的用户身份登录才能在 Debian 系统上安装软件包。

在Debian上安装Gradle

在以下部分中，我们将逐步说明如何在 Debian 9 上下载和安装最新 Gradle 版本。

1.安装OpenJDK

Gradle 需要安装 Java JDK 或 JRE 7 或更高版本。

我们将安装OpenJDK 8：

sudo apt update
sudo apt install openjdk-8-jdk

运行以下命令验证 Java 安装：

java -version

输出应该如下所示：

openjdk version "1.8.0_181"
OpenJDK Runtime Environment (build 1.8.0_181-8u181-b13-2~deb9u1-b13)
OpenJDK 64-Bit Server VM (build 25.181-b13, mixed mode)

如果您更喜欢 Oracle Java 而不是OpenJDK，请按照安装 Oracle Java。

2.下载Gradle

在撰写本文时，最新版本的 Gradle 是 5.2.1。在继续下一步之前，您应该到 Gradle 版本页面检查是否有更新的版本。

使用以下的 wget 命令下载 Gradle Binary-only zip 文件到 /tmp 目录中：

wget https://services.gradle.org/distributions/gradle-5.2.1-bin.zip -P /tmp

下载完成后，解压缩目录中的 zip 文件 /opt/gradle：

sudo unzip -d /opt/gradle /tmp/gradle-*.zip

通过列出 /opt/gradle/gradle-5.0 目录来验证是否已提取 Gradle 文件：

ls /opt/gradle/gradle-5.2.1
bin  getting-started.html  init.d  lib  LICENSE  media  NOTICE

3.设置环境变量

接下来，我们需要把 Gradle 的 bin 目录配置到 PATH 环境变量中 。为此，请打开文本编辑器并在 目录 /etc/profile.d/ 创建一个名为 gradle.sh 的新文件。

sudo nano /etc/profile.d/gradle.sh

粘贴以下配置：

/etc/profile.d/gradle.sh

export GRADLE_HOME=/opt/gradle/gradle-5.0
export PATH=${GRADLE_HOME}/bin:${PATH}

保存并关闭文件。键入以下命令给脚本文件添加可执行权限：

sudo chmod +x /etc/profile.d/gradle.sh

使用以下命令加载环境变量：

source /etc/profile.d/gradle.sh

4.验证Gradle安装

要验证 Gradle 是否已正确安装，请使用 gradle -v 将显示 Gradle 版本的命令：

gradle -v

您应该看到如下内容：

Welcome to Gradle 5.2.1!

Here are the highlights of this release:
 - Define sets of dependencies that work together with Java Platform plugin
 - New C++ plugins with dependency management built-in
 - New C++ project types for gradle init
 - Service injection into plugins and project extensions

For more details see https://docs.gradle.org/5.2.1/release-notes.html


------------------------------------------------------------
Gradle 5.2.1
------------------------------------------------------------

Build time:   2019-02-08 19:00:10 UTC
Revision:     f02764e074c32ee8851a4e1877dd1fea8ffb7183

Kotlin DSL:   1.1.3

JVM:          1.8.0_181 (Oracle Corporation 25.181-b13)

到此为止，您已在 Debian 9 上成功安装了 Gradle。

结论

您现在可以访问 Gradle官方文档页面了解如何开始使用 Gradle。
