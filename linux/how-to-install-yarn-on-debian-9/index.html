如何在Debian 9上安装Yarn

2019年5月9日 • 约 1100 字 , 预计阅读时间 5 分钟

Yarn是一个兼容npm的JavaScript包管理器。它的创建是为了解决npm的一系列问题，例如通过并行化操作和减少与网络连接相关的错误来加速软件包安装过程。

在本教程中，我们将指导您如何通过Yarn APT包存储库在Debian 9系统上安装Yarn。官方Yarn存储库始终如一，并提供最新版本。

先决条件

在继续本教程之前，请确保以具有sudo权限的用户身份登录。

在Debian上安装Yarn

按照以下步骤在Debian 9系统上安装Yarn：

  1. 第一步是启用Yarn存储库。首先使用以下curl命令导入存储库的GPG密钥：

    curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -

    使用以下命令启用Yarn APT存储库：

    echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list

  2. 启用存储库后，更新包索引并安装Yarn，其中包括：

    sudo apt updatesudo apt install yarn

    如果您的系统上尚未安装Node.js，则上面的命令将安装它。那些使用nvm的人可以跳过Node.js安装：

    sudo apt install --no-install-recommends yarn

  3. 通过打印Yarn版本号验证安装：

    yarn --version

    在撰写本文时，Yarn的最新版本是版本1.10.1。

    1.10.1

使用纱线

现在您已经在Ubuntu系统上安装了Yarn，下一步是探索一些最常见的Yarn命令。

创建一个新项目

要创建新的Yarn项目，请使用yarn init如下所示的命令：

yarn init my_yarn_project

init脚本会问你几个问题。您可以应答或按enter以使用默认值。

yarn init v1.10.1
question name (vagrant): Linuxize
question version (1.0.0): 0.0.1
question description: Testing Yarn
question entry point (index.js): 
question repository url: 
question author: Linuxize
question license (MIT): 
question private: 
success Saved package.json
Done in 20.18s.

完成后，脚本将创建一个基本package.json文件，其中包含您提供的信息。您可以稍后打开并编辑此文件。

添加依赖

如果要在项目中使用其他包，则需要将其添加到项目依赖项中。为此，请使用yarn add命令后跟包名：

yarn add [package_name]

上面的命令还将更新package.json和yarn.lock文件，以便在运行时处理此项目的任何人都yarn将获得相同的依赖项。

您还可以指定包版本或包标签：

yarn add [package_name]@[version_or_tag]

升级依赖

要升级依赖项，请使用以下方法之一：

yarn upgrade [package_name]yarn upgrade [package_name]@[version_or_tag]

上面的命令将根据package.json文件中指定的版本范围将项目依赖项更新为其最新版本。

删除依赖

要删除依赖项，请使用yarn remove命令后跟包名：

yarn remove [package_name]

此命令还将更新项目package.json和yarn.lock文件。

安装所有项目依赖项

要安装package.json文件运行中指定的所有项目依赖项，请执行以下操作：

yarn

要么

yarn install

结论

我们已经向您展示了如何在Debian 9机器上安装纱线。有关纱线的更多信息，请访问Yarn文档页面。
