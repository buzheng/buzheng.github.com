如何在 CentOS 7 上安装 R

2019年5月25日 • 约 1000 字 , 预计阅读时间 4 分钟

R 是一种快速增长的开源编程语言和自由环境，专门用于统计计算和图形表示。它由 R 统计计算基金会支持，主要由统计人员和数据挖掘者用于开发统计软件和执行数据分析。

本教程将教您如何在 CentOS 7 上安装 R 以及如何从官方的 Comprehensive R Archive Network（CRAN） 安装 R 软件包。

先决条件

在继续本教程之前，请确保满足以下先决条件：

  • 您的系统至少有 1G 的 RAM 。否则，您可以创建交换文件。
  • 您以具有 sudo 权限的用户身份登录。

在 Centos 上安装 R

在撰写本文时， R 的最新稳定版本是 3.5 版本。

要在 CentOS 7 上安装 R ，请按照下列步骤操作：

  1. EP 包存储库中提供了 R 包。如果您的计算机上没有安装 EPEL 存储库，可以输入以下命令：

    sudo yum install epel-release

    系统将提示您接受存储库 GPG 密钥，键入 y 并按下 Enter 。

  2. 添加存储库后，键入以下命令安装 R ：

    sudo yum install R

    R 是一个元包，包含所有必需的 R 组件。

  3. 通过键入以下命令来验证安装，该命令将打印 R 版本：

    R --version

    R version 3.5.0 (2018-04-23) -- "Joy in Playing"

Platform: x86_64-redhat-linux-gnu (64-bit)

从 CRAN 安装 R 软件包

R 如此受欢迎的主要原因之一是通过 Comprehensive R Archive Network（CRAN）提供的各种软件包。

出于演示目的，我们将安装一个名为 stringr 的包，它提供了常见字符串操作的快速和正确的实现。

以 root 身份启动时，程序包将全局安装并可供所有系统用户使用。如果您在没有 sudo 的情况下启动 R ，则会为您的用户设置个人库。

首先打开 R 控制台作为 root ：

sudo -i R

以下所有命令均在 R 控制台中执行。

通过运行以下命令安装程序包 stringr ：

install.packages("stringr")

系统将要求您选择 CRAN 镜像：

Installing package into ‘/usr/lib64/R/library’
(as ‘lib’ is unspecified)
--- Please select a CRAN mirror for use in this session ---
Secure CRAN mirrors

选择距离您所在位置最近的镜像。

安装需要一些时间，一旦完成，请键入以下内容加载库：

library(stringr)

接下来，创建一个名为 tutorial 的简单字符向量：

tutorial <- c("How", "to", "Install", "R", "on", "CentOS", "7")

运行以下函数，它将打印每个字符串的长度：

str_length(tutorial)

[1] 3 2 7 1 2 6 1

您可以在可用的 CRAN 包名称中找到更多 R 包，并通过 install.packages() 安装它们。

结论

您已成功安装 R 您的 CentOS 机器并学习如何安装 R 软件包。
