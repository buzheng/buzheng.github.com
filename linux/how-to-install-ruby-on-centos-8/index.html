如何在 CentOS 8 上安装 ruby

2020年1月29日 • 约 900 字 , 预计阅读时间 4 分钟

Ruby 是当今最受欢迎的语言之一。它具有优雅的语法，并且是 Ruby on Rails 框架背后的语言。

在本文中，我们将探讨在 CentOS 8 上安装 Ruby 的不同方法。

我们将展示如何从默认的 CentOS 8 存储库以及使用 Rbenv 和 RVM 脚本安装 Ruby 。选择最适合您的设置和环境的安装方法。

从 CentOS 储存库安装 Ruby

这是在 CentOS 上安装 Ruby 的最简单方法。在撰写本文时，标准 CentOS 储存库中的版本为 2.5.5 。

以超级用户或具有 sudo 特权的用户身份运行以下命令来安装 ruby 软件包：

sudo dnf install ruby

安装完成后，您可以通过打印 Ruby 版本来验证安装是否成功：

ruby --version

输出将如下所示：

ruby 2.5.5p157 (2019-03-15 revision 67260) [x86_64-linux]

您的 Ruby 版本可能与上面显示的版本不同。

而已！您已经在 CentOS 系统上成功安装了 Ruby ，并且可以开始使用它了。

使用 Rbenv 安装 Ruby

Rbenv 是轻量级的 Ruby 版本管理实用程序，可让您轻松切换 Ruby 版本。

我们将使用 ruby-build 扩展 Rbenv 核心功能的插件，并允许您从源代码安装任何 Ruby 版本。

首先安装 git 和从源代码构建 Ruby 所需的其他依赖项：

sudo dnf install git wget gcc bzip2 openssl-devel libffi-devel readline-devel zlib-devel gdbm-devel ncurses-devel

运行以下命令以安装 rbenv 和 ruby-build ：

wget -q https://github.com/rbenv/rbenv-installer/raw/master/bin/rbenv-installer -O- | bash

该脚本会将 rbenv 和 ruby-build 仓库从 GitHub  复制到 ~/.rbenv 目录。

在开始使用 rbenv 之前，您需要添加 $HOME/.rbenv/bin 到 PATH 。

如果您正在使用 Bash ，请键入：

echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.bashrc

如果您使用的是 Zsh ，请输入：

echo 'export PATH="$HOME/.rbenv/bin:$PATH"' >> ~/.zshrc

运行 rbenv -v 命令以确保安装成功：

rbenv -v

rbenv 1.1.2-17-g7795476

要获取可以安装的 rbenv 的所有 Ruby 版本的列表，请输入：

rbenv install -l

例如，如果要安装 Ruby 2.7.0 并将其设置为默认版本，则应输入：

rbenv install 2.7.0

通过打印版本号来验证 Ruby 是否已正确安装：

ruby -v

ruby 2.7.0p0 (2019-12-25 revision 647ee6f091) [x86_64-linux]

使用 RVM 安装 Ruby

RVM(Ruby 版本管理器)是一个命令行工具，可让您安装，管理和使用多个 Ruby 环境。

首先，安装从源代码构建 Ruby 所需的依赖项：

sudo dnf install curl gcc bzip2 openssl-devel libffi-devel readline-devel zlib-devel gdbm-devel ncurses-devel

运行以下命令以导入 GPG 密钥并安装 RVM ：

gpg2 --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB

要开始使用 RVM ，您需要运行以下 source 命令：

source ~/.rvm/scripts/rvm

要获取所有已知 Ruby 版本的列表，请输入：

rvm list known

例如，如果要安装 Ruby 2.6 并将其设置为默认版本，则可以发出以下命令：

rvm install 2.6

验证安装：

ruby -v

ruby 2.6.3p62 (2019-04-16 revision 67580) [x86_64-linux]

有关如何使用 RVM 管理 Ruby 安装的更多信息，请访问 RVM 文档页面。

结论

我们向您展示了三种在 CentOS 8 服务器上安装 Ruby 的方法。您选择的方法取决于您的要求和偏好。尽管从 CentOS 信息库安装打包版本更容易，但 Rbenv 和 RVM 方法为您提供了更大的灵活性，可以基于每个用户添加和删除不同的 Ruby 版本。
