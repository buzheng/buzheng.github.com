如何在CentOS 7上安装VMware Workstation Player

May 9, 2019

VMware是一种成熟稳定的虚拟化解决方案，允许您在一台计算机上运行多个独立的操作系统。您可以创建自己的虚拟机，并评估从VMware解决方案交换中心提供的许多软件供应商分发为虚拟设备的软件。

本教程将指导您完成在CentOS 7上安装VMware Workstation Player的步骤。

VMware Workstation Player不是开源的，仅供个人非商业用途免费使用。如果您正在寻找开源虚拟化平台，那么您应该尝试使用Oracle的VirtualBox。

先决条件

在继续学习本教程之前，请确保以具有sudo权限的用户身份登录。

要安装和使用VMware Workstation Player，您的系统应满足以下要求：

  • 1.3GHz或更快的64位处理器
  • 最低2 GB RAM / 4GB RAM或更多RAM

在CentOS上安装VMware Workstation Player

在撰写本文时，最新版本的VMware Workstation Player版本为15.0.2。

完成以下步骤以在CentOS 7上安装VMware Workstation Player：

1.下载VMware Workstation Player

通过运行以下wget命令下载最新版本的VMware Workstation Player ：

wget https://www.vmware.com/go/getplayer-linux

下载完成后，使用以下命令使安装文件可执行：

chmod +x getplayer-linux

2.安装VMware Workstation Player

安装非常简单，只需按照屏幕上的说明操作即可。键入以下命令启动安装向导：

sudo ./getplayer-linux

将显示类似于以下的屏幕。接受许可协议中的条款，然后单击Next按钮。

接下来，系统将询问您是否要在启动时检查产品更新。进行选择并单击Next按钮。

VMware的客户体验改善计划（“CEIP”）通过向VMware发送匿名系统数据和使用信息，帮助VMware改进其产品和服务。如果您不想参加该计划No，请选择并单击Next按钮。

在下一步中，系统将提示您输入许可证密钥。VMware Workstation Player可免费用于个人非商业用途。购买许可证密钥将使Pro功能可以在商业上使用。

如果您没有许可证密钥，请将该字段留空，然后单击Next按钮。

接下来，您将看到以下页面，通知您已准备好安装VMware Workstation Player。单击Install按钮。

安装可能需要几分钟。在安装过程中，您将看到有关进度的信息。完成后，将显示以下屏幕，通知您已成功安装VMware Workstation Player。

3.启动VMware Workstation Player

现在，您的CentOS系统上安装了VMware Workstation Player，您可以通过单击VMware Player图标（Applications -> System Tools -> VMware Workstation Player）启动它。

当您第一次启动VMware Player时，将出现如下窗口，询问您是否要将VMware Workstation Player免费用于个人非商业用途或输入许可证。

进行选择，单击OK按钮，VMware Player应用程序将启动。

结论

此时，您已在CentOS系统上成功安装了VMware Workstation Player。您现在可以浏览应用程序选项并创建第一个虚拟机。

要了解有关VMware Workstation Player的更多信息，请访问其官方文档页面。
