如何在 Ubuntu 18.04 上安装 Visual Studio 代码

Visual Studio Code 是 Microsoft 开发的一种开源跨平台代码编辑器。它具有内置的调试支持，内置了 Git 控件，语法高亮，代码完成，集成终端，代码重构和代码片段。 在 Ubuntu 机器上安装 Visual Studio Code 的最简单和推荐的方法是启用 VS Code 存储库并通过命令行安装 VS Code 包。 虽然本教程是为 Ubuntu 18.04 编写的，但 Ubuntu 16.04 可以使用相同的步骤。
