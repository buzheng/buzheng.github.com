如何在 Ubuntu 18.04 上安装 VLC Media Player

VLC 是最受欢迎的开源多媒体播放器之一。它是跨平台的，几乎可以播放所有多媒体文件以及 DVD ，音频 CD 和不同的流媒体协议。

本教程介绍如何在 Ubuntu 18.04 上安装 VLC 媒体播放器。相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。
