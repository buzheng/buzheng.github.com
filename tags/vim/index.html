如何在 Vim/Vi 中显示行号

Vim/Vi 是许多软件开发人员和 Linux 系统管理员选择的文本编辑器。 默认情况下， Vim 不显示行号，但可以轻松打开它们。 Vim 支持三种行编号模式，可帮助您浏览文件。除了标准的绝对行编号之外， Vim 还支持相对行和混合行编号模式。 在本指南中，我们将向您展示如何在 Vim/Vi 文本编辑器中显示或隐藏行号。 除了帮助导航代码外，行编号在其他情况下(例如结对编程，调试脚本，代码检查，引用特定行等)也很有用。
