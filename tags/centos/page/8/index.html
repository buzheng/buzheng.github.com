如何在 CentOS 7 上安装 CouchDB

Apache CouchDB 是 Apache Software Foundation 开发的免费开源 NoSQL 数据库。

CouchDB 服务器将其数据存储在指定名称的数据库中，数据库中包含具有 JSON 结构的文档。每个文档都包含许多字段和附件。字段可以包括文本，数字，列表，布尔值等。它包括一个 RESTful HTTP API ，允许您读取，创建，编辑和删除数据库文档。

在本教程中，我们将解释如何在 CentOS 7 上安装最新版本的 CouchDB 。

如何在 CentOS 7 上使用 FirewallD 设置防火墙

正确配置的防火墙是整个系统安全性最重要的方面之一。

FirewallD 是一个完整的防火墙解决方案，可管理系统的 iptables 规则，并提供 D-Bus 接口以对其进行操作。从 CentOS 7 开始， FirewallD 将 iptables 替换为默认的防火墙管理工具。

在本教程中，我们将向您展示如何在 CentOS 7 系统上使用 FirewallD 设置防火墙，并向您解释基本的 FirewallD 概念。

如何编辑 hosts 文件

hosts 文件用于将域名（主机名）映射到 IP 地址。它是所有操作系统（包括 Linux ， Windows 和 macOS ）使用的纯文本文件。 hosts 文件允许您设置给定域名的 IP 地址。这将仅影响进行更改的计算机，而不是影响全局域的解析方式。

如何在 CentOS 7 上安装 Git

本教程将引导您在 CentOS 7 安装 Git 并完成基本的配置。

Git 是最受欢迎的版本控制系统，已被数十万个项目使用。 Git 允许您跟踪代码更改，恢复到以前的阶段，在多个分支上同时工作并与您的开发人员协作。 Git 最初是由 Linux 内核的创建者 Linus Torvalds 开发的。

在撰写本文时，默认的 CentOS 7 存储库中可用的 Git 版本 1.8.3 已经过时了。

安装最新版 Git (v2.18) 的最简单方法是使用 Wandisco 存储库中的 yum 软件包管理工具进行安装。

另一种选择是从源代码编译 Git ，这将允许您安装最新的 Git 版本并自定义构建选项，但您将无法通过 yum 包管理器维护 Git 安装。
