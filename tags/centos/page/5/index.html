如何在 CentOS 8 上安装 Nginx

Nginx 发音为 “engine x” ，是一种开源的高性能 HTTP 和反向代理服务器，负责处理 Internet 上一些最大站点的负载。它可用作 HTTP 和非 HTTP 服务器的独立 Web 服务器，负载平衡器，内容缓存和反向代理。 与 Apache 相比， Nginx 可以处理大量并发连接，并且每个连接的内存占用量较小。 本教程说明了如何在 CentOS 8 上安装和管理 Nginx 。
