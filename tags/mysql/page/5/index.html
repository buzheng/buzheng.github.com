一个简单的 Mysql 备份 BAT 脚本

这是一个简单的 Windows 下备份 Mysql 的 BAT 脚本，脚本使用 mysqldump 命令来备份一个指定的 Mysql 数据库到一个文件，文件格式为 %dbname%-yyyyMMddHHmmss.sql，只保留最近60天的备份。如果想定时执行，在 Windows 中添加任务计划即可。
