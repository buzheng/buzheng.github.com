如何在 Debian 9 上安装 Apache Cassandra

Apache Cassandra 是一个免费的开源 NoSQL 数据库，没有单点故障。它可以在不影响性能的前提下提供线性可扩展性和高可用性。 Apache Cassandra 被许多拥有大型活跃数据集的公司使用，包括 Reddit ， NetFlix ， Instagram 和 Github 。
