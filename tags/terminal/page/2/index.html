Linux 中的 less 命令

Less 是一个命令行实用程序，用于显示文件或命令输出的内容，它一次只显示一个页面。它类似于 more ，但具有更高级的功能，允许您在文件中向前和向后导航。 该 less 命令主要用于打开大文件。less 不会读取整个文件，相比于 vim 或 nano 等文本编辑器，加载时间会更快。

Linux Nohup 命令

SIGHUP 是当控制终端关闭时发送到进程的信号。而 nohup 命令参数指定执行的另一个命令在执行后会忽略 SIGHUP (挂断) 信号。 通常，当您通过 SSH 运行命令时，如果连接断开，或者您注销， SSH 会话将终止，并且终端执行的所有命令都将停止。这是 nohup 命令派上用场的地方。它将忽略所有挂断信号，命令将继续运行。
