如何在 Ubuntu 18.04 上安装和配置 NFS 服务器

网络文件系统 (NFS) 是一种分布式文件系统协议，允许您通过网络共享远程目录。使用 NFS ，您可以在系统上挂载远程目录，并使用远程计算机上的文件，就像它们是本地文件一样。

默认情况下， NFS 协议未加密，与 Samba 不同，它不提供用户身份验证。对服务器的访问受客户端 IP 地址或主机名的限制。

在本教程中，我们将讨论如何在 Ubuntu 18.04 上设置 NFSv4 服务器。我们还将向您展示如何在客户端上安装 NFS 文件系统。

如何在 Ubuntu 18.04 上安装 Chromium 浏览器

Chromium 是一款快速，稳定，安全的浏览器，专为现代网络而打造。它是许多浏览器的基础，包括世界上最广泛使用的 Google Chrome 浏览器。

Chromium 和谷歌 Chrome 之间的主要区别在于 Chromium 是开源的。与 Chrome 不同， Chromium 不包含专有的网络编解码器，也不会向 Google 发送崩溃报告和使用情况跟踪。

如果您更喜欢开源，那么 Chromium 是一个很好的选择。

在本教程中，我们将解释如何在 Ubuntu 18.04 上安装 Chromium Web 浏览器。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Ubuntu 的发行版，包括 Kubuntu ， Linux Mint 和 Elementary OS 。

如何在 Ubuntu 18.04 上设置 SSH 密钥

Secure Shell(SSH) 是一种加密网络协议，用于客户端和服务器之间的安全连接，并支持各种身份验证机制。

两种最流行的机制是基于密码的身份验证和基于公钥的身份验证。使用 SSH 密钥比传统的密码验证更安全，更方便。
