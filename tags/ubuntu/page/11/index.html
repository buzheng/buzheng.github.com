如何在 Ubuntu 18.04 上安装 VirtualBox

VirtualBox 是一种开源的跨平台虚拟化软件，允许您同时运行多个客户操作系统（虚拟机）。

在本教程中，我们将向您展示如何在 Ubuntu 18.04 计算机上安装 VirtualBox 的两种不同方法。第一种方法描述了从 Oracle 软件仓库安装 VirtualBox 6.0 所需的步骤，而第二种方法将向您展示如何从官方 Ubuntu 软件仓库安装最新版本的 VirtualBox 。

Ubuntu multiverse 软件仓库中提供的 VirtualBox 包可能并不总是最新版本。如果您希望始终拥有最新版本的 VirtualBox ，那么最好的方法是从 Oracle 软件仓库安装 VirtualBox 。

相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Linux Mint 和 Elementary OS 。

如何查看你的 Ubuntu 版本

当您第一次登录 Ubuntu 系统时，在进行任何工作之前，最好查看一下该机器上运行的 Ubuntu 版本。

新的 Ubuntu 版本每 6 个月发布一次，而 LTS (长期支持) 版本每两年发布一次。桌面和服务器上支持 LTS 版本 5 年，其他标准版本支持 9 个月。

在本教程中，我们将向您展示如何查看您的 Ubuntu 版本。有几种方法可以查看您正在运行的 Ubuntu 版本，包括可以从命令行或图形界面中进行查看。

如何在 Ubuntu 18.04 上安装 Slack

Slack 是世界上最受欢迎的协作平台之一，它将您的所有通信融合在一起。 Slack 中的会话通过频道进行组织。您可以为团队，项目，主题或任何其他目的创建频道，以保持信息和对话的有序性。您可以在频道或消息中搜索发布的所有内容。 Slack 还允许您通过音频或视频通话与您的队友交谈，并共享文档，图像，视频和其他文件。

Slack 不是一个开源应用程序，它不包含在 Ubuntu 存储库中。

本教程介绍如何在 Ubuntu 18.04 上安装 Slack 。相同的说明适用于 Ubuntu 16.04 和任何其他基于 Debian 的发行版，包括 Debian ， Kubuntu ， Linux Mint 和 Elementary OS 。
