如何在 Ubuntu 18.04 上安装 CouchDB

CouchDB 是 Apache Software Foundation 维护的免费开源 NoSQL 数据库。 CouchDB 服务器将其数据存储在命名数据库中，以 JSON 结构存储文档。每个文档可以包含许多字段和附件。字段可以包括文本，数字，列表，布尔值等。CouchDB 提供了一个 RESTful HTTP API ，通过 API 您可以完成读取，创建，编辑和删除数据库文档。 在本教程中，我们将介绍在 Ubuntu 18.04 上从 CouchDB 官方仓库安装最新版本的 CouchDB 的过程。
