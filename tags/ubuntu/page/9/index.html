如何在 Ubuntu 18.04 上安装和使用 Wine

Wine 是一个开源兼容层，允许您在类 Unix 操作系统（如 Linux ， FreeBSD 和 macOS ）上运行 Windows 应用程序。Wine 就是 Wine，不是模拟器。它是一个接口，将 Windows 系统调用转换为 Linux 和其他基于 Unix 的操作系统使用的等效 POSIX 调用。 Wine 并不总是在 Linux 上运行 Windows 程序的最佳选择。并非所有 Windows 应用程序都能在 Wine 中运行，即使它们确实如此，它们可能也不会像通常那样运行。数据库 Wine AppDB 包含了它已经过测试，证实在 Wine 工作的应用程序列表。 您还可以使用 VirtualBox 或 VMware 等虚拟化工具，但它们需要更多系统资源，并且需要安装 Windows。 本教程介绍如何在 Ubuntu 18.04 上安装 Wine 。我们将向您展示如何安装稳定发行版 3.0 和最新版本的 Wine 4.0 。相同的说明适用于 Ubuntu 16.04 和任何基于 Ubuntu 的发行版，包括 Linux Mint 和 Elementary OS 。
