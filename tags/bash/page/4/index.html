Bash 中 Case 语句用法

在 Bash 中，当您有多个不同的选择时， case 语句通常用于简化复杂的条件。 使用 case 语句而不是嵌套的 if 语句将帮助您使 bash 脚本更易读，更易于维护。 Bash 中的 case 语句与 Javascript 或 C 中的 switch 语句具有类似的概念。与 C 中的 switch 语句不同主要在于， Bash 中的 case 语句一旦找到一个并执行与该模式关联的语句，就不会继续搜索模式匹配。 在本教程中，我们将介绍 Bash 案例语句的基础知识，并向您展示如何在 shell 脚本中使用它们。
