如何在Ubuntu 18.04上安装Gradle

Gradle是一个主要用于Java项目的通用构建工具。它结合了Ant和Maven的最佳功能。与使用XML进行脚本编写的前辈不同，Gradle使用Groovy，这是一种动态的，面向对象的Java平台编程语言，用于定义项目和构建脚本。

在本教程中，我们将向您展示如何在Ubuntu 18.04上安装Gradle。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Linux Mint和Elementary OS。

如何在Ubuntu 18.04上安装IntelliJ IDEA

IntelliJ IDEA是用于JVM和Android开发的全功能IDE 。它具有内置的调试支持，Docker和Docker Compose支持，嵌入式Git控件，集成了主要的构建自动化工具，如Maven和Gradle，语法突出显示，代码完成，ssh终端，代码重构和丰富的导航功能。

IntelliJ IDEA可通过插件扩展，并提供对Java框架（如Swing和JavaFX）的支持。它还可以用于其他编程语言的开发，如Python，Kotlin，Groovy，Scala，Erlang，Dart等。

如何在Ubuntu 18.04上安装Netbeans

Netbeans是一个功能齐全的跨平台IDE，可帮助您使用Java，PHP和许多其他语言构建应用程序。

它具有内置的调试支持，嵌入式Git控件，集成了主要的构建自动化工具，如Maven，Ant和Gradle，语法高亮，代码生成器，代码模板，代码重构和丰富的导航功能。

Netbeans可通过插件进行扩展，并且具有完善的API。它还可以用于其他编程语言的开发，例如C / C ++，Groovy，Javadoc，JavaScript等。

本教程介绍如何在Ubuntu 18.04上安装Netbeans IDE。相同的说明适用于Ubuntu 16.04和任何基于Ubuntu的发行版，包括Kubuntu，Linux Mint和Elementary OS。

如何在 Debian 9 上安装 TensorFlow

TensorFlow是由 Google 构建的机器学习的免费开源平台。它被许多组织使用，包括 Twitter ， PayPal ，英特尔，联想和Airbus。

TensorFlow 可以在系统范围内安装，在 Python 虚拟环境中安装，也可以使用 pip 安装为 Docker 容器。出于学习目的，最好在 Python 虚拟环境中安装 TensorFlow 。这样，您可以在一台计算机上拥有多个不同的独立 Python 环境，并在每个项目的基础上安装特定版本的模块，而不必担心它会影响您的其他项目。

本教程将指导您完成在 Debian 9 上安装 TensorFlow 的过程。
