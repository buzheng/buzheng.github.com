如何更改 Ubuntu 的 root 密码

root 用户(或超级用户)是在所有类似 Linux 和 Unix 的系统上都存在的特殊用户帐户。它具有对系统上的每个命令和任何资源的完全访问权限，而没有任何限制。 如果您是使用的是 Ubuntu ，您可能会想知道默认的 root 密码是什么或如何更改 root 密码。默认情况下，在 Ubuntu 中，出于安全原因，禁用了 root 用户帐户。 本教程说明了如何临时更改 root 用户帐户，以及如何在 Ubuntu 系统上设置 root 密码。
