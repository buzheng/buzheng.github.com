如何在Ubuntu中更改用户密码

定期更新密码并为每个帐户使用唯一密码始终是个好主意。

作为Ubuntu的普通用户，您只能更改自己的密码。root用户和具有sudo权限的用户可以更改其他用户的密码，并定义如何使用或更改密码。

本教程介绍如何从命令行或通过Ubuntu GUI更改Ubuntu中的用户密码。
