如何编辑 hosts 文件

hosts 文件用于将域名（主机名）映射到 IP 地址。它是所有操作系统（包括 Linux ， Windows 和 macOS ）使用的纯文本文件。 hosts 文件允许您设置给定域名的 IP 地址。这将仅影响进行更改的计算机，而不是影响全局域的解析方式。
