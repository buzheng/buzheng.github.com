如何在后台运行 Linux 命令

通常，在终端中运行命令时，必须等到命令完成后才能输入另一条命令。这称为在前台或前台进程中运行命令。当进程在前台运行时，它会占用您的 Shell 会话，也允许您使用输入设备与其进行交互。

Linux Nohup 命令

SIGHUP 是当控制终端关闭时发送到进程的信号。而 nohup 命令参数指定执行的另一个命令在执行后会忽略 SIGHUP (挂断) 信号。 通常，当您通过 SSH 运行命令时，如果连接断开，或者您注销， SSH 会话将终止，并且终端执行的所有命令都将停止。这是 nohup 命令派上用场的地方。它将忽略所有挂断信号，命令将继续运行。
