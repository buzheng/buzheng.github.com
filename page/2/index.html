如何在 Linux 中更改 swappiness

交换空间是 RAM 内存已满时使用的硬盘的一部分。交换空间可以是专用交换分区或交换文件。 当 Linux 系统耗尽物理内存时，非活动页面将从 RAM 移动到交换空间。 Swappiness 是一个 Linux 内核属性，用于设置将页面从物理内存交换到交换空间以及从页面缓存中删除页面之间的平衡。它基本上定义了系统使用交换空间的频率。 本文介绍如何更改 Linux 系统上的 swappiness 值。

如何在 Ubuntu 中将用户添加到 sudoers
